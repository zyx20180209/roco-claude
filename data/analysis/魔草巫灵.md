# 魔草巫灵 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 3维 | 特攻112（74%）+坍缩击败后魔攻+70%+心灵洞悉星陨印记+二律背反星陨翻倍 |
| 耐久 | 3维 | 特防139（95%顶级）+物防109（72%）+酶浓度调整减伤90%+应对回血20% |
| 功能性 | 4维 | 木桶戏法队友伪装入场+移花接木中转+晒太阳驱散+富养化队友回能 |

**主定位**：中转伪装型工兵（木桶戏法让队友"伪装"入场+星陨体系+顶级特防联防）
**战术角色**：工兵 + 中转

## 二、种族值与速度

- HP99 / 攻43 / 特攻112 / 防109 / 特防139 / 速90（总592）
- **有效种族549**（去除物攻43）
- **极速220.4**（37%分位，中速）/ 满速192.0
- 启动速度：⭐⭐⭐⭐ 快速（木桶戏法被动触发，汲取1能即可出手）

## 三、属性抗性（幻/草双属性）

- **3x弱点**：虫（极危险）
- **2x弱点**：火、冰、毒、翼、幽（5条偏多）
- **半抗**：水、光、地、电、武、幻（6条）
- **免疫**：无

## 四、特性「木桶戏法」

> 离场后，更换入场的精灵以木桶状态登场

核心机制：魔草巫灵离场后，下一只入场队友以"木桶状态"登场：

- **木桶状态**：伪装成不显示实际精灵，对手难以预判即将上场的是谁，增加博弈成本
- 配合移花接木（回血+脱离）：主动脱离触发木桶戏法
- **战略价值**：对手无法针对性选择克星，主动权在己方
- **局限性**：木桶状态只影响视觉信息，实际机制受限；特性需要主动下场触发

## 五、技能池亮点

**星陨体系**：
- 二律背反 4能 → 敌方3层星陨（应对防御星陨层数翻倍）
- 超维投射 4能 → 敌方4层星陨
- 星轨裂变 1能 → 敌方2层星陨（低费铺星陨）
- 心灵洞悉 7能 → 敌方星陨印记=敌方现有印记层数
- 多维击打 4能15×N → 敌方每层星陨连击+1

**输出**：
- 坍缩 3能85 → 若击败敌方自己获得魔攻+70%
- 棘突 3能100 → 稳定魔攻
- 虹光冲击 3能100 → 光系打击

**防御续航**：
- 酶浓度调整 3能 → 减伤90%+应对回血20%
- 嗜痛 2能 → 减伤80%+每次受伤双攻+40%

**功能/中转**：
- 移花接木 2能 → 回血15%+脱离（主动触发木桶戏法）
- 富养化 3能 → 场下每只精灵回3能
- 晒太阳 1能 → 驱散敌方增益
- 取念 0能 → 复制敌方技能-2能耗

**增益**：
- 魔法增效 0能 → 魔攻+70%
- 盛开 1能 → 威力+30（应对防御+60）

## 六、推荐配招方向

### 路线A：星陨体系流（工兵+破盾）
- 二律背反 + 星轨裂变 + 多维击打 + 心灵洞悉
- 二律背反应对防御星陨翻倍，星轨裂变低费铺垫，多维击打每层星陨连击+1爆发

### 路线B：木桶中转流（中转+工兵）
- 移花接木 + 坍缩 + 酶浓度调整 + 晒太阳
- 移花接木主动触发木桶戏法让队友伪装入场，坍缩击败后魔攻+70%

### 路线C：特防联防流（拦截）
- 酶浓度调整 + 嗜痛 + 棘突 + 魔法增效
- 特防139顶级+酶浓度减伤90%回血，嗜痛受伤双攻+40%反打
