## 四、特性「珊瑚骨」

> 敌方精灵退场时，自己获得全技能能耗-3。

每次敌方精灵退场（被击败/主动换人）触发全技能-3能耗：

- **被击败触发**：击败敌方精灵→珊瑚骨触发全技能-3能耗
- **换人触发**：敌方主动换人→珊瑚骨触发全技能-3能耗
- **天洪联动**（7能150威力，应对状态永久-6能耗）：珊瑚骨-3+天洪应对-6=天洪能耗从7降至-2（实际0能）
- **打鼾联动**（6能165威力）：珊瑚骨-3后打鼾3能165威力
- **润泽联动**（7能魔攻+190%）：珊瑚骨-3后润泽4能使用
- **涌泉联动**（能耗每-1威力+10）：珊瑚骨-3+打湿-1=涌泉威力大幅提升
- **核心价值**：每次敌方退场=全技能-3能耗，换人频繁的对手让海枝枝越来越低费
- **局限性**：需要敌方退场才触发；5条2x弱点偏多；HP78偏低

## 五、技能池亮点

**珊瑚骨联动高费技**：
- 天洪 7能150 → 应对状态永久-6能耗（珊瑚骨-3后4能，应对后-2能）
- 打鼾 6能165 → 顶级威力（珊瑚骨-3后3能）
- 润泽 7能 → 魔攻+190%（珊瑚骨-3后4能）
- 涌泉 6能60 → 能耗每-1威力+10（珊瑚骨-3后3能+威力+30）
- 水波术 6能90 → 回合结束永久+20威力（珊瑚骨-3后3能）

**能量控场**：
- 勾魂 1能 → 偷取敌方3能量
- 精神扰乱 0能 → 全技能能耗+1（应对防御+3）
- 摇篮曲 5能 → 能耗+3+应对防御打断+眩晕
- 背袭 2能40 → 敌方能量=0时20倍=800威力（极限）

**高威力输出**：
- 打鼾 6能165 → 顶级魔攻
- 天洪 7能150 → 高威力+应对永久降费
- 消毒法 4能115 → 驱散5层增益+魔攻
- 恐吓 2能80 → 低费魔攻

**增益**：
- 润泽 7能 → 魔攻+190%
- 魔法增效 0能 → 魔攻+70%
- 嘲弄 2能 → 魔攻+90%（换人时速度+70）
- 伺机而动 1能 → 下次攻击+70威力

**续航**：
- 根吸收 2能 → 回血15%+4能
- 汲取 1能30 → 吸血100%
- 落雨 5能 → 雨天水系+50%威力
