# 卷胡巨獭 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 2维 | 双攻96均衡+水炮永久递减+触底强击能耗耗尽+175威力 |
| 耐久 | 4维 | HP121（76%）+双防101均衡+保守派低能耗双防+80%+水环应对-2能耗 |
| 功能性 | 3维 | 激怒/操控能耗压制+咆哮物攻-130%+洗礼自清+水环资源回收 |

**主定位**：耐久型拦截（保守派低能耗触发双防+80%+能耗控场）
**战术角色**：拦截 + 工兵

## 二、种族值与速度

- HP121 / 攻96 / 特攻96 / 防101 / 特防101 / 速80（总595）
- **有效种族499**（去除特攻96）
- **极速207.2**（24%分位，偏慢）/ 满速181.0
- 启动速度：⭐⭐⭐⭐ 快速（保守派被动即时，低能耗技能即可触发）

## 三、属性抗性（普通/水双属性）

- **2x弱点**：草、电、武（3条）
- **半抗**：火、幽、机械（3条）
- **免疫**：无

## 四、特性「保守派」

> 总技能能耗小于4时，自己获得双防+80%

触发条件：技能栏中所有技能的能耗总和 < 4，即构筑时主动选择低能耗技能池：

- 0能+0能+1能+1能 = 2能 → 触发，双防+80%（物防101→182，特防101→182）
- 0能+1能+2能+1能 = 4能 → **不触发**（需严格小于4）
- **构筑约束极强**：4个技能栏全部必须是低能耗技能，高威力大招无法携带
- **局限性**：触发后输出能力极受限；速度80偏慢

## 五、技能池亮点（保守派触发条件下）

**满足保守派的低能耗组合**：
- 落星0能 + 水弹0能 + 咆哮1能 + 洗礼1能（总2能）→ 触发
- 落星0能 + 水弹0能 + 有效预防1能 + 洗礼1能（总2能）→ 触发

**不触发保守派但高价值技能**：
- 水炮 5能110 → 永久-1能耗，叠加后递减
- 触底强击 3能65 → 能耗耗尽后+110=175威力
- 激怒 3能 → 敌方其他技能能耗+3持续3回合
- 操控 1能 → 敌方本回合技能能耗+7

**防御技**：
- 水环 2能 → 减伤60%+全技能能耗-2
- 水泡盾 2能 → 减伤80%+魔攻+70%

## 六、推荐配招方向

### 路线A：保守派拦截流（拦截）
- 落星 + 水弹 + 咆哮 + 洗礼
- 总能耗2，触发保守派双防+80%，咆哮压制物攻-130%，洗礼自清

### 路线B：保守派+有效预防流（拦截）
- 落星 + 水弹 + 有效预防 + 洗礼
- 总能耗2，触发保守派，有效预防应对后先手+1

### 路线C：放弃保守派的能耗控场流（工兵）
- 激怒 + 操控 + 水炮 + 咆哮
- 放弃保守派，激怒+操控彻底封锁对手能耗，水炮递减后高威力
