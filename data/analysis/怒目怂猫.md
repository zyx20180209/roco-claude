## 四、特性「威慑」

> 打断敌方时，被打断的技能进入2回合冷却。

每次成功打断敌方技能，该技能进入2回合冷却：

- **截拳联动**（应对状态打断+回能）：应对触发打断→被打断技能冷却2回合，自己回能继续出手
- **摇篮曲联动**（应对防御打断+眩晕）：打断后冷却2回合+眩晕双重控场
- **地刺联动**（应对状态打断）：应对触发打断→冷却2回合
- **核心价值**：打断不只是1回合干扰，而是2回合封锁对手核心技能
- **局限性**：需要触发打断才生效；速度95中速，被高速先手压制；翼/萌/幻3条弱点

## 五、核心战术：打断封锁

**威慑触发链**：
1. 截拳（应对状态打断+回能）→ 威慑触发2回合冷却 → 自己回能再出手
2. 摇篮曲（应对防御打断+眩晕）→ 威慑触发2回合冷却 → 对手核心技能被封2回合
3. 地刺（应对状态打断）→ 威慑触发2回合冷却

**打断封锁价值**：对手依赖特定核心技（力量增效/高费大招/增益技）时，被打断后2回合无法使用，节奏完全被压制。

## 六、技能池亮点

**打断控场核心**：
- 截拳 3能90 → 应对状态打断+回能（威慑触发2回合冷却）
- 摇篮曲 5能 → 能耗+3+应对防御打断+眩晕（威慑触发2回合冷却）
- 地刺 3能95 → 应对状态打断（威慑触发2回合冷却）
- 无影脚 3能85 → 应对状态2倍威力=170

**高威力输出**：
- 一拳 5能140 → 稳定高威力物攻
- 技巧打击 2能35 → 应对状态10倍=350威力（极限爆发）
- 偷袭 3能85 → 应对状态3倍=255威力
- 顶端优势 3能100 → 稳定武系输出

**破盾/增益**：
- 锐利眼神 2能 → 敌方双防-120%（团队破盾）
- 力量增效 1能 → 物攻+100%
- 提气 4能 → 全技能威力+40（换人+50）
- 化劲 2能 → 全技能威力+40

**防御反击**：
- 有效预防 2能 → 减伤50%+应对后先手+1
- 加固 2能 → 物防+140%

## 七、推荐配招方向

### 路线A：打断封锁流（拦截+炮台）
- 截拳 + 摇篮曲 + 力量增效 + 锐利眼神
- 截拳应对打断+回能触发威慑2回合冷却，摇篮曲应对防御打断+眩晕，锐利眼神破盾

### 路线B：应对爆发流（炮台）
- 技巧打击 + 无影脚 + 截拳 + 力量增效
- 技巧打击应对10倍=350威力，无影脚应对2倍=170威力，截拳应对打断回能

### 路线C：稳定输出流（炮台）
- 一拳 + 顶端优势 + 力量增效 + 提气
- 力量增效+100%物攻，提气+40威力，一拳140稳定输出
