- **免疫**：无

## 四、特性「毒腺」

> 使用能耗小于等于1的技能时，敌方获得4层中毒。

每次使用0能或1能技能时触发4层中毒：

- **贮藏联动**（携带0能技能越多双攻越高）：芋香巨角蛛技能池大量0能技，贮藏+50%/个0能技
- **以毒攻毒联动**：敌方每1层中毒=自己魔攻+30%，4层=魔攻+120%
- **落井下毒联动**：减益层数翻倍，4层中毒→8层，以毒攻毒=魔攻+240%
- **毒液渗透联动**：敌方每1层中毒本技能能耗-1，8层中毒=能耗-8（几乎0能120威力）
- **核心价值**：0能技能随手触发4层中毒，配合落井下毒翻倍后毒液渗透近0能120威力爆发
- **局限性**：双攻61极低，依赖中毒叠层才有输出；5条2x弱点偏多

## 五、核心战术：中毒叠层爆发链

**叠层流程**：
1. 毒腺触发4层中毒（任意0能技）
2. 落井下毒→8层中毒
3. 以毒攻毒→魔攻+240%
4. 毒液渗透→能耗-8=近0能120威力爆发

## 六、技能池亮点

**中毒叠层核心**：
- 毒腺（被动）→ 0能技能触发4层中毒
- 落井下毒 1能 → 减益层数翻倍（4层→8层）
- 以毒攻毒 1能 → 魔攻+30%/中毒层（8层=+240%）
- 毒液渗透 8能120 → 中毒每层-1能耗（8层=0能120威力）
- 感染病 4能85 → 击败后中毒转印记（持续伤害）

**0能触发技（毒腺触发源）**：
- 虫网 0能30 → 回1能（资源循环）
- 腐蚀酸液 0能35 → 魔伤+2层中毒（叠加毒腺=6层）
- 毒泡泡 0能40 → 魔伤
- 毒针 0能20 → 物伤+1层中毒

**控场**：
- 激怒 3能 → 敌方其他技能能耗+3持续3回合
- 落井下毒 1能 → 减益翻倍

**防御续航**：
- 应激反应 2能 → 回血25%（应对防御50%）
- 虫茧 3能 → 回血20%+奉献吸血
- 防御 0能 → 减伤70%

## 七、推荐配招方向

### 路线A：中毒爆发流（炮台）
- 落井下毒 + 以毒攻毒 + 毒液渗透 + 腐蚀酸液
- 腐蚀酸液0能触发毒腺4层+自带2层=6层，落井下毒翻倍12层，以毒攻毒+360%，毒液渗透近0能120威力

### 路线B：中毒控场流（工兵）
- 落井下毒 + 激怒 + 感染病 + 虫网
- 落井下毒翻倍+激怒能耗+3双重控场，感染病击败后中毒转印记，虫网0能回能循环

### 路线C：贮藏爆发流（炮台）
- 贮藏 + 以毒攻毒 + 毒液渗透 + 腐蚀酸液
- 贮藏携带多个0能技双攻叠加，以毒攻毒+中毒层数，毒液渗透低费爆发
