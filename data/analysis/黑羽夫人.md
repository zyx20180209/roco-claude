# 黑羽夫人 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |

| 攻击力度 | 3维 | 双攻93/87偏中+灾厄1能150威力应对状态转敌方+栽赃6能150+贪婪偷取敌方增益 |

**主定位**：偷取增益型联防（贪婪反向继承敌方buff+物防130联防+极速260高速）
**战术角色**：拦截 + 工兵 + 中转

- **2x弱点**：光、冰、电、萌（4条）

- **免疫**：无

## 四、特性「贪婪」

> 敌方精灵离场后，其增益和减益会被更换入场的精灵继承

**关键性质**：
- **敌方下场时，敌方的buff被新入场的敌方继承**（不是自己继承）
- **核心价值**：让对手的强化精灵"被迫传递"给不该有buff的队友
- 如果对手原本铺强化的是主C，主C下场后新入场精灵继承这些buff但可能根本用不上
- 配合风隐（双方脱离）：主动强制对手换人触发贪婪继承
- **与翠顶夫人洁癖的对比**：
  - 洁癖 = 己方下场传给己方入场者（正向增益传递）
  - 贪婪 = 敌方下场传给敌方入场者（反向增益"误传"）

**局限性**：如果对手精灵不下场贪婪不触发；对手如果无减益buff，贪婪价值有限

## 五、技能池亮点

**贪婪联动**：
- 风隐 1能 → 双方脱离，强制对手换人触发贪婪
- 飞羽 0能 → 迅捷+驱散敌方1种增益（贪婪没传过去的也能清掉）

**高效输出**：
- 灾厄 1能150 → 应对状态转敌方，1能高爆发神技
- 栽赃 6能150 → 高费高威力魔攻
- 水刃 4能115 → 应对状态能耗永久-4
- 俯冲猛击 5能140 → 翼系高威力物攻

**迅捷工具**：
- 飞羽 0能 → 迅捷+驱散
- 翼击 3能50 → 迅捷
- 龙卷风 5能70 → 迅捷+应对状态1.5倍
- 风墙 2能 → 减伤50%+迅捷

**防御核心**：
- 等价交换 3能 → 减伤90%+应对50%吸血
- 无畏之心 5能 → 减伤100%+回血
- 嗜痛 2能 → 减伤80%+每次受伤双攻+40%

**破盾/控场**：
- 锐利眼神 2能 → 敌方双防-120%（团队破盾神技）
- 聒噪 3能 → 敌方攻击技能耗+3持续3回合
- 迫害 0能40 → 低费骚扰

## 六、推荐配招方向

### 路线A：贪婪反制流（拦截+工兵）
- 风隐 + 飞羽 + 灾厄 + 等价交换
- 风隐强制对手换人触发贪婪，飞羽驱散残留buff，灾厄应对状态1能150威力

### 路线B：联防破盾流（拦截+破盾）
- 锐利眼神 + 水刃 + 等价交换 + 力量增效
- 锐利眼神双防-120%破盾，水刃叠加后0能115威力

### 路线C：迅捷压制流（收割）
- 飞羽 + 翼击 + 龙卷风 + 俯冲猛击
- 极速260+全迅捷技能池，开局连续迅捷压制

## 八、擅长应对的情况

- 强化型队伍（贪婪反向误传对手buff）
- 中转体系（风隐强制换人+贪婪触发）
- 物攻核心对手（物防130+等价交换90%减伤）
- 快节奏对局（极速260顶级先手）

## 九、棘手的对手特点

- 光/电/萌系先手压制（4条2x弱点+热门）
- 不换人的对手（贪婪无触发机会）
- 无增益的对手（贪婪价值打折）
- 特防90偏中等，高特攻威胁大

- 飞羽0能迅捷驱散
- 锐利眼神双防-120%破盾（团队破盾工具）
- 灾厄1能150应对状态转嫁神技

### 体系归属

- 翼系迅捷体系（飞羽/翼击/龙卷风/风墙）
- 恶/翼主题队
- 换人节奏队（风隐+贪婪）
