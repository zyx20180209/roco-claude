# 巨灵石 PVP 对战分析

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 3维 | 双攻99均衡+石天平高费技能偷能+穿膛低能5倍+岩土暴击被攻击递减 |

**战术角色**：工兵 + 破盾（穿膛低能5倍）

- HP95 / 攻99 / 特攻99 / 防126 / 特防70 / 速95（总584）

- **2x弱点**：草、水、光、冰、幽、恶、机械（7条，极多）
- **半抗**：火、电、虫、翼（4条）
- **免疫**：无

## 四、特性「石天平」

> 若使用技能能耗高于敌方，回合结束敌方失去能耗之差的能量

使用高费技能时，若能耗高于对手本回合技能能耗，差值从对手能量中扣除：

- 自己用10能地震，对手用3能技能 → 对手失去7能
- 自己用8能岩土暴击，对手用1能技能 → 对手失去7能
- 自己用5能幽灵爆发，对手用0能技能 → 对手失去5能
- **核心价值**：使用高费技能同时偷取对手能量，攻击+偷能双重效果
- **局限性**：7条2x弱点极多；特防70极脆；需要使用高费技能才触发

## 五、叠加型技能发育分析

**识别**：岩土暴击（8能，每被攻击1次永久-1能耗）

- **岩土暴击**：初始8能，每被攻击-1。被攻击8次后趋近0能140威力。
- **石天平联动**：岩土暴击8能使用时，若对手用低费技能，偷取大量能量。
- **发育阶段风险**：特防70极脆，魔攻爆发可绕过物防直接秒杀。

## 六、技能池亮点

**石天平联动高费技**：
- 地震 10能190 → 石天平最大偷能（对手用0能时偷10能）
- 岩土暴击 8能140 → 石天平偷能+被攻击递减
- 幽灵爆发 5能140 → 石天平偷能+高威力
- 陨石 3能100 → 稳定输出

**能量压制**：
- 勾魂 1能 → 偷3能
- 恶作剧 1能 → 敌方失去3能（应对防御6能）
- 穿膛 2能65 → 敌方能量≤2时5倍=325威力
- 降灵 2能 → 降灵印记换人-1能
- 聒噪 3能 → 敌方攻击技能耗+3持续3回合
- 瞬间零度 0能 → 本回合敌方技能能耗+3

**控场**：
- 流沙 2能 → 禁换3回合（应对防御双防-60%）
- 棘刺 2能 → 棘刺印记换人-6%血
- 精神扰乱 0能 → 敌方能耗+1

**防御**：
- 不动如山 2能 → 减伤90%
- 均势 3能 → 物防+140%+速度-30

## 七、推荐配招方向

### 路线A：石天平高费偷能流（工兵+破盾）
- 地震 + 勾魂 + 穿膛 + 降灵
- 地震10能触发石天平偷能，勾魂偷3能，穿膛低能5倍终结，降灵封锁换人

### 路线B：岩土暴击发育流（炮台+工兵）
- 岩土暴击 + 恶作剧 + 穿膛 + 不动如山
- 岩土暴击被攻击递减+石天平偷能，恶作剧偷3能，穿膛终结

### 路线C：能耗封锁流（工兵）
- 聒噪 + 瞬间零度 + 勾魂 + 穿膛
- 聒噪+3能耗+瞬间零度+3能耗彻底封锁，勾魂偷能，穿膛低能终结
