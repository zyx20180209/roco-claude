# 花魁蜂后 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 2维 | 物攻54偏弱+虫群鼓舞虫系队友每只攻速+10%+虫群奉献机制 |
| 耐久 | 1维 | HP145（95%顶级）；双防45极脆（2%/3%分位） |
| 功能性 | 4维 | 奉献机制核心（虫群过境/捆缚/虫茧奉献）+翅刃驱散/偷印记+虫鸣队伍联动+虫群鼓舞虫系队伍增益 |

**主定位**：虫系队奉献核心工兵（虫群鼓舞+奉献机制）
**战术角色**：工兵 + 清场（翅刃驱散印记）

## 二、种族值与速度

- HP145 / 攻54 / 特攻49 / 防45 / 特防45 / 速44（总382）
- **有效种族333**（去除特攻49，全游戏3%分位）
- **极速159.7**（1%分位，全游戏倒数）/ 满速141.4
- 启动速度：⭐⭐ 慢速（虫群鼓舞入场被动，但速度44极慢）

## 三、属性抗性（翼/虫双属性）

- **2x弱点**：火、冰、电、翼（4条）
- **半抗**：虫（仅1条）
- **免疫**：无

## 四、特性「虫群鼓舞」

> 队伍中每有1只其他的虫系精灵，自己入场时获得攻速+10%

虫系队专属入场增益：

- 队伍带1只虫系队友 → 入场攻速+10%
- 队伍带3只虫系队友 → 入场攻速+30%
- 虫系专队全员虫系（5只虫）→ 入场攻速+50%
- **局限性**：必须虫系队才能发挥；非虫系队中特性失效；+10%/只增益幅度较小

## 五、核心战术：奉献机制

**奉献**是虫系队专属机制——部分技能触发"己方队伍获得1次奉献"，其他技能可消耗奉献获得额外效果：

**花魁蜂后的奉献生产技**：
- 虫茧 1能 → 回血20%+生产奉献（给队友吸血10%）
- 捆缚 2能 → 给敌方2层中毒+生产奉献（再+2层中毒）
- 虫群过境 5能45×2 → 生产奉献（连击数+1）
- 假寐（血脉技）2能 → 生产奉献（能耗-2）

**核心价值**：花魁蜂后作为奉献生产核心，为虫系队友提供吸血/连击/能耗等多重增益

## 六、技能池亮点

**奉献机制核心**：
- 虫茧 1能 → 回血20%+全队吸血10%奉献
- 捆缚 2能 → 2层中毒+再2层中毒奉献
- 虫群过境 5能45×2 → 奉献+1连击
- 虫群 7能20 → 高费低威力但受奉献影响（多重奉献后威力暴涨）

**功能**：
- 翅刃 4能95 → 驱散敌方所有印记（应对状态偷取印记）
- 虫鸣 2能15×N → 队伍每带1个虫鸣连击+1
- 倾泻 3能70 → 未被应对则驱散双方印记
- 龙卷风 5能70 → 迅捷，应对状态1.5倍
- 精神扰乱 0能 → 敌方能耗+1

**输出（有限）**：
- 草虫冲击 3能80 → 敌方换人时威力+50且无视抵抗
- 虫击 3能90 → 应对状态2倍+无视抵抗

## 七、推荐配招方向

### 路线A：虫系队奉献核心流（工兵）
- 虫茧 + 捆缚 + 虫群过境 + 翅刃
- 虫茧给队友吸血10%，捆缚中毒+奉献再中毒，虫群过境连击奉献，翅刃驱散印记

### 路线B：翅刃清印记流（清场+工兵）
- 翅刃 + 虫茧 + 捆缚 + 倾泻
- 翅刃+倾泻双重驱散印记，虫茧/捆缚奉献支援

### 路线C：中毒控场流（工兵+破盾）
- 捆缚 + 腐化 + 鸩毒 + 瘴气喷射
- 捆缚铺中毒+奉献再中毒，腐化每层双攻-30%，鸩毒每层+10威力
