# 古卷匣魔像 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 3维 | 双攻109均衡+崩拳应对物攻+100%+听桥应对反伤+伺机而动攻击+70威力+垂死反击低血爆发 |
| 耐久 | 5维 | HP114+物防134（92%）+构装契约者敌方魔力1时双防+50%+无畏之心减伤100%回血+嗜痛减伤80%+虚假破产能量不足用血代能 |

**战术角色**：拦截 + 炮台

- **3x弱点**：无

## 四、特性「构装契约者」

> 入场时，若敌方魔力值为1，自己获得双防+50%。

敌方魔力值为1时入场触发双防+50%：

- **末位魔力触发**：敌方魔力消耗至1点时，古卷匣魔像入场即获得双防+50%
- **物防134+50%=201等效**：物防134（92%）触发后等效201，顶级防御
- **与无畏之心联动**：无畏之心减伤100%+回血，双防+50%基础更高
- **与嗜痛联动**：嗜痛减伤80%+应对双攻+40%×N，双防+50%让嗜痛更耐打
- **与虚假破产联动**：能量不足时用5%血代1能，双防+50%让血量消耗更安全
- **核心价值**：敌方魔力紧张时反而更难击倒，天然适合消耗战中期拦截
- **局限性**：需要敌方魔力恰好为1才触发；特防89偏低；速度95中等

## 五、核心战术：末位魔力拦截反制

**拦截流程**：
1. 等待敌方魔力消耗至1点
2. 古卷匣魔像入场→构装契约者触发双防+50%
3. 嗜痛（2能）→ 减伤80%+应对双攻+40%叠加
4. 崩拳（2能65）→ 应对状态物攻+100%爆发

**听桥反伤流**：
1. 构装契约者入场双防+50%
2. 听桥（4能）→ 减伤60%+应对反伤（威力=被应对技能威力）
3. 对高威力技能反伤极高

## 六、技能池亮点

**核心输出**：
- 影袭 3能100 → 稳定武系物攻
- 崩拳 2能65 → 应对状态物攻+100%（低费爆发）
- 垂死反击 4能80 → 每失去5%血+5威力（低血爆发）
- 斩断 2能75 → 应对打断（武系）

**增益**：
- 伺机而动 1能 → 下次攻击+70威力
- 力量增效 1能 → 物攻+100%
- 防御反击 3能 → 减伤80%+应对全技能+40威力

**防御续航**：
- 无畏之心 5能 → 减伤100%+回血（能耗永久+2）
- 嗜痛 2能 → 减伤80%+应对双攻+40%
- 虚假破产 2能 → 减伤80%+能量不足用血代能
- 血气 2能 → 减伤60%+应对致命保留1血
- 吓退 2能 → 减伤60%+应对敌方脱离

**反伤**：
- 听桥 4能 → 减伤60%+应对反伤（威力=被应对技能）

**控场**：
- 掠夺 1能60 → 低费恶系魔攻

## 七、推荐配招方向

### 路线A：末位魔力拦截流（拦截核心）
