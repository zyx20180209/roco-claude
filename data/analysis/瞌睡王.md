# 瞌睡王 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 4维 | 物攻157（97%）+慢热型应对回能+技巧打击应对10倍+能量刃应对永久+90威力 |
| 耐久 | 3维 | HP144（94%）+物防135（93%）+特防110（71%）均衡三防 |
| 功能性 | 2维 | 晒太阳驱散+气沉丹田应对后递减+回旋踢换人翻倍 |

**主定位**：应对反击型炮台（慢热型应对回能+顶级三防+物攻97%分位）
**战术角色**：炮台 + 拦截

## 二、种族值与速度

- HP144 / 攻157 / 特攻64 / 防135 / 特防110 / 速75（总685）
- **有效种族621**（去除特攻64，全游戏顶级）
- **极速200.6**（19%分位，极慢）/ 满速175.5
- 启动速度：⭐⭐ 慢速（初始0能，依赖队友应对积累能量）

## 三、属性抗性（普通/武双属性）

- **2x弱点**：武、翼、萌、幻（4条）
- **半抗**：地、虫、幽、恶（4条）
- **免疫**：无

## 四、特性「慢热型」

> 初始能量为0，入场前己方精灵每成功应对1次，回复5能量

核心机制：瞌睡王入场时能量为0，但队友每次成功应对+5能：

- 队友应对2次 → 入场即有10能，可直接释放压扁（5能）或后发制人（3能）
- 队友应对4次 → 入场即有20能（上限10），满能入场
- **应对流联动**：队友带大量防御技，每次应对为瞌睡王积累能量
- **气沉丹田联动**：10能大招，每次应对后能耗-3，应对3次后0能使用
- **局限性**：完全依赖队友应对积累；速度75极速200极慢

## 五、叠加型技能发育分析

**识别**：能量刃（每应对成功1次，本技能威力永久+90）

- **发育曲线**：能量刃初始80威力，每次应对+90。应对1次→170，应对3次→350，应对5次→530威力。
- **慢热型联动**：队友应对积累能量的同时，能量刃威力也在叠加。
- **发育阶段风险**：初始0能，完全依赖队友；但三防顶级，入场后自保能力极强。
- **队伍依赖度**：极高。需要专门的应对流队伍配合。

## 六、推荐配招方向

### 路线A：应对反击流（炮台+拦截）
- 技巧打击 + 能量刃 + 压扁 + 力量增效
- 技巧打击应对10倍350威力，能量刃叠加，力量增效+100%物攻后压扁爆发

### 路线B：气沉丹田流（炮台）
- 气沉丹田 + 后发制人 + 斩断 + 应激反应
- 气沉丹田应对3次后0能使用（回血60%+物攻+130%），后发制人155威力终结

### 路线C：应对控场流（拦截+工兵）
- 斩断 + 回旋踢 + 晒太阳 + 应激反应
- 斩断应对打断，回旋踢换人翻倍，晒太阳驱散，应激反应续航

## 八、擅长应对的情况

- 应对流队伍（慢热型+能量刃双重积累）
- 物攻/特攻输出（三防顶级全面压制）
- 依赖增益积累的对手（晒太阳驱散）
- 换人频繁的对手（回旋踢换人翻倍）

## 九、棘手的对手特点

- 萌/幻系先手压制（2x弱点）
- 速度75极速200，几乎所有精灵先手压制
- 初始0能，若队友无法积累应对则入场完全无法出手
- 依赖队友应对，队伍构筑门槛极高

## 十、队伍需求

### 精灵缺的
- 应对流队友（慢热型核心需求，至少2只有大量防御技的队友）
- 萌/幻系克星清理
- 速度补位（极速200极慢）

### 精灵给的
- 全游戏顶级三防联防（有效种族621）
- 物攻157顶级爆发
- 能量刃应对叠加后530威力
- 气沉丹田应对递减后0能回血+物攻+130%
- 技巧打击应对10倍350威力

### 体系归属
- 应对流队伍绝对核心
- 全能联防位（三防顶级）
- 反换人针对队（回旋踢）
