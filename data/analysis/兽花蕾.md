# 兽花蕾 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 2维 | 双攻79/82偏中等；稀兽花宝入场随血脉提供多种增益（魔攻+80%/物攻+80%等） |
| 耐久 | 1维 | HP70（3%分位）+双防70/97偏脆；极速253中高速 |
| 功能性 | 4维 | 稀兽花宝血脉定制入场效果+折射血脉定制+放晴光系威力+50%+移花接木中转 |

**主定位**：血脉定制型工兵（稀兽花宝根据血脉提供不同入场效果）
**战术角色**：工兵 + 中转

## 二、种族值与速度

- HP70 / 攻79 / 特攻82 / 防70 / 特防97 / 速115（总513）
- **有效种族434**（全游戏14%分位，面板偏弱）
- **极速253.4**（79%分位，中高速）/ 满速219.5
- 启动速度：⭐⭐⭐⭐⭐ 即时（稀兽花宝入场即触发）

## 三、属性抗性（光/草双属性）

- **2x弱点**：草、火、冰、毒、虫、翼、幽（7条，极多）
- **半抗**：水、光、地、电、恶、幻（6条）
- **免疫**：无

## 四、特性「稀兽花宝」——血脉定制核心

> 根据自身的血脉，入场时获得不同效果

**18种血脉对应18种入场效果**（按战术价值分级）：

**强力入场增益**（主流推荐）：
- **武血脉** → 物攻+80%（最强进攻增益）
- **光血脉** → 魔攻+80%（最强魔攻增益）
- **电血脉** → 速度+80（极速253→333，进入绝对先手段）
- **机械血脉** → 双防+60%（HP低的救命补血）

**DOT入场破盾**：
- **火血脉** → 对手6层灼烧
- **冰血脉** → 对手2层冰冻（冻结可绕过化茧斩杀）
- **毒血脉** → 对手3层中毒
- **幻血脉** → 对手2层星陨

**能量压制**：
- **幽血脉** → 对手-4能（配合背袭联动）

**属性压制**：
- **萌血脉** → 对手双攻-60%
- **龙血脉** → 对手魔防-80%（极强破盾）
- **虫血脉** → 对手物防-70%
- **地血脉** → 对手速度-60+连击-3

**续航**：
- **草血脉** → 入场回20%血
- **恶血脉** → 50%吸血

**其他**：
- **普通血脉** → 技能威力+30
- **翼血脉** → 连击数+2
- **水血脉** → 能耗-2

## 五、技能池亮点

**输出**：
- 迫近攻击 3能90 → 每次使用威力永久+45（发育型）
- 过曝 3能60 → 每使用1个其他系别技能威力永久+30（发育型）
- 叶绿光束 4能120 → 稳定魔攻
- 彗星 0能240 → 消耗全部生命换爆发

**血脉定制技**：
- **折射 4能50** → 根据其他系别技能触发不同效果（水-1能耗/电速度+50/草回15%血/火4层灼伤）
- 放晴 0能 → 光系技能威力永久+50%（应对防御+100%）

**增益/续航**：
- 魔法增效 0能 → 魔攻+70%
- 根吸收 2能 → 回血15%+4能
- 移花接木 2能 → 回血15%+脱离
- 徒长 2能 → 回10能

**控场**：
- 晒太阳 1能 → 驱散敌方增益

## 六、叠加型技能发育分析

**识别**：迫近攻击（每次使用威力永久+45）+ 过曝（每使用其他系别技能永久+30）

- **迫近攻击**：初始90，使用3次→225威力
- **过曝**：初始60，配合多系别技能池快速叠加
- **发育阶段风险**：HP70+双防偏脆，7条弱点多，发育期极易被秒
- **队伍依赖度**：高，需要保护

## 七、推荐配招方向

### 路线A：武血脉物攻流（工兵+收割）
- 迫近攻击 + 顶端优势 + 放晴 + 移花接木
- 武血脉入场+80%物攻→迫近攻击爆发，放晴光系技威力+50%

### 路线B：光血脉魔攻流（工兵+炮台）
- 魔法增效 + 叶绿光束 + 折射 + 移花接木
- 光血脉入场+80%魔攻→配合魔法增效+70%，叶绿光束爆发

### 路线C：电血脉高速流（收割+中转）
- 透射 + 顶端优势 + 快速移动 + 移花接木
- 电血脉入场速度+80→极速333绝对先手，快速移动再+80速度

### 路线D：龙血脉破盾流（工兵+破盾）
- 叶绿光束 + 折射 + 过曝 + 移花接木
- 龙血脉入场对手魔防-80%→叶绿光束+过曝爆发

## 九、擅长应对的情况

- 血脉针对（选择对手克星属性的血脉定制）
- 需要一次性战略效果的队伍（火6层灼烧/冰2层冻结/幽-4能）
- 中转队伍（移花接木反复入场触发稀兽花宝）
- 破盾场景（龙血脉魔防-80%/虫血脉物防-70%）

## 十、棘手的对手特点

- 几乎所有高物攻/高特攻精灵（面板极低被秒）
- 7条2x弱点涵盖所有热门属性
- 清除效果（洗礼驱散入场增益）
- HP70+双防脆，一击必死

## 十一、队伍需求

### 精灵缺的
- 保护型队友（HP70极脆）
- 克星清理（7条弱点太多）
- 血脉定制规划（选择适合队伍战术的血脉）

### 精灵给的
- 高度定制化入场效果（18种血脉任选）
- 战略级一次性增益（武/光+80%/电+80速/龙-80%魔防等）
- 移花接木中转反复触发
- 博弈威慑（对手无法预判血脉）

### 体系归属
- 中转队反复触发入场效果
- 针对性定制位（按克星选血脉）
- 工兵铺设位
