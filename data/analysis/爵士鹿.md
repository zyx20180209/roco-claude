## 四、特性「蓄电池」

> 每入场1次，永久获得双攻+20%。

每次入场都永久叠加双攻+20%：

- **过载回路联动**（1能返场+下回合次数+1）：返场后再次入场=再次+20%双攻
- **闪击折返联动**（5能45×2+自己脱离）：脱离后再入场=再次+20%双攻
- **远程访问联动**（强制敌方返场）：敌方换人后己方可选择换入爵士鹿=再次+20%
- **层数示例**：入场5次=双攻+100%；入场10次=双攻+200%
- **与力量增效叠加**：蓄电池+100%+力量增效+100%=双攻+200%（5次入场后）
- **核心价值**：每次入场都变强，过载回路循环让双攻无限叠加
- **局限性**：需要多次入场积累；地系唯一2x弱点；特防79偏低

## 五、核心战术：入场循环叠加

**循环流程**：
1. 入场→蓄电池+20%双攻→使用过载回路（1能返场+下回合次数+1）
2. 队友出场1回合
3. 爵士鹿再入场→蓄电池再+20%双攻（累计+40%）
4. 重复循环→双攻持续叠加
5. 积累到+100%~+200%后，力量增效+100%，极速260先手爆发

## 六、技能池亮点

**入场循环核心**：
- 过载回路 1能 → 返场+下回合次数+1（蓄电池循环核心）
- 闪击折返 5能45×2 → 自己脱离（触发再入场）

**高威力输出**：
- 光刃 4能120 → 稳定物攻
- 裂石 3能95 → 应对状态敌方物防-80%
- 偷袭 3能85 → 应对状态3倍=255威力
- 电弧 3能80 → 迸发+40威力=120
- 当头棒喝 3能80 → 换人时威力+100=180

**增益**：
- 力量增效 1能 → 物攻+100%（与蓄电池叠加）
- 泥浆铠甲 2能 → 物攻物防+60%+应对防御增益翻倍
- 漫反射 1能 → 每系别至多一个技能+35威力
- 贪婪 2能 → 100%吸血

**控场**：
- 远程访问 2能 → 强制敌方返场（触发己方再入场机会）
- 晒太阳 1能 → 驱散敌方所有增益
- 震击 3能80 → 敌方连击-3

**应对**：
- 强制重启 3能90 → 应对状态敌方返场

## 七、推荐配招方向

### 路线A：过载循环滚雪球流（收割核心）
- 过载回路 + 光刃 + 力量增效 + 裂石
- 过载回路循环叠加蓄电池双攻，力量增效+100%，裂石应对物防-80%，光刃120威力爆发

### 路线B：换人惩罚流（炮台）

- 远程访问强制敌方换人，当头棒喝换人时180威力，蓄电池每次入场+20%
