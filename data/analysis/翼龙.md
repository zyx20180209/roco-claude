# 翼龙 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 4维 | 物攻114（75%）+暴食龙系技能全部迅捷+极速253先手+龙吟蓄力双攻+100%+吹炎蓄力170威力 |

**主定位**：龙系全迅捷型收割（暴食携带的龙系技能全部获得迅捷+极速253+物攻114）
**战术角色**：收割 + 炮台

## 二、种族值与速度

- HP83 / 攻114 / 特攻31 / 防91 / 特防79 / 速115（总513）
- **有效种族482**（去除特攻31）
- **极速253.4**（82%分位，高速）/ 满速219.5
- 启动速度：⭐⭐⭐⭐⭐ 即时（暴食龙系技能全部迅捷+极速253）

## 三、属性抗性（翼/龙双属性）

- **3x弱点**：冰（极危险）
- **2x弱点**：龙、萌（2条）
- **半抗**：火、水、虫、武、翼（5条）
- **1/3抗**：草（1条）
- **免疫**：无

## 四、特性「暴食」

> 携带的龙系技能获得迅捷。

所有携带的龙系技能都获得迅捷效果：

- **龙炮迅捷**（3能100威力）：迅捷100威力先手
- **龙爪迅捷**（4能120威力）：迅捷120威力先手
- **俯冲猛击迅捷**（5能140威力）：迅捷140威力先手
- **龙之利爪迅捷**（3能130蓄力+吸血50%）：迅捷蓄力130威力+吸血
- **龙吟迅捷**（3能蓄力双攻+100%+速度+60）：迅捷蓄力增益
- **吹炎迅捷**（3能170蓄力，应对状态翻倍）：迅捷蓄力170威力
- **核心价值**：龙系技能全部迅捷，配合极速253形成绝对先手优势
- **局限性**：冰系3x弱点致命；HP83+特防79偏低；龙/萌2x弱点

## 五、技能池亮点

**暴食迅捷龙系核心**：
- 俯冲猛击 5能140 → 迅捷140威力（最高威力迅捷）
- 龙爪 4能120 → 迅捷120威力
- 龙炮 3能100 → 迅捷100威力（低费迅捷）
- 龙之利爪 3能130 → 迅捷蓄力+吸血50%
- 吹炎 3能170 → 迅捷蓄力（应对状态翻倍=340威力）
- 龙吟 3能 → 迅捷蓄力双攻+100%+速度+60

**跳过蓄力**：
- 龙息环爆 1能55 → 应对状态下次技能无蓄力（龙之利爪/吹炎直接释放）

**应对爆发**：
- 爆冲 3能65 → 应对状态5倍=325威力
- 偷袭 3能85 → 应对状态3倍=255威力
- 吹炎 3能170 → 应对状态翻倍=340威力

**增益**：
- 力量增效 1能 → 物攻+100%
- 三连破 1能 → 物攻+30%+3连击
- 伺机而动 1能 → 下次攻击+70威力
- 龙威 3能 → 龙噬印记5能技双攻+30%

**连击叠加**：
- 乘胜追击 2能40×N → 每次使用永久+1连击
- 热身运动 2能 → 连击+3

**控场**：
- 羽刃 2能75 → 应对状态强制换人

## 六、推荐配招方向

### 路线A：迅捷爆发流（收割核心）
- 俯冲猛击 + 龙炮 + 力量增效 + 龙息环爆
- 暴食让俯冲猛击/龙炮全部迅捷，力量增效+100%物攻，龙息环爆应对后龙之利爪无蓄力

### 路线B：蓄力迅捷流（炮台）
- 吹炎 + 龙吟 + 龙息环爆 + 伺机而动
- 吹炎迅捷蓄力170威力（应对状态340），龙吟迅捷蓄力双攻+100%，龙息环爆跳过蓄力

### 路线C：连击叠加流（收割）
- 乘胜追击 + 龙炮 + 力量增效 + 热身运动
- 乘胜追击永久+1连击，热身运动+3连击，龙炮迅捷100威力×N连击
