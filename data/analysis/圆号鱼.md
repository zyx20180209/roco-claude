## 四、特性「泛音列」

> 使用状态技能后，敌方获得「聒噪」技能的效果，持续3回合。

每次使用状态技都附加聒噪（敌方全攻击技能能耗+3持续3回合）：

- **与激怒联动**：激怒（除本回合使用的技能其他技能能耗+3×3回合）+泛音列聒噪（全攻击技能能耗+3×3回合）→ 全技能+6能耗
- **与操控联动**：操控（本回合使用的技能能耗+7×3回合）+泛音列聒噪（全攻击+3）→ 点名+10能耗
- **与精神扰乱联动**：精神扰乱（全技能能耗+1，应对防御+3）+泛音列聒噪（全攻击+3）→ 全技能+4~6能耗
- **与恶作剧联动**：恶作剧-3能（应对防御-6能）+泛音列聒噪使对手无法出新技能
- **打湿/盐水浴联动**（自给）：打湿/盐水浴本身也是状态技，触发泛音列聒噪同时自给降费
- **核心价值**：连锁状态技彻底锁死对手能量，对手被迫聚能而无法攻击
- **局限性**：只针对攻击能耗，状态技不受影响；聒噪持续3回合需要持续维持

## 五、核心战术：能量锁死链

**封锁流程**：
1. 激怒（+3能耗）+ 泛音列聒噪（+3能耗）→ 对手全攻击技能+6能耗
2. 操控（+7能耗点名）→ 对手核心大招被锁死
3. 恶作剧（-3~-6能）+ 勾魂（-3能）→ 直接抽空对手能量
4. 润泽（魔攻+190%）→ 水炮/涌泉爆发

## 六、技能池亮点

**能量封锁核心（每个都触发泛音列）**：
- 激怒 3能 → 敌方除本回合技能其他技能+3能耗×3回合（+泛音列聒噪=双重+6能耗）
- 操控 5能 → 本回合技能能耗+7×3回合（+聒噪=+10能耗）
- 精神扰乱 2能 → 全技能+1能耗（应对防御+3）+聒噪
- 恶作剧 2能 → 敌方-3能（应对防御-6能）+聒噪
- 勾魂 2能 → 偷3能+聒噪

**增益（触发泛音列）**：
- 润泽 7能 → 魔攻+190%+聒噪
- 落雨 5能 → 雨天水系+50%+聒噪
- 打湿 4能 → 湿润印记全技能-1能耗+聒噪
- 盐水浴 2能 → 全技能-2能耗+聒噪
- 洗礼 2能 → 驱散自己减益+全技能-1能耗+聒噪

**水系高威力输出**：
- 许愿星 6能110 → 稳定魔攻
- 水炮 6能110 → 每次使用永久-1能耗（自递减）
- 涌泉 6能60 → 能耗每-1威力+10（搭配盐水浴/打湿滚雪球）
- 灵媒 5能100 → 魔攻输出
- 肥皂泡 3能60 → 低费魔攻

**防御续航**：
- 泡沫幻影 2能 → 减伤80%+脱离
- 甩水 0能30 → 回1能（资源循环）

**印记**：
- 棘刺 2能 → 换人-6%血+聒噪

## 七、推荐配招方向
