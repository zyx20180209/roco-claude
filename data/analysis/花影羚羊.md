# 花影羚羊 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 3维 | 物攻111（66%）+碰瓷恶系技能偷能+坟场搏击敌方低能高威力 |
| 耐久 | 2维 | HP112+物防120（82%）；特防81偏脆 |
| 功能性 | 4维 | 碰瓷偷能+降灵印记+勾魂偷能+恶作剧偷能+恶意逃离中转+晒太阳驱散 |

**主定位**：功能型工兵（能量压制+降灵印记+中转）
**战术角色**：工兵 + 中转 + 破盾（能量压制逼迫拦截位下场）

## 二、种族值与速度

- HP112 / 攻111 / 特攻48 / 防120 / 特防81 / 速110（总582）
- **有效种族534**（去除特攻48）
- **极速246.8**（69%分位，中高速）/ 满速214.0
- 启动速度：⭐⭐⭐⭐⭐ 即时（勾魂/恶作剧/恶意逃离均1能即可发动）

## 三、属性抗性

- **3x弱点**：光（幽+恶双克光）
- **2x弱点**：萌
- **半抗**：普通、毒
- **免疫**：无

## 四、特性「碰瓷」

> 自己使用恶系技能后，敌方失去2能量

每次使用恶系技能自动触发，无需额外条件：

- **恶能量**（1能）→ 触发碰瓷偷2能，净能量差3能
- **报复**（防御技）→ 应对触发偷3能+碰瓷偷2能=单次偷5能
- **核心价值**：持续压低对手能量，坟场搏击（敌方每有1能威力-10%）与碰瓷形成正向循环
- **局限性**：特防81偏脆，光系3x弱点是硬伤；需持续出手恶系技能才能维持压制

## 五、技能池亮点

**能量压制核心**：
- 勾魂 1能 → 偷取敌方3能量
- 恶作剧 1能 → 敌方失去3能（应对防御失去6能）
- 碰瓷+恶能量 1能 → 偷2能，净差3能
- 报复 2能（防御）→ 应对触发偷3能+碰瓷偷2能=单次偷5能

**输出**：
- 坟场搏击 4能180威力 → 敌方每有1能威力-10%，配合压能后威力接近满值
- 极限撕裂 4能135威力 → 低血量无惩罚
- 撕咬 3能30×3连击 → 低血量时5连击，破盾利器

**控场/工兵**：
- 降灵 2能 → 降灵印记，换人入场失去1能，封锁中转节奏
- 晒太阳 1能 → 驱散敌方所有增益
- 恶意逃离 1能 → 脱离+应对防御敌方攻击技能能耗+4

## 六、推荐配招方向

### 路线A：能量压制流（工兵+破盾）
- 勾魂 + 恶能量 + 坟场搏击 + 降灵
- 勾魂+恶能量持续压能，降灵封锁换人，坟场搏击在敌方低能时爆发

### 路线B：中转控场流（工兵+中转）
- 恶作剧 + 恶意逃离 + 坟场搏击 + 晒太阳
- 恶作剧骚扰+恶意逃离中转，晒太阳驱散，坟场搏击收割

### 路线C：报复偷能流（拦截+工兵）
- 报复 + 勾魂 + 坟场搏击 + 降灵
- 报复应对触发偷5能，配合降灵印记彻底封锁对手资源

## 八、擅长应对的情况

- 高费技能池的炮台（碰瓷+勾魂持续压能，使高费技无法释放）
- 依赖中转节奏的队伍（降灵印记封锁换人入场能量）
- 依赖增益积累的对手（晒太阳1能驱散所有增益）
- 敌方低能量状态（坟场搏击威力接近满值180）

## 九、棘手的对手特点

- 光系精灵（3x弱点，几乎必死）
- 萌系精灵（2x弱点）
- 低费技能池（碰瓷压能效果有限，对手仍能正常出手）
- 特防81极脆，高特攻魔攻核心威胁大
