# 伊贝粉粉 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 2维 | 双攻91/89均衡偏中；光能聚集永久叠加+藤绞回能续战 |
| 耐久 | 4维 | 腐蚀循环每回能+5%血+蜡质膜/纤维化/无畏之心多重防御+聚盐回血回能 |
| 功能性 | 2维 | 光合印记回能+耀眼连击-4+聚盐叠加 |

**主定位**：续航型拦截（腐蚀循环回能即回血+多重防御技）
**战术角色**：拦截 + 炮台（光能聚集发育）

## 二、种族值与速度

- HP106 / 攻91 / 特攻89 / 防95 / 特防95 / 速90（总566）

- 启动速度：⭐⭐⭐⭐ 快速（腐蚀循环被动即时，种子弹0能即可出手）

## 三、属性抗性（草系）

- **2x弱点**：火、冰、毒、虫、翼（5条偏多）
- **半抗**：水、光、地、电（4条）
- **免疫**：无

## 四、特性「腐蚀循环」

> 每回复1能量，同时回复5%生命

每次回能自动触发回血：

- 种子弹（0能，回1能）→ 触发+5%血
- 藤绞（4能，回5能）→ 触发+25%血
- 蜡质膜（应对触发回3能）→ 触发+15%血
- 聚盐（每连击回1能）→ 每连击+5%血
- 光合印记（每回合回1能）→ 每回合+5%血
- **核心价值**：将能量回复转化为持续回血，形成自给自足的续航循环
- **局限性**：5条2x弱点多；双攻偏中等，输出天花板有限

## 五、叠加型技能发育分析

**识别**：光能聚集（每次使用其他草系技能后，本技能威力永久+60）+ 聚盐（每次使用连击数永久+1）

- **光能聚集**：初始100威力，每次草系技能+60。使用3次→280威力，5次→400威力。
- **聚盐**：初始2连击，每次+1。使用3次→5连击，每连击回血5%+1能（腐蚀循环触发）。
- **发育阶段风险**：速度90中速，5条弱点多，但腐蚀循环+防御技自保能力强。
- **队伍依赖度**：低。腐蚀循环自给续航，发育期基本自给自足。

## 六、推荐配招方向

### 路线A：腐蚀循环续航流（拦截）
- 蜡质膜 + 藤绞 + 聚盐 + 抽枝
- 蜡质膜应对回3能+15%血，藤绞回5能+25%血，聚盐叠连击回血，抽枝应对状态回50%血+5能

### 路线B：光能聚集发育流（炮台）
- 光能聚集 + 叶绿光束 + 藤绞 + 蜡质膜
- 叶绿光束/藤绞叠光能聚集，藤绞回能触发腐蚀循环，蜡质膜自保

### 路线C：无畏续航流（拦截）
- 无畏之心 + 酶浓度调整 + 叶绿光束 + 聚盐
- 无畏之心减伤100%+回血，酶浓度调整减伤90%+回血20%，双重续航
