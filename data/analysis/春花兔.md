- **1/3抗**：无
- **免疫**：无

## 四、特性「系统发育」

> 获得能量或生命时，会将等量的能量或生命随机分配给场下的精灵。

自己获得能量/生命时，等量随机分配给场下队友：

- **能量共享**：根吸收回4能→场下随机精灵也回4能；甜心续航回4能→场下也回4能
- **生命共享**：根吸收回15%血→场下随机精灵也回15%血
- **与富养化联动**：富养化场下每精灵+3能，系统发育不影响（富养化直接给场下）
- **与甜心续航联动**：甜心续航双方回40%血+4能，系统发育让场下也回血+能
- **与根吸收联动**：根吸收回15%血+4能，系统发育等量分配给场下
- **核心价值**：自己回复资源=队伍回复资源，支援效率翻倍
- **局限性**：随机分配（不能指定队友）；HP79偏低；5条2x弱点偏多

## 五、核心战术：资源共享+续航支援

**资源共享流**：
1. 根吸收（2能）→ 自己回15%血+4能，系统发育→场下随机精灵也回15%血+4能
2. 甜心续航（3能）→ 双方回40%血+4能，系统发育→场下也回血+能
3. 富养化（3能）→ 场下每精灵+3能（直接支援）

**赤子之心萌化流**：
1. 赤子之心（2能）→ 萌化全技能-3能耗
2. 根吸收/甜心续航低费循环
3. 系统发育持续分配资源给场下

## 六、技能池亮点

**资源共享核心（系统发育联动）**：
- 根吸收 2能 → 回15%血+4能（系统发育等量分配）
- 甜心续航 3能 → 双方回40%血+4能（系统发育分配）
- 富养化 3能 → 场下每精灵+3能（直接支援）

**输出**：
- 筛管奔流 3能80 → 高血量时+75=155威力
- 棘突 3能100 → 稳定草系魔攻
- 抽枝 4能90 → 应对状态回50%血+5能
- 汲取 1能30 → 100%吸血
- 种皮爆裂 1能25×2 → 低费2连击

**增益**：
- 赤子之心 2能 → 萌化全技能-3能耗
- 盛开 1能 → 威力+30（应对防御+60）
- 芳香诱引 1能 → 连击+3（应对防御打断+眩晕）
- 预备势 1能 → 物攻+80%（应对防御敌方物防-80%）

**防御续航**：
- 酶浓度调整 3能 → 减伤90%+应对回20%血
- 移花接木 2能 → 回15%血+脱离
- 无畏之心 5能 → 减伤100%+回血

**控场**：
- 聒噪 3能 → 敌方全攻击技能+3能耗（3回合）
- 孢子 3能 → 寄生6%/回合

## 七、推荐配招方向

### 路线A：资源共享流（工兵核心）
- 根吸收 + 富养化 + 甜心续航 + 酶浓度调整
- 系统发育根吸收/甜心续航等量分配场下，富养化场下每精灵+3能，酶浓度调整减伤90%续航

### 路线B：赤子之心萌化流（工兵）
- 赤子之心 + 根吸收 + 富养化 + 移花接木
- 赤子之心萌化全技能-3能耗，根吸收低费循环，富养化场下+3能，移花接木回血脱离

### 路线C：高血量爆发流（炮台）
- 筛管奔流 + 盛开 + 根吸收 + 酶浓度调整
- 筛管奔流高血量155威力，盛开+30威力，根吸收回血维持高血量，酶浓度调整续航
