## 四、特性「蚀刻」

> 回合结束时，敌方每2层中毒转化为1层中毒印记。

每回合结束自动将中毒转化为印记（2层中毒→1层印记）：

- **毒孢子联动**（5层中毒）：回合结束→5层中毒转2层印记（余1层中毒）
- **剧毒联动**（8层中毒）：回合结束→8层中毒转4层印记
- **与琉璃水母联动**：裘卡叠印记，琉璃水母水系技能触发=印记×2层中毒
- **感染病联动**（击败后中毒转印记）：蚀刻+感染病双重转化
- **中毒印记价值**：印记不会自然衰减（中毒每回合衰减），持续存在
- **核心价值**：将短暂的中毒DOT转化为永久印记，配合印记联动技形成持续控场
- **局限性**：HP73+物防65极脆；地/恶/幻3条弱点；双攻偏低

## 五、技能池亮点

**中毒叠层（蚀刻转印记）**：
- 毒孢子 3能 → 5层中毒（蚀刻转2层印记）
- 剧毒 2能 → 3层中毒（应对防御8层→4层印记）
- 连续毒针 2能15×2 → 每连击1层中毒
- 毒囊 2能25 → 2层中毒（应对状态6层→3层印记）
- 疫病吐息 3能 → 直接1层中毒印记

**中毒联动爆发**：
- 以毒攻毒 1能 → 中毒每层+30%魔攻
- 腐化 1能 → 中毒每层敌方双攻-30%
- 毒液渗透 5能120 → 中毒每层-1能耗
- 感染病 4能85 → 击败后中毒转印记

**控场**：
- 聒噪 3能 → 敌方全攻击技能能耗+3×3回合
- 欺诈契约 3能 → 交换增益和减益
- 晒太阳 1能 → 驱散敌方所有增益

**高威力输出**：
- 瘴气喷射 3能100 → 稳定毒系魔攻
- 超导 3能95 → 迸发-1能耗
- 电弧 3能80 → 迸发+40威力=120

**应对**：
- 阻断 2能80 → 应对状态打断
- 崩拳 2能65 → 应对状态物攻+100%
- 暗突袭 4能70 → 吸血50%+应对状态翻倍
