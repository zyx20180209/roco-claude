# 夜枭 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 3维 | 搜刮敌方聚能/换人×+20%魔攻+嘲弄魔攻+90%+幽灵爆发140威力+背袭敌方0能20倍伤害 |

**主定位**：反聚能/换人型累积爆发工兵（搜刮敌方每聚能/换人入场魔攻+20%+勾魂偷3能+降灵换人-1能）
**战术角色**：工兵 + 炮台

## 四、特性「搜刮」

> 敌方每使用1次「聚能」技能或更换精灵，自己入场时获得魔攻+20%。

敌方聚能/换人次数转化为夜枭入场时魔攻加成：

- **反聚能核心**：敌方每聚能1次=夜枭入场+20%魔攻；聚能5次=+100%魔攻
- **反换人核心**：敌方每换人1次=夜枭入场+20%魔攻；换人5次=+100%魔攻
- **与嘲弄联动**：嘲弄魔攻+90%+搜刮+N%=双重叠加
- **与幽灵爆发联动**：幽灵爆发140威力+搜刮+N%+嘲弄+90%=极限爆发
- **与背袭联动**：勾魂偷3能→敌方0能→背袭20倍伤害（能量清零流）
- **核心价值**：天然克制聚能发育型和换人频繁型对手，对手操作越多夜枭越强
- **局限性**：特攻61偏低，即使+100%也仅中等；速度80偏慢；需要积累才能爆发

## 五、核心战术：搜刮积累+能量清零

**搜刮积累流**：
1. 等待敌方聚能/换人积累层数
2. 嘲弄（2能）→ 魔攻+90%
3. 幽灵爆发（5能140）→ 搜刮+N%+嘲弄+90%爆发

**能量清零流**：
1. 勾魂（1能）→ 偷敌方3能量
2. 降灵（2能）→ 换人-1能印记
3. 背袭（2能40）→ 敌方0能时20倍=800威力

## 六、技能池亮点

**核心控场**：
- 勾魂 1能 → 偷敌方3能量（低费能量控制）
- 降灵 2能 → 换人-1能印记（持续消耗）
- 精神扰乱 0能 → 敌方全技能+1能耗（0能控场）
- 伪造账单 1能 → 敌方回血变失血×2（先手+1）

**爆发输出**：
- 幽灵爆发 5能140 → 顶级幽系魔攻
- 背袭 2能40 → 敌方0能时20倍=800威力
- 音爆 4能130 → 稳定普通系魔攻
- 灵媒 3能100 → 稳定幽系魔攻
- 灵光 3能25×3 → 敌方换人连击翻倍

**增益**：
- 嘲弄 2能 → 魔攻+90%（敌方换人+速度+70）
- 阳火增辉 3能75 → 击败后威力永久翻倍（成长）

**火系辅助**：
- 烈焰风暴 4能75 → 6层灼烧
- 易燃物质 3能30×2 → 每次+2层灼烧
- 持续高温 2能70 → 应对状态下次威力翻倍

## 七、推荐配招方向

### 路线A：搜刮爆发流（炮台核心）
- 嘲弄 + 幽灵爆发 + 勾魂 + 精神扰乱

### 路线B：能量清零流（工兵）
- 勾魂 + 降灵 + 背袭 + 精神扰乱
- 勾魂偷3能+降灵换人-1能+精神扰乱+1能耗，背袭敌方0能20倍=800威力

### 路线C：换人惩罚流（工兵）
- 降灵 + 灵光 + 嘲弄 + 伪造账单
