# 音速犬 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 4维 | 物攻128（87%）+专注力入场+100%物攻+怒火双攻+120%+流星火雨击败永久+75 |
| 耐久 | 2维 | 物防101（56%）+防御技；特防82偏脆 |
| 功能性 | 2维 | 焚烧烙印驱散印记+引燃灼烧+高温回火中转+晒太阳驱散 |

**主定位**：高速物攻收割（专注力入场即+100%物攻+极速260）
**战术角色**：收割 + 清场

## 二、种族值与速度

- HP85 / 攻128 / 特攻46 / 防101 / 特防82 / 速120（总562）
- **有效种族516**（去除特攻46）
- **极速260.0**（88%分位，顶级高速）/ 满速225.0
- 启动速度：⭐⭐⭐⭐⭐ 即时（专注力入场即触发，火苗0能即可出手）

## 三、属性抗性（火系）

- **2x弱点**：水、地（仅2条，极少）
- **半抗**：草、冰、虫、萌、机械（5条）
- **免疫**：无

## 四、特性「专注力」

> 入场首回合，获得物攻+100%

每次入场第一回合自动触发：

- 入场即物攻128×2=256等效，配合怒火（双攻+120%）：物攻进一步叠加
- 配合热身（下次攻击翻倍）：专注力+热身后火云车140×2=280等效威力
- 配合闪燃（应对状态4倍）：专注力+闪燃应对状态=160威力
- **局限性**：只在入场首回合触发，被强制换场后再入场可再次触发；特防82偏脆

## 五、叠加型技能发育分析

**识别**：流星火雨（每次击败敌方，本技能威力永久+75）

- **发育曲线**：流星火雨初始75威力，每次击败+75。击败1只→150，击败2只→225，击败3只→300。
- **专注力联动**：每次换场入场都触发专注力+100%物攻，流星火雨叠加后威力极高。
- **发育阶段风险**：极速260先手优势强，发育期风险低；但特防82偏脆被魔攻威胁。
- **队伍依赖度**：低。专注力入场即触发，发育期自给自足。

## 六、推荐配招方向

### 路线A：专注力爆发流（收割）
- 怒火 + 火云车 + 灼伤 + 热身
- 专注力+怒火双攻+120%，热身后火云车280等效威力

### 路线B：流星火雨发育流（收割）
- 流星火雨 + 怒火 + 灼伤 + 高温回火
- 流星火雨叠加后威力极高，高温回火中转再次触发专注力

### 路线C：焚烧烙印清场流（清场+收割）
- 焚烧烙印 + 怒火 + 火云车 + 晒太阳
- 焚烧烙印驱散印记+灼烧，晒太阳驱散增益，怒火+火云车收割

## 八、擅长应对的情况

- 草/冰/虫/萌/机械系对手（5条半抗+火系克制）
- 印记体系（焚烧烙印驱散所有印记+灼烧）
- 依赖增益积累的对手（晒太阳驱散）
- 快节奏爆发对局（极速260先手压制）

## 九、棘手的对手特点

- 水系高速核心（水2x克+速度可能超越）
- 地系物攻爆发（地2x克）
- 高特攻魔攻核心（特防82偏脆）
- 专注力只在首回合触发，被控制后续回合物攻下降

## 十、队伍需求

### 精灵缺的
- 水/地系克星清理
- 特殊面保护（特防82偏脆）

### 精灵给的
- 顶级高速物攻爆发（极速260+专注力入场+100%）
- 焚烧烙印清场（驱散印记+灼烧）
- 流星火雨后期无解
- 高温回火中转（再次触发专注力）
- 火系克机械联防

### 体系归属
- 核心队主C
- 反印记针对队（焚烧烙印）
- 高速压制队

## 十一、首领化形态：风暴战犬

**种族值**：HP85 / 攻128 / 特攻46 / 防101 / 特防82 / 速120（与音速犬完全相同）
**有效种族**：516
**特性「全神贯注」**：入场时获得物攻+100%，每次行动后-20%

### 专注力 vs 全神贯注

| 对比 | 音速犬（专注力） | 风暴战犬（全神贯注） |
|------|------|------|
| 触发时机 | 入场首回合 | 入场时即触发 |
| 第1次行动 | +100% | +100% |
| 第2次行动 | 0% | +80% |
| 第3次行动 | 0% | +60% |
| 第4次行动 | 0% | +40% |
| 第5次行动 | 0% | +20% |

**全神贯注的质变**：专注力只有首回合+100%，之后归零；全神贯注持续衰减但多回合保持增益，配合高费技能（火云车5能）在第2-3次行动仍有+80%/+60%物攻加成，实战输出更稳定。
