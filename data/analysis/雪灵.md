# 雪灵 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |

| 攻击力度 | 3维 | 双攻112均衡+冰封能耗压制放大冰锋横扫+冰冻光线7能90+2能耗 |
| 耐久 | 2维 | 双防105+HP94；均衡面板 |
| 功能性 | 5维 | 冰封全场能耗压制+冰捆缚+1能耗+冰冻光线+2能耗+瞬间零度+3能耗+聒噪/操控/摇篮曲多重能耗工具 |

**主定位**：全场能耗压制核心（冰封光环+多重能耗压制技+冰锋横扫能耗×10爆发）
**战术角色**：工兵 + 炮台

## 二、种族值与速度

- HP94 / 攻112 / 特攻112 / 防105 / 特防105 / 速115（总643）
- **有效种族531**（去除特攻112）
- **极速253.4**（79%分位，中高速）/ 满速219.5
- 启动速度：⭐⭐⭐⭐⭐ 即时（冰封在场即生效，冰锥0能即可出手）

## 三、属性抗性（冰系）

- **2x弱点**：火、地、武、机械（4条）
- **半抗**：水、光、冰（3条）
- **免疫**：无

## 四、特性「冰封」

> 在场时，敌方全技能能耗+1

持续在场即持续压制对手能耗：

- 对手全技能能耗+1，配合冰捆缚（+1能耗）/冰冻光线（+2能耗）/瞬间零度（+3能耗）/聒噪（+3能耗）/操控（+7能耗）/摇篮曲（+3能耗）形成**全游戏最强能耗压制链**
- 配合冰锋横扫（威力=敌方技能总能耗×10）：对手能耗越高，冰锋横扫威力越高
- **局限性**：只在场时生效，下场后失效；4条2x弱点偏多

## 五、核心战术：冰锋横扫威力计算

**冰锋横扫联动**（敌方技能能耗总和×10）：
- 敌方4技能默认能耗总和假设为8能 → 冰锋横扫80威力
- 冰封+1能耗/技能=+4能 → 总和12能 → 冰锋横扫120威力
- +冰捆缚2次每次+1能耗=+8能 → 总和20能 → 冰锋横扫200威力
- +聒噪+3能耗（3技能）=+9能 → 总和29能 → 冰锋横扫290威力
- +操控+7能耗=+7能 → 总和36能 → **冰锋横扫360威力**

## 六、技能池亮点

**能耗压制链（核心）**：
- 冰封（在场被动）→ 敌方全技能能耗+1
- 冰捆缚 3能 → 2连击，每次连击敌方全技能能耗+1
- 冰冻光线 7能90 → 敌方全技能能耗+2
- 瞬间零度 0能 → 本回合敌方技能能耗+3
- 聒噪 3能 → 敌方攻击技能耗+3持续3回合
- 操控 1能 → 敌方本回合技能能耗+7
- 摇篮曲 5能 → 敌方能耗+3+眩晕
- 激怒 3能 → 敌方其他技能能耗+3持续3回合

**输出**：
- 冰锋横扫 4能0 → 敌方技能总能耗×10（能耗链叠满后威力极高）
- 冰冻光线 7能90 → 稳定魔攻+能耗压制
- 冰锥 0能40 → 低费骚扰

**能量工具**：
- 雾气环绕 1能 → 回能=敌方技能能耗的一半（敌方能耗越高自己回越多）
- 雪替身 2能 → 减伤70%+回能=被应对能耗×2
- 勾魂 1能 → 偷3能

**防御**：
- 雪替身 2能 → 减伤70%+回能
- 冰墙 2能 → 减伤80%+2层冻结

**控场**：
- 雪球 1能 → 敌方速度-90

## 七、推荐配招方向

### 路线A：能耗压制爆发流（工兵+炮台，核心玩法）
- 冰封 + 聒噪 + 冰捆缚 + 冰锋横扫
- 冰封+聒噪+冰捆缚叠能耗后冰锋横扫200-300+威力

### 路线B：操控极限爆发流
- 操控 + 冰封 + 冰捆缚 + 冰锋横扫
- 操控+7能耗+冰封+1+冰捆缚×2=+10能耗，冰锋横扫300+威力

### 路线C：瞬间零度低费流
- 瞬间零度 + 冰捆缚 + 冰冻光线 + 冰锋横扫
- 瞬间零度0能+3能耗，冰捆缚每连击+1，冰冻光线+2，冰锋横扫爆发
