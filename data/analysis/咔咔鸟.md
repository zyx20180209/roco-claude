## 四、特性「咔咔冲刺」

> 若先于敌方行动，行动后获得连击数+1

先手行动后触发连击+1：

- 配合极速260先手：几乎每次都能先手触发
- 配合连击技（啄击2连击/旋转突击3连击/疾风刺3连击/连续爪击2连击）：每次连击数都+1
- 配合暴风眼（连击数+100%）：连击数先+1再翻倍
- 配合热身运动（+3连击）：叠加后连击数极高
- **核心价值**：高速先手每次出手连击都在增长，配合连击技爆发极强
- **局限性**：必须先手触发；遇到更高速或先手技敌手时失效

## 五、叠加型技能发育分析

**识别**：乘胜追击（每次使用连击数永久+1）+ 聚盐（每次使用连击数永久+1）

- **乘胜追击**：初始1连击，每次+1。配合咔咔冲刺先手+1，使用3次后→乘胜追击+咔咔冲刺=5连击
- **聚盐**：每连击回血5%+1能，自给续航
- **队伍依赖度**：低。极速260先手+咔咔冲刺自给自足。

## 六、技能池亮点

**连击核心**：
- 啄击 0能15×2 → 0能连击触发咔咔冲刺（+1变3连击）+暴风眼（6连击）
- 疾风刺 2能25 → 先手时3连击（+咔咔冲刺=4连击）
- 旋转突击 3能35×3 → 3连击+咔咔冲刺=4连击
- 连续爪击 2能30×2 → 应对状态连击翻倍，+咔咔冲刺=5连击
- 乘胜追击 2能40 → 永久+1连击（发育型）

**暴风眼联动**：
- 暴风眼 2能 → 行动时连击数+100%（翻倍）
- 配合疾风刺/旋转突击/连续爪击，连击数爆炸性增长

**其他输出**：
- 猛烈撞击 1能65 → 稳定低费
- 回旋风暴 3能100 → 翼系魔攻
- 叶绿光束 4能120 → 高威力
- 龙卷风 5能70 → 迅捷+应对状态1.5倍

**增益**：
- 热身运动 2能 → 连击+3
- 三连破 1能 → 物攻+30%+3连击
- 花炮 2能 → 2连击+每连击魔攻+60%
- 快速移动 1能 → 速度+80（应对防御+160）

**功能**：
- 羽刃 2能75 → 应对状态强制换人
- 羽化加速 2能 → 迅捷+威力+20
