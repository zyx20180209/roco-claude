# 熔岩布丁 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 3维 | 物攻87+怒火双攻+120%+淬火应对下次攻击威力翻倍+吹火每次使用永久+20威力+水炮能耗成长 |
| 耐久 | 4维 | HP128（84%）+物防112（75%）+火焰护盾减伤70%+应对敌方6层灼烧+捧杀减伤90%+应对萌化 |
| 功能性 | 5维 | 美拉德反应离场队友双攻+20%免疫灼烧+蓄水-6能耗+洗礼驱减益+水环应对-2能耗+高温回火自脱离 |

**主定位**：中转攻势支援型物理炮台（美拉德反应离场队友双攻+20%+免疫灼烧+HP128高耐久+极速260高速）
**战术角色**：中转 + 炮台

## 二、种族值与速度

- HP128 / 攻87 / 特攻33 / 防112 / 特防84 / 速120（总564）
- **有效种族531**（去除特攻33）
- **极速260.0**（88%分位，高速）/ 满速225.0
- 启动速度：⭐⭐⭐⭐⭐ 极快（极速260高速先手+怒火1能双攻+120%爆发）

## 三、属性抗性（水/火双属性）

- **3x弱点**：无
- **2x弱点**：水、地、电（3条）
- **半抗**：火、冰、虫、萌（4条）
- **1/3抗**：机械（1条）
- **免疫**：无

## 四、特性「美拉德反应」

> 离场后，更换入场的精灵获得双攻+20%且免疫灼烧。

离场后下个入场精灵双攻+20%+免疫灼烧：

- **中转攻势支援**：区别于抹茶布丁（回20%血），美拉德反应是攻势支援+20%双攻
- **与泡沫幻影联动**：泡沫幻影减伤80%+应对自脱离，触发美拉德反应让队友入场双攻+20%
- **与高温回火联动**：高温回火2能40威力+自己脱离（主动触发）
- **免疫灼烧**：下个入场精灵免疫灼烧，破除火焰护盾/灼烧体系压制
- **战术节奏**：作为中转位，进场释放能耗/输出工具后主动撤下让队友加攻入场
- **核心价值**：队伍级别攻势增益+免疫灼烧，支援爆发体系强
- **局限性**：需要主动离场才触发；特攻33极低不能魔攻流；3条2x弱点

## 五、核心战术：中转攻势+能耗降低

**支援流程**：
1. 蓄水（1能）→ 下次技能-6能耗
2. 水炮/天洪（5-7能）→ 蓄水后低费高威力释放
3. 高温回火（2能40）→ 2能魔伤+自己脱离
4. 美拉德反应触发→ 队友入场双攻+20%+免疫灼烧

**怒火爆发流**：
1. 怒火（1能）→ 双攻+120%（双防-40%）
2. 水幕冲击（3能100威力）→ 物攻+120%爆发
3. 或火焰切割（3能100威力）→ 双系别覆盖

## 六、技能池亮点

**高威力输出**：
- 天洪 7能150 → 应对状态永久-6能耗（成长爆发）
- 水炮 5能110 → 每次使用永久-1能耗（成长）
- 水刃 4能115 → 应对状态永久-4能耗
- 水幕冲击 3能100 → 稳定水系物攻
- 火焰切割 3能100 → 稳定火系物攻
- 超级糖果 3能100 → 萌化自身+本次威力+60=160

**低费高效**：
- 肥皂泡 1能60 → 低费水系魔攻
- 泡沫 1能60 → 低费水系物攻
- 吹火 1能50 → 每次使用永久+20威力（成长）
- 潮涌 2能80 → 低费水系物攻

**增益**：
- 怒火 1能 → 双攻+120%（双防-40%）
- 淬火 4能 → 减伤80%+应对下次威力×2
- 蓄水 1能 → 下次技能-6能耗

**防御**：
- 火焰护盾 2能 → 减伤70%+应对敌方6层灼烧
- 泡沫幻影 2能 → 减伤80%+应对自脱离（触发美拉德反应）
- 水环 2能 → 减伤60%+应对全技能-2能耗
- 捧杀 4能 → 减伤90%+应对敌方萌化

**资源**：
- 高温回火 2能40 → 主动脱离（触发美拉德反应）
- 洗礼 1能 → 驱减益+全技能-1能耗

## 七、推荐配招方向

### 路线A：中转支援流（中转核心）
- 怒火 + 水幕冲击 + 泡沫幻影 + 火焰护盾
- 怒火双攻+120%，水幕冲击100威力爆发，泡沫幻影应对自脱离触发美拉德反应，火焰护盾反灼烧

### 路线B：蓄水成长流（炮台）
- 蓄水 + 水炮 + 吹火 + 高温回火
- 蓄水-6能耗→水炮110威力+永久-1能耗，吹火低费成长，高温回火主动脱离

### 路线C：应对爆发流（拦截）
- 淬火 + 火焰切割 + 火焰护盾 + 泡沫幻影
- 淬火应对下次威力×2，火焰切割100威力→200等效，火焰护盾反灼烧
