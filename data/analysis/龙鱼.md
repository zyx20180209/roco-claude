## 四、特性「洄游」

> 每次进入蓄力状态，获得全技能能耗永久-1。

每次使用蓄力技能时全技能永久-1能耗：

- **升龙咆哮联动**（3能200威力蓄力）：每次蓄力-1能耗，蓄力3次=全技能-3能耗
- **龙吟联动**（3能蓄力双攻+100%+速度+60）：蓄力增益同时降费
- **龙之利爪联动**（3能130蓄力+吸血50%）：蓄力降费+吸血续航
- **怨力打击联动**（3能蓄力，被攻击=敌方技能威力×3）：蓄力降费+反击
- **天洪联动**（7能150，应对状态永久-6能耗）：洄游-N+天洪应对-6=天洪极低费
- **水炮联动**（每次使用永久-1能耗）：洄游-N+水炮自递减=双重降费
- **核心价值**：每次蓄力=全技能永久-1能耗，多次蓄力后所有技能极低费
- **局限性**：HP75偏低；冰/龙/萌3条弱点；需要蓄力积累

## 五、核心战术：蓄力降费滚雪球

**降费流程**：
1. 龙吟（蓄力）→ 洄游全技能-1能耗（累计-1）
2. 升龙咆哮（蓄力）→ 洄游全技能-1能耗（累计-2）
3. 龙之利爪（蓄力）→ 洄游全技能-1能耗（累计-3）
4. 全技能-3能耗后：升龙咆哮0能200威力，天洪4能150威力

## 六、技能池亮点

**蓄力核心（洄游触发）**：
- 升龙咆哮 3能200 → 蓄力顶级威力（洄游-N后低费）
- 龙吟 3能 → 蓄力双攻+100%+速度+60（洄游-N）
- 龙之利爪 3能130 → 蓄力+吸血50%（洄游-N）
- 怨力打击 3能1 → 蓄力被攻击=敌方技能威力×3（洄游-N）

**跳过蓄力**：
- 架势 2能 → 回血20%+下次技能无蓄力
- 龙息环爆 1能55 → 应对状态下次技能无蓄力

**高威力输出**：
- 升龙咆哮 3能200 → 顶级威力
- 天洪 7能150 → 应对状态永久-6能耗（洄游-N后极低费）
- 吞噬 6能150 → 击败后回6能
- 水刃 4能115 → 应对状态永久-4能耗
- 水炮 5能110 → 每次使用永久-1能耗

**增益**：
- 魔法增效 0能 → 魔攻+70%
- 力量增效 1能 → 物攻+100%
- 润泽 7能 → 魔攻+190%（洄游-N后低费）

**先手/速度**：
- 极速267顶级先手（95%分位）
- 扇风 3能75 → 先手+50%威力
- 龙卷风 5能70 → 迅捷+应对状态1.5倍
- 翼击 3能50 → 迅捷

**控场**：
- 咆哮 1能 → 敌方物攻-130%

## 七、推荐配招方向

### 路线A：蓄力降费爆发流（炮台核心）
- 升龙咆哮 + 龙吟 + 架势 + 魔法增效
- 龙吟蓄力洄游-1，架势下次无蓄力，升龙咆哮直接200威力；多次蓄力后升龙咆哮0能200威力

### 路线B：天洪降费流（炮台）
- 天洪 + 升龙咆哮 + 龙之利爪 + 架势
- 蓄力3次洄游-3能耗，天洪应对-6=天洪1能150威力，升龙咆哮0能200威力

### 路线C：水系降费流（炮台）
- 水刃 + 水炮 + 天洪 + 龙吟
- 水刃应对永久-4+水炮永久-1+洄游-N，天洪极低费150威力，龙吟蓄力双攻+100%
