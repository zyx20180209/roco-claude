- **免疫**：无

## 四、特性「生长」

> 回合结束时，回复12%生命。

每回合结束自动回复12%生命：

- **持续回血**：每回合12%=约13血（HP108），消耗战中极难被磨死
- **与根吸收联动**（回血15%+4能）：生长12%+根吸收15%=单回合回血27%
- **与孢子联动**（寄生6%/回合）：孢子寄生6%+生长12%=净回血6%（对手每回合-6%，自己+12%）
- **与抽枝联动**（应对状态回血50%+5能）：应对触发后回血50%+生长12%=极强续航
- **核心价值**：消耗战中每回合自动回血，配合其他回血技形成极强续航
- **局限性**：速度60极慢；5条2x弱点偏多；草系精灵免疫孢子寄生

## 五、技能池亮点

**生长续航联动**：
- 根吸收 2能 → 回血15%+4能（生长+根吸收=单回合27%回血）
- 孢子 3能 → 寄生6%/回合（生长12%-寄生6%=净+6%/回合）
- 抽枝 4能90 → 应对状态回血50%+5能

**光能聚集叠加型技能**：
- 光能聚集 7能100 → 每使用其他草系技永久+60威力（叠加后极高威力）
- 棘突 3能100 → 草系魔攻（触发光能聚集+60）
- 叶绿光束 4能120 → 草系魔攻（触发光能聚集+60）
- 瘴气喷射 3能100 → 草系魔攻（触发光能聚集+60）

**花炮叠加**：
- 花炮 2能 → 2连击每次+60%魔攻（2次=+120%魔攻）

**高威力输出**：
- 叶绿光束 4能120 → 顶级草系魔攻
- 棘突 3能100 → 稳定草系
- 瘴气喷射 3能100 → 稳定草系

**控场**：
- 摇篮曲 5能 → 能耗+3+应对防御打断+眩晕
- 晒太阳 1能 → 驱散敌方所有增益
- 孢子 3能 → 寄生6%/回合

**增益**：
- 魔法增效 0能 → 魔攻+70%
- 盛开 1能 → 威力+30（应对防御+60）
- 漫反射 1能 → 每系别至多一个技能+35威力
