# 幻影灵菇 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 2维 | 双攻98均衡+惊吓盒子应对状态偷6能+坟场搏击低能高威力 |
| 耐久 | 3维 | 特防136（95%分位顶级）+酶浓度调整减伤90%+无畏之心减伤100% |
| 功能性 | 4维 | 毒蘑菇每回合偷能+降灵印记+报复偷能+嘲弄魔攻增益+取念复制技能 |

**主定位**：功能型工兵（毒蘑菇持续偷能+顶级特防联防）
**战术角色**：工兵 + 拦截（特防型）+ 破盾（能量压制）

## 二、种族值与速度

- HP111 / 攻98 / 特攻98 / 防103 / 特防136 / 速75（总621）
- **有效种族523**（双攻均等，去除特攻98）
- **极速200.6**（19%分位，偏慢）/ 满速175.5
- 启动速度：⭐⭐⭐⭐ 快速（毒蘑菇被动触发，鬼火/种子弹0能即可出手）

## 三、属性抗性（幽/草双属性）

- **2x弱点**：火、冰、翼、幽、恶（5条偏多）
- **半抗**：普通、水、地、电、武（5条）
- **免疫**：无

## 四、特性「毒蘑菇」

> 回合结束时，偷取敌方场上所有精灵1能量

每回合结束自动触发，无需出手：

- 持续在场即持续偷能，配合降灵印记（换人入场-1能）形成双重能量压制
- 配合报复（应对触发偷3能）+惊吓盒子（应对状态偷6能）= 单回合最多偷10能
- 配合坟场搏击（敌方每有1能威力-10%）：毒蘑菇压能后坟场搏击威力接近满值180
- **局限性**：速度75偏慢，5条2x弱点多，容易被针对；偷能效果需要持续在场才能积累

## 五、叠加型技能发育分析

**识别**：孢子爆散（每次使用连击数永久+2）

- **发育曲线**：孢子爆散初始1连击，每次使用+2。使用3次→7连击，5次→11连击，配合热身运动（连击数+3）后期连击数极高。
- **发育阶段风险**：速度75偏慢，发育期容易被先手压制；但酶浓度调整/无畏之心可自保。
- **队伍依赖度**：中等。孢子爆散发育路线需要多次安全出手，但毒蘑菇被动偷能不需要出手，发育期仍有贡献。

## 六、推荐配招方向

### 路线A：能量压制流（工兵+破盾）
- 报复 + 惊吓盒子 + 坟场搏击 + 降灵
- 报复偷3能+毒蘑菇每回合偷1能，降灵封锁换人，坟场搏击低能收割

### 路线B：特防联防流（拦截）
- 酶浓度调整 + 无畏之心 + 叶绿光束 + 嘲弄
- 酶浓度调整+无畏之心双重续航，嘲弄魔攻+90%后叶绿光束爆发

### 路线C：孢子发育流（炮台）
- 孢子爆散 + 热身运动 + 坟场搏击 + 酶浓度调整
- 孢子爆散叠加连击数，热身运动+3，后期高连击破盾

## 八、擅长应对的情况

- 高费技能池（毒蘑菇持续压能，使高费技无法释放）
- 依赖中转节奏（降灵印记封锁换人）
- 依赖特攻输出（特防136能扛）
- 消耗战节奏（毒蘑菇积累优势越久越大）

## 九、棘手的对手特点

- 低费技能池（毒蘑菇压能效果有限）
- 物防103中等，高物攻爆发仍有威胁
