- **2x弱点**：火、地、武、机械（4条）
- **半抗**：水、光、冰（3条）
- **免疫**：无

## 四、特性「加个雪球」

> 使敌方获得冻结时，也会使其获得2层冻结。

每次给敌方施加冻结时额外+2层：

- **霜降联动**（4层冻结）：加个雪球+2=6层冻结
- **冰点联动**（5层冻结）：加个雪球+2=7层冻结（应对防御10+2=12层）
- **暴风雪联动**（1层冻结）：加个雪球+2=3层冻结
- **极寒领域联动**（应对状态冻结翻倍）：先叠层后翻倍，加个雪球让基础层数更高
- **霜天联动**（冻结每层能耗+1）：加个雪球让冻结层数更多=能耗封锁更强
- **碎冰冰联动**（冻结每层+20威力）：加个雪球+2层=碎冰冰额外+40威力
- **核心价值**：所有冻结技能效果放大，冻结叠层速度极快
- **局限性**：冰系精灵免疫冻结；物防70偏低；4条2x弱点

## 五、技能池亮点

**冻结叠层核心（加个雪球放大）**：
- 冰点 2能 → 5层冻结+加个雪球=7层（应对防御12层）
- 霜降 1能 → 4层冻结+加个雪球=6层
- 暴风雪 3能85 → 1层冻结+加个雪球=3层+物攻
- 霜天 4能 → 1层冻结+加个雪球=3层+每层能耗+1

**冻结联动爆发**：
- 极寒领域 6能105 → 冻结时+60威力=165，应对状态冻结翻倍
- 碎冰冰 3能40 → 冻结每层+20威力（7层=40+140=180威力）
- 冰锋横扫 4能 → 威力=敌方技能总能耗×10（霜天叠层后极高）

**能耗控场**：
- 霜天 4能 → 冻结每层能耗+1（7层=+7能耗）
- 冰冻光线 7能90 → 能耗+2
- 冰晶坠 4能90 → 能耗+1
- 精神扰乱 0能 → 能耗+1（应对防御+3）
- 勾魂 1能 → 偷3能

**高威力输出**：
- 幽灵爆发 5能140 → 顶级魔攻
- 冰雹 4能105 → 应对状态能耗+3
- 超级糖果 3能100 → 萌化威力+60=160
- 双星 3能100 → 稳定物攻

**增益**：
- 魔法增效 0能 → 魔攻+70%
- 力量增效 1能 → 物攻+100%
- 生日蛋糕 4能 → 驱减益+增益翻倍

## 六、推荐配招方向

### 路线A：冻结叠层爆发流（炮台核心）
- 冰点 + 极寒领域 + 碎冰冰 + 霜天

- 霜降6层+霜天每层+1能耗=+6能耗，精神扰乱+1，冰锋横扫威力=总能耗×10爆发
