## 四、特性「消波块」

> 每携带1个水系技能进入战斗，地系技能能耗-1。

携带水系技能数量直接降低地系技能费用：

- **携带1个水系技能**：地系技能能耗-1
- **携带2个水系技能**：地系技能能耗-2
- **携带3个水系技能**：地系技能能耗-3
- **抛石联动**（回合结束永久-5能耗）：消波块-2~3+抛石自身递减，快速达到低费
- **岩土暴击联动**（被攻击永久-1能耗）：拦截位被攻击叠加降费
- **沙涌联动**（沙暴地系能耗减半）：沙暴+消波块双重降费，地系技能极低费
- **核心价值**：构筑时携带2-3个水系技能，地系大招低费释放
- **局限性**：速度70偏慢；草系3x弱点致命；需要构筑水系技能占用技能栏

## 五、技能池亮点

**降费核心**：
- 消波块（被动）→ 水系技能数×地系能耗-1
- 沙涌 3能 → 沙暴天气地系能耗减半（持续8回合）
- 蓄水 3能 → 下次技能能耗-6（一次性大降费）
- 抛石 8能140 → 回合结束永久-5能耗（自递减）
- 岩土暴击 7能140 → 被攻击永久-1能耗（拦截叠加）

**高威力地系输出**：
- 抛石 8能140 → 永久-5能耗后快速低费
- 岩土暴击 7能140 → 被攻击-1能耗
- 践踏 6能130 → 稳定地系
- 地陷 4能80 → 应对状态威力翻倍+物防+70%

**水系触发源（消波块）**：
- 水刃 5能115 → 应对状态永久-4能耗（双重降费）
- 水光冲击 7能140 → 高威力魔攻
- 气泡 5能100 → 稳定魔攻
- 水环 3能 → 减伤60%+应对全技能-2能耗

**防御续航**：
- 嗜痛 2能 → 减伤80%+应对双攻+40%
- 水环 3能 → 减伤60%+应对全技能-2能耗
- 硬化 4能 → 减伤90%（上回合攻击后-2能耗）
- 无畏之心 4能 → 减伤100%+回血（能耗永久+2）

**控场**：
- 棘刺 2能 → 棘刺印记换人-6%血
- 退化 1能 → 敌方萌化

## 六、推荐配招方向
