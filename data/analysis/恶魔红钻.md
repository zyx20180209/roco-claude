## 四、特性「振奋虫心」

> 主动击败敌方后，己方队伍获得5次随机奉献。

主动击败敌方精灵后全队获得5次随机奉献：

- **奉献机制**：随机奉献包括吸血/威力+20/连击+1/能耗-1等多种效果
- **与虫茧联动**（奉献：10%吸血）：振奋虫心触发后虫茧奉献叠加，全队吸血
- **与飞断联动**（奉献：威力+20）：振奋虫心触发后飞断奉献叠加，全队威力+20
- **连续击败滚雪球**：每次击败=5次奉献，连续击败=奉献叠加
- **核心价值**：击败敌方后全队获得5次随机增益，雪球效应极强
- **局限性**：需要主动击败（不是被动）；随机奉献不可控；5条2x弱点偏多；特防74偏低

## 五、技能池亮点

**击败触发核心**：
- 极限撕裂 4能135 → 高威力物攻（生命>50%时使用后双攻-50%）
- 虫击 3能90 → 应对状态2倍=180威力+无视系别抵抗
- 偷袭 3能85 → 应对状态3倍=255威力
- 跌落 3能120 → 应对状态物攻+50%（否则-50%）
- 以重制重 3能 → 敌方体重越高威力越高

**奉献联动**：
- 虫茧 1能 → 回血20%+奉献10%吸血（振奋虫心触发后叠加）
- 飞断 1能20 → 奉献威力+20（振奋虫心触发后叠加）

**破盾/增益**：
- 锐利眼神 2能 → 敌方双防-120%（团队破盾）
- 破绽 1能 → 敌方双防-70%+应对防御物攻+70%
- 力量增效 1能 → 物攻+100%
- 力量吞噬 4能 → 敌方威力-20+自己全技能+20
- 提气 4能 → 全技能威力+40（换人+50）

**特殊技**：
- 灾厄 1能150 → 对自己造成物伤（应对状态改为对敌方）
- 暗箱操作 1能 → 自己双攻双防-100%（应对防御改为敌方-100%）
- 等价交换 3能 → 减伤90%+应对50%吸血

**控场**：
- 伪造账单 1能 → 敌方回血改为失去2倍+先手+1
- 当头棒喝 3能80 → 换人时威力+100=180
- 草虫冲击 3能80 → 换人时威力+50+无视系别抵抗
