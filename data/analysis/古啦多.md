## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 3维 | 双攻88均衡+毒牙中毒时魔攻魔防-40%+鸩毒每层中毒+10威力+以毒攻毒每层魔攻+30% |
| 耐久 | 4维 | 物防136（95%顶级）+特防110（71%）+不可接触中毒层减伤+壁垒减伤90%+石肤术物防+160% |

**主定位**：毒系联防拦截（毒牙中毒附带魔攻魔防-40%+物防136顶级）
**战术角色**：拦截 + 破盾（毒系DOT）

## 二、种族值与速度

- HP79 / 攻88 / 特攻88 / 防136 / 特防110 / 速90（总591）

## 三、属性抗性（地/毒双属性）

- **2x弱点**：水、地、冰、恶、机械、幻（6条偏多）
- **半抗**：普通、火、电、虫、翼、萌（6条）
- **免疫**：无

## 四、特性「毒牙」

> 使敌方获得中毒时，也会使其获得魔攻和魔防-40%

每次给对手施加中毒时自动触发：

- 腐蚀酸液（2层中毒）→ 触发毒牙魔攻魔防-40%
- 毒囊（2-6层中毒）→ 触发毒牙魔攻魔防-40%
- 毒孢子（5层中毒）→ 触发毒牙魔攻魔防-40%
- **核心价值**：中毒不仅是DOT，还附带魔攻魔防-40%，克制特攻型主C
- 配合以毒攻毒（每层中毒魔攻+30%）：5层中毒→自己魔攻+150%
- 配合鸩毒（每层中毒+10威力）：5层中毒→鸩毒75+50=125威力
- **局限性**：6条2x弱点偏多；HP79偏脆；毒系精灵免疫中毒

## 五、技能池亮点

**毒牙联动**：
- 腐蚀酸液 2能35 → 2层中毒+毒牙魔攻魔防-40%
- 毒囊 2能25 → 2层中毒（应对状态6层）+毒牙
- 毒孢子 3能 → 5层中毒+毒牙
- 以毒攻毒 1能 → 每层中毒魔攻+30%（5层后+150%）
- 鸩毒 3能75 → 每层中毒+10威力（5层后125威力）
- 落井下毒 6能 → 减益层数翻倍（中毒翻倍）

**防御核心**：
- 不可接触 1能 → 减伤50%+每层中毒+10%减伤（5层后减伤100%）
- 壁垒 2能 → 减伤90%+防御技CD-1
- 石肤术 3能 → 物防+160%+魔防-60%
- 硬化 2能 → 减伤90%（攻击后-2能耗）

**控场**：
- 石锁 3能50 → 禁换3回合
- 激怒 3能 → 敌方其他技能能耗+3持续3回合
- 倾泻 3能70 → 未被应对驱散双方印记

## 六、推荐配招方向

### 路线A：毒牙破盾流（拦截+破盾）
- 腐蚀酸液 + 以毒攻毒 + 鸩毒 + 不可接触
- 腐蚀酸液触发毒牙-40%魔攻魔防，以毒攻毒+150%魔攻，鸩毒125威力终结

### 路线B：毒孢子爆发流（破盾）
- 毒孢子 + 以毒攻毒 + 鸩毒 + 壁垒
- 毒孢子5层+毒牙-40%，以毒攻毒+150%魔攻，鸩毒125威力

### 路线C：石锁联防流（拦截+工兵）
- 石锁 + 不可接触 + 腐蚀酸液 + 落井下毒
- 石锁禁换，不可接触中毒层减伤，落井下毒减益翻倍
