## 四、特性「契约的形状」

> 根据捕捉所用的咕噜球，入场时获得不同效果。

**绝缘球效果**（社区共识核心道具）：入场时速度+50+敌方获得1层中毒：

- **速度质变**：速度100+50=150，极速从233.6跃升至283.6，进入顶级速度层（超越绝大多数精灵）
- **中毒附加**：入场即给敌方1层中毒，为队伍提供持续压力
- **与飞断联动**：飞断1能产奉献（威力+20），绝缘球高速先手飞断循环
- **与假寐联动**：假寐2能回2能+奉献能耗-2，能量循环核心
- **与虫群联动**：虫群7能受奉献影响，奉献积累后爆发
- **核心价值**：道具机制将二线辅助变为顶级速度收割核心，体现道具对定位的决定性影响
- **局限性**：完全依赖绝缘球；HP81/物防76偏低脆弱；火/翼2x弱点

## 五、核心战术：奉献循环+虫群收割

**奉献循环流程**：
1. 绝缘球入场→速度+50+敌方1层中毒
2. 飞断（1能20威力）→ 队伍奉献威力+20
3. 假寐（2能）→ 回2能+队伍奉献能耗-2
4. 飞断→假寐→飞断循环积累奉献
5. 虫群（7能）→ 受奉献影响爆发收割

**虫击应对流**：
1. 掩护（2能）→ 减伤70%+应对脱离
2. 虫击（3能90）→ 应对状态威力×2=180+无视抗性

## 六、技能池亮点

**奉献核心**：
- 飞断 1能20 → 队伍奉献威力+20（低费产奉献）
- 假寐 2能 → 回2能+队伍奉献能耗-2（能量循环）
- 虫群 7能20×N → 受奉献影响爆发（收割核心）
- 虫群过境 5能45×2 → 队伍奉献连击+1

**高威力输出**：
- 虫击 3能90 → 应对状态威力×2=180+无视抗性
- 虫蛊 3能100 → 稳定虫系魔攻
- 陨石 3能100 → 地系魔攻覆盖

**防御**：
- 虫结阵 2能 → 减伤80%+应对随机奉献
- 掩护 2能 → 减伤70%+脱离（下个入场减伤）
- 壁垒 2能 → 减伤90%+应对防御冷却-1
- 硬化 2能 → 减伤90%（上回合攻击后-2能耗）

**资源**：
- 食腐 2能 → 驱印记+每层回10%血
- 乘胜追击 2能40 → 每次使用永久+1连击（成长）
