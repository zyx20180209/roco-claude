# 疾光千兽 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 4维 | 物攻146（96%）+月光审判首领血脉威力+100%+三连破物攻+30%+3连击+漫反射每系别+35+偷袭应对3倍威力 |
| 耐久 | 3维 | HP104+物防112（75%）+刺盾减伤70%+应对物攻-70%+崩拳应对物攻+100% |
| 功能性 | 3维 | 乘风速度+120+羽化加速迅捷+风隐双脱离+星轨裂变叠星陨+折射多系别联动 |

**主定位**：反首领血脉爆发型物理炮台（月光审判首领血脉威力+100%+物攻146顶级+极速247先手）
**战术角色**：炮台 + 收割

## 二、种族值与速度

- HP104 / 攻146 / 特攻67 / 防112 / 特防73 / 速110（总612）
- **有效种族545**（去除特攻67）
- **极速246.8**（69%分位，中高速）/ 满速214.0
- 启动速度：⭐⭐⭐⭐ 快速（物攻146顶级+三连破1能即可叠攻）

## 三、属性抗性（光系）

- **3x弱点**：无
- **2x弱点**：草、幽（2条，极少）
- **半抗**：恶、幻（2条）
- **1/3抗**：无
- **免疫**：无

## 四、特性「月光审判」

> 攻击时，若敌方血脉是首领血脉，技能威力+100%。

敌方携带首领血脉时攻击威力翻倍：

- **首领血脉针对**：专门克制携带首领血脉的精灵（棋契陛下/奇梦咪/伊兰龙/波普鹿等首领化体系）
- **与光刃联动**：光刃120威力+月光审判+100%=240威力等效
- **与三连破联动**：三连破物攻+30%+3连击，月光审判+100%后每击伤害极高
- **与漫反射联动**：漫反射每系别技能+35，月光审判+100%后等效更高
- **核心价值**：天然克制首领化体系，对首领血脉精灵威慑极强
- **局限性**：仅对首领血脉触发，对普通血脉无加成；特防73偏低；草/幽2x弱点

## 五、核心战术：首领血脉针对爆发

**针对首领血脉流**：
1. 三连破（1能）→ 物攻+30%+3连击
2. 漫反射（1能）→ 每系别技能+35
3. 光刃（4能120）→ 月光审判+100%=240威力等效爆发

**偷袭应对流**：
1. 刺盾（2能）→ 减伤70%+应对物攻-70%
2. 偷袭（3能85）→ 应对状态威力×3=255威力爆发

## 六、技能池亮点

**核心输出（月光审判加成）**：
- 光刃 4能120 → 光系物攻（月光审判+100%=240等效）
- 光之矛 3能30×3 → 3连击光系物攻
- 透射 1能60 → 低费光系物攻
- 偷袭 3能85 → 应对状态威力×3=255

**增益**：
- 三连破 1能 → 物攻+30%+3连击（低费核心）
- 漫反射 1能 → 每系别技能+35
- 乘风 2能 → 速度+120（极速提升）
- 羽化加速 2能 → 技能+20威力+迅捷
- 快速移动 1能 → 速度+80（应对防御+160）

**防御**：
- 刺盾 2能 → 减伤70%+应对物攻-70%
- 崩拳 2能65 → 应对状态物攻+100%

**控场**：
- 星轨裂变 1能 → 敌方+2层星陨
- 风隐 1能 → 双方脱离（先手-1）
- 精神扰乱 0能 → 敌方全技能+1能耗

**特殊**：
- 折射 4能50 → 携带其他系别技能有不同效果（草回血/火灼伤/幻叠星陨等）
- 复写 0能 → 随机变成未携带技能（能耗-2）

## 七、推荐配招方向

### 路线A：首领针对爆发流（炮台核心）
- 三连破 + 光刃 + 漫反射 + 刺盾
- 三连破物攻+30%+3连击，漫反射+35，光刃+月光审判240威力等效，刺盾防御应对

### 路线B：偷袭应对流（收割）
- 刺盾 + 偷袭 + 三连破 + 乘风
- 刺盾减伤70%+应对物攻-70%，偷袭应对255威力，三连破叠攻，乘风速度+120

### 路线C：折射联动流（炮台）
- 折射 + 光刃 + 三连破 + 漫反射
- 折射携带幻系技能叠星陨/草系回血，光刃月光审判爆发，三连破+漫反射加成
