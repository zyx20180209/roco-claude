# 鳗尾兽 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 3维 | 双攻91均衡+坟场搏击180威力（敌方每1能量-10%）+岩脉崩毁固定4能120+岩土暴击被攻击永久-1能耗 |
| 耐久 | 5维 | HP290（爆表，入场减半=145仍顶级）+物防138（94%）+特防113+丰富应对防御池+酶浓度调整应对回20%血 |

**主定位**：超高耐久拦截型炮台（HP290爆表即使减半仍145顶级+物防138+丰富应对池）
**战术角色**：拦截 + 炮台

## 二、种族值与速度

- HP290 / 攻91 / 特攻91 / 防138 / 特防113 / 速65（总788）

## 三、属性抗性（地/草双属性）

- **3x弱点**：冰（极危险）

- **1/3抗**：电（1条）
- **免疫**：无

## 四、特性「铃兰晚钟」

> 首次入场时，失去自己一半的当前生命。

首次入场HP减半：

- **HP爆表弥补**：HP290首次入场减半=145仍高于绝大多数精灵
- **与坟场搏击联动**：坟场搏击180威力（敌方每1能量-10%），鳗尾兽高HP可撑到敌方低能量时释放
- **与血气联动**（致命伤保留1血）：HP145+血气可延长生存至多一回合
- **与酶浓度调整联动**（应对回20%血=29血）：HP减半后回血量绝对值仍高
- **与嗜痛联动**（每次受伤双攻+40%）：HP高=可多次被打叠加双攻
- **核心价值**：HP290是稀有超高数值，即使减半仍顶级，换来特殊机制空间
- **局限性**：首次入场固定减半，非首次进场无此机制；速度65极慢被先手压制；冰系3x弱点致命

## 五、核心战术：高HP续航爆发

**核心逻辑**：
- 鳗尾兽即使首次入场减半，仍有145 HP（普通精灵HP顶级才130左右），实际生存能力最强
- 物防138（94%分位）顶级+丰富应对防御池（淤泥表皮/酶浓度调整/纤维化/嗜痛/血气等）
- 后期关键大招：坟场搏击180威力（对手能耗越低威力越高，鳗尾兽拖到对手低能量时爆发）

## 六、技能池亮点

**高威力输出**：
- 坟场搏击 4能180 → 敌方每1能量-10%威力（消耗战后期爆发）
- 岩脉崩毁 4能120 → 固定4能120威力（高性价比）
- 践踏 4能130 → 稳定地系
- 岩土暴击 8能140 → 被攻击永久-1能耗
- 叶绿光束 4能120 → 稳定草系魔攻
- 蓄能轰击 6能130 → 每使用其他普通系技永久-2能耗
- 地刺 3能95 → 应对状态打断

**能量封锁**：
- 操控 1能 → 敌方本回合技能能耗+7×3回合（超低费的极品控场技）

**应对防御（HP290撑池）**：
- 酶浓度调整 3能 → 减伤90%+应对回20%血
- 淤泥表皮 2能 → 减伤80%+应对敌方连击-3
- 嗜痛 2能 → 减伤80%+应对双攻+40%
- 纤维化 2能 → 减伤80%+应对物防+70%
- 血气 2能 → 减伤60%+应对致命伤保留1血
- 吓退 2能 → 减伤60%+应对敌方脱离
- 刺盾 2能 → 减伤70%+应对敌方物攻-70%
- 遁地 3能 → 减伤50%+脱离+应对攻击

**回血续航**：
- 聚盐 3能 → 2连击每次回5%血+1能+使用后永久+1连击
- 抽枝 4能90 → 应对状态回50%血+5能

**增益**：
- 蓄势待发 4能 → 蓄势印记全攻击技+30%威力

**控场**：
- 棘刺 2能 → 棘刺印记换人-6%血
- 埋伏 4能30×3 → 换人时连击+3
- 沙涌 7能 → 沙暴地系能耗减半（8回合）

## 七、推荐配招方向

### 路线A：高耐久拦截流（拦截核心）
- 操控 + 酶浓度调整 + 坟场搏击 + 聚盐
- 操控1能锁对手能量，酶浓度调整应对回20%血，鳗尾兽HP290拖到对手低能量时坟场搏击180威力爆发

### 路线B：应对反击流（拦截）
- 嗜痛 + 纤维化 + 岩脉崩毁 + 聚盐
- 嗜痛+应对双攻+40%×N叠加，纤维化物防+70%，岩脉崩毁固定4能120威力，聚盐回血续航

### 路线C：能量封锁流（工兵）
- 操控 + 棘刺 + 酶浓度调整 + 坟场搏击
- 操控+棘刺双重控场消耗，酶浓度调整高应对回血，坟场搏击对手低能量爆发
