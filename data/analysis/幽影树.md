# 幽影树 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 3维 | 双攻96均衡+彗星0能240威力+坟场搏击低能高威力+穿膛低能5倍 |
| 耐久 | 2维 | HP111+特防123（85%分位）；物防65极脆（6%分位） |
| 功能性 | 4维 | 小偷小摸入场偷2能+勾魂偷3能+恶作剧偷3能+降灵印记+操控/激怒能耗压制 |

**主定位**：功能型工兵（入场即偷能+多重偷能体系）
**战术角色**：工兵 + 破盾（能量压制逼迫拦截位下场）

## 二、种族值与速度

- HP111 / 攻96 / 特攻96 / 防65 / 特防123 / 速80（总571）
- **有效种族475**（去除特攻96）
- **极速207.2**（24%分位，偏慢）/ 满速181.0
- 启动速度：⭐⭐⭐⭐⭐ 即时（小偷小摸入场即触发，彗星0能即可释放）

## 三、属性抗性（幽/草双属性）

- **2x弱点**：火、冰、翼、幽、恶（5条偏多）
- **半抗**：普通、水、地、电、武（5条）
- **免疫**：无

## 四、特性「小偷小摸」

> 入场时偷取所有敌方精灵2能量

每次入场自动触发，无需出手：

- 入场即偷全场敌方精灵2能，配合中转队伍可多次反复入场偷能
- 配合降灵印记（换人入场-1能）：对手换人即损失3能（印记-1+小偷小摸-2）
- 配合勾魂（偷3能）+恶作剧（偷3能）：单回合最多偷8能
- 配合坟场搏击（敌方每有1能威力-10%）：偷能后坟场搏击威力接近满值
- **局限性**：只在入场时触发一次，持续战斗中不再偷能；物防65极脆

## 五、叠加型技能发育分析

**识别**：聚盐（每次使用连击数永久+1）

- **发育曲线**：聚盐初始2连击，每次+1。使用3次→5连击，每连击回血5%+1能。
- **发育阶段风险**：速度80偏慢，但特防123能扛特攻，发育期相对安全。
- **队伍依赖度**：低。小偷小摸入场即偷能，即使发育中断也有基础贡献。

## 六、推荐配招方向

### 路线A：入场偷能流（工兵+破盾）
- 勾魂 + 恶作剧 + 坟场搏击 + 移花接木
- 小偷小摸入场偷2能→勾魂偷3能→恶作剧偷3能，坟场搏击低能收割，移花接木换人再次触发入场偷能

### 路线B：彗星爆发流
- 彗星 + 勾魂 + 盛开 + 移花接木
- 盛开+30威力，彗星0能240威力一击必杀，移花接木脱离换人

### 路线C：降灵+操控控场流（工兵）
- 降灵 + 操控 + 激怒 + 坟场搏击
- 降灵印记换人-1能，操控+7能耗，激怒+3能耗，彻底封锁对手资源
