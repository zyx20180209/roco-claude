## 四、特性「盲拧」

> 回合开始时，技能顺序打乱，4号位的技能能耗-4。

每回合开始技能顺序随机打乱，4号位技能-4能耗：

- **随机打乱机制**：每回合技能位置随机重排，4号位技能获得-4能耗
- **高费技联动**：将高费技（钢铁洪流7能/齿轮切开5能等）放入构筑，随机到4号位时-4能耗
- **钢铁洪流联动**（3能70，1号位+90=160威力）：若到4号位=-4能耗=-1能160威力
- **联动装置联动**（两侧技能永久+20威力）：盲拧打乱后联动装置位置变化，两侧技能叠加
- **杠杆置换联动**（0能交换两侧）：手动调整位置，配合盲拧随机性
- **核心价值**：4号位技能-4能耗，高费技随机到4号位时极低费释放
- **局限性**：技能顺序随机，不可控；速度85偏慢；火/水/武3条弱点

## 五、技能池亮点

**盲拧4号位降费核心**：
- 钢铁洪流 3能70 → 1号位+90=160威力，4号位-4能=-1能160威力
- 械斗 1能45 → 1号位+60=105威力，4号位-4能=-3能105威力
- 齿轮扭矩 3能80 → 每回合位置变化永久+20威力，4号位-4能=-1能80威力

**传动位置联动**：
- 啮合传递 1能 → 速度+80，1/3号位物攻+60%，传动1
- 轴承支撑 3能 → 主动-1能耗，被动两侧-1能耗，传动1
- 联动装置 0能 → 两侧技能永久+20威力（应对防御+30）
- 杠杆置换 0能 → 回2能+交换两侧技能位置

**高威力输出**：
- 钢铁洪流 3能70 → 1号位160威力（盲拧4号位-1能）
- 电弧 3能80 → 迸发+40=120威力
- 超级糖果 3能100 → 萌化威力+60=160威力
- 见招拆招 2能65 → 上回合状态技后+55=120威力

**增益**：
- 力量增效 1能 → 物攻+100%
- 主场优势 3能 → 攻击印记全技能+10%威力
- 啮合传递 1能 → 1/3号位物攻+60%

**防御续航**：
- 防反 2能 → 减伤60%+应对双攻+40%
- 有效预防 1能 → 减伤50%+应对先手+1
- 电磁偏转 3能 → 减伤70%+应对次数+1
- 应激反应 2能 → 回血25%（应对防御50%）

**控场**：
- 麻痹 2能 → 敌方先手-1+应对防御双攻-70%
- 精神扰乱 0能 → 能耗+1（应对防御+3）
