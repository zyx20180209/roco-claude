## 四、特性「自由飘」

> 自己每有1层萌化，获得连击数+2

萌化层数越多，连击数越高：

- 1层萌化 → 连击+2
- 2层萌化 → 连击+4
- 3层萌化 → 连击+6
- **配合超级糖果**（3能100，自己获得萌化+60威力）：每次使用叠1层萌化，连击+2
- **配合甜心续航**（触发萌化+附加回血40%+4能）：叠萌化+回血续航
- **配合月光合奏**（双方每层萌化连击+1）：萌化层数越多月光合奏连击越高
- **注意萌化机制**：萌化=退化形态，每层萌化触发退化，但自由飘的连击+2是基于萌化层数的永久状态加成
- **局限性**：5条2x弱点多；HP87偏脆

## 五、叠加型技能发育分析

**识别**：超级糖果（每次使用自己获得萌化+60威力）

- **发育曲线**：超级糖果初始100威力，每次使用萌化+60威力。使用3次→280威力+自由飘+6连击。
- **极速280先手**：发育期几乎不会被先手，自给自足。
- **队伍依赖度**：低。极速顶级+自由飘自给自足。

## 六、技能池亮点

**萌化连击核心**：
- 超级糖果 3能100 → 萌化+60威力（自由飘+2连击）
- 月光合奏 3能30×N → 双方每层萌化连击+1（萌化叠满后爆发）
- 甜心续航 3能 → 萌化+附加回血40%+4能（续航+叠萌化）
- 生日蛋糕 4能 → 驱除减益+增益翻倍（萌化翻倍后连击数翻倍）

**高速输出**：
- 扇风 3能75 → 先手+50%威力（极速280几乎必先手）
- 疾风刺 2能25 → 先手3连击（+自由飘萌化连击）
- 啄击 0能15×2 → 0能连击
- 逆袭 3能100 → 能耗每+1威力+50（高费后威力极高）

**增益**：
- 羽化加速 2能 → 威力+20+迅捷入场
- 风起 4能 → 风起印记先手攻击+20%威力
- 热身运动 2能 → 连击+3
- 三连破 1能 → 物攻+30%+3连击

**功能**：
- 晒太阳 1能 → 驱散敌方增益
- 倾泻 3能70 → 未被应对驱散双方印记

## 七、推荐配招方向
