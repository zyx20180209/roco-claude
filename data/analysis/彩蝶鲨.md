# 彩蝶鲨 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 3维 | 双攻77偏中+水翼推进入场能耗递减+水炮永久-1+天洪应对-6+盐水浴全技能-2 |
| 耐久 | 2维 | 双防105均衡（66%）；HP90偏脆 |
| 功能性 | 3维 | 水翼推进队友水系技能积累入场能耗-1+风隐双方脱离+落雨天气+打湿湿润印记 |

**主定位**：水系队入场能耗递减型工兵（水翼推进队友水系技能积累→入场即低费）
**战术角色**：工兵 + 中转

## 二、种族值与速度

- HP90 / 攻77 / 特攻77 / 防105 / 特防105 / 速125（总579）
- **有效种族502**（去除特攻77）
- **极速266.6**（95%分位，顶级高速）/ 满速230.5
- 启动速度：⭐⭐⭐⭐⭐ 即时（水翼推进被动积累，入场即低费）

## 三、属性抗性（水/翼双属性）

- **3x弱点**：电（致命）
- **2x弱点**：冰
- **半抗**：火、虫、武、机械
- **免疫**：无

## 四、特性「水翼推进」

> 己方精灵每使用1次水系技能，自己入场时获得全技能能耗-1

队友每用1次水系技能，彩蝶鲨入场时全技能能耗-1：

- 队友使用3次水系技能 → 入场全技能-3能耗
- 队友使用5次水系技能 → 入场全技能-5能耗（水炮5能→0能，天洪7能→2能）
- **配合水系队友**（水灵/水泡壳/千棘盔等）：队友每次出手水系技能都在为彩蝶鲨积累
- **配合盐水浴**（全技能-2能耗）：入场后再-2，叠加后几乎0能出手
- **核心价值**：水系队中入场即满能低费，天洪/水炮等高费技能几乎0能释放
- **局限性**：需要水系队友积累；电系3x弱点致命

## 五、叠加型技能发育分析

**识别**：水炮（永久-1能耗）+ 涌泉（能耗每-1威力+10）

- **水炮**：初始5能，每次使用-1。配合水翼推进入场-N能，水炮可能入场即0能。
- **涌泉**：初始6能60威力，能耗每-1威力+10。配合水翼推进+盐水浴后涌泉威力大幅提升。
- **队伍依赖度**：高（需要水系队友积累水翼推进）。

## 六、技能池亮点

**水翼推进联动输出**：
- 天洪 7能150 → 应对状态永久-6能耗（入场-5能后天洪2能，应对触发后0能150威力）
- 水炮 5能110 → 永久-1能耗（入场-5能后水炮0能）
- 涌泉 6能60 → 能耗每-1威力+10（入场-5能后涌泉1能110威力）
- 回旋风暴 3能100 → 翼系稳定输出

**降费工具**：
- 盐水浴 2能 → 全技能-2能耗（应对防御-3）
- 打湿 4能 → 湿润印记全技能-1能耗
- 落雨 5能 → 雨天水系+50%威力

**功能**：
- 风隐 1能 → 双方脱离+先手-1（主动触发再次入场积累水翼推进）
- 翼击 3能50 → 迅捷
- 消毒法 4能115 → 驱散敌方5层增益

**增益**：
- 魔法增效 0能 → 魔攻+70%
- 贮藏 2能 → 双攻+50%+每个0能技能额外+50%

## 七、推荐配招方向

### 路线A：水翼推进爆发流（工兵+炮台）
- 天洪 + 水炮 + 盐水浴 + 魔法增效
- 水系队友积累5次→入场全技能-5能，盐水浴再-2，天洪0能150威力

### 路线B：涌泉发育流（炮台）
- 涌泉 + 水炮 + 盐水浴 + 落雨
- 水翼推进+盐水浴后涌泉低费高威力，落雨雨天水系+50%

### 路线C：风隐中转流（中转+工兵）
- 风隐 + 天洪 + 盐水浴 + 魔法增效
- 风隐主动脱离再入场触发水翼推进，天洪低费终结

## 十二、首领化形态：神谕鲨

**种族值**：HP90 / 攻77 / 特攻77 / 防105 / 特防105 / 速125（与彩蝶鲨完全相同）
**特性「水翼飞升」**：己方精灵每使用1次水系技能，自己入场时获得全技能能耗-1，且**能耗为0的技能威力+30%**

### 水翼推进 vs 水翼飞升

| 对比 | 彩蝶鲨（水翼推进） | 神谕鲨（水翼飞升） |
|------|------|------|
| 入场能耗递减 | 水系技能×1能耗-1 | 水系技能×1能耗-1（相同） |
| 额外效果 | 无 | **0能技能威力+30%** |
| 种族值 | 相同 | 相同 |

**水翼飞升的质变**：在水翼推进基础上，0能技能威力+30%。配合贮藏（每个0能技能双攻+50%）+水翼飞升（0能技能+30%威力），0能技能爆发力极强。
