# 圣羽翼王 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |

**主定位**：翼系迅捷体系绝对核心（飓风共享迅捷+HP132+物防128顶级联防+极速266）
**战术角色**：炮台 + 拦截 + 收割

## 二、种族值与速度

- HP132 / 攻109 / 特攻109 / 防128 / 特防83 / 速125（总686）
- **有效种族577**（去除特攻109）
- **极速266.6**（95%分位，顶级高速）/ 满速230.5
- 启动速度：⭐⭐⭐⭐⭐ 即时（飓风被动即时，极速266先手）

## 三、属性抗性（翼系）

- **2x弱点**：冰、电（仅2条）
- **半抗**：草、虫、武（3条）
- **免疫**：无

## 四、特性「飓风」

> 对本精灵的技能，若其他翼系精灵携带相同技能，则获得迅捷。被敌方精灵击败时，自己额外损失1点魔力

核心机制：圣羽翼王的技能如果有其他翼系队友也携带相同技能，圣羽翼王使用该技能时获得迅捷：

- **共享技能触发**：翠顶夫人/岚鸟/女王蜂等翼系队友带水刃/力量增效/龙卷风等技能 → 圣羽翼王使用该技能时迅捷
- **社区天作之合**：翠顶夫人（水+翼双系）与圣羽翼王共同携带水刃 → 圣羽翼王水刃迅捷115威力
- 配合自带迅捷技（翼击/龙卷风/羽化加速）：进一步扩大迅捷池
- **代价**：被击败时额外损失1点魔力（4点魔力全队共享，额外-1是重大惩罚）
- **局限性**：需要翼系队友构筑；特防83偏中；被击败惩罚大

## 五、核心战术：翼系迅捷体系

**飓风触发条件**：其他翼系精灵与圣羽翼王携带相同技能

**最佳搭档**（社区共识）：
- **翠顶夫人**（水+翼双系）：与圣羽翼王同时携带**水刃** → 飓风触发水刃迅捷
- **岚鸟/霜翼领主**：共享翼系技能
- **女王蜂**：虫系联动+翼系技能

**共享池可选**：力量增效、水刃、龙卷风、羽刃、疾风刺、扇风等

## 六、技能池亮点

**飓风迅捷核心**：
- 水刃 4能115 → 飓风后迅捷+应对状态永久-4能耗
- 龙卷风 5能70 → 自带迅捷+应对状态1.5倍
- 翼击 3能50 → 自带迅捷
- 疾风连袭 0能 → 释放自己释放过的迅捷技能（核心输出技）

**高速输出**：
- 扇风 3能75 → 先手+50%威力（极速266几乎必先手）
- 闪击 4能60 → 速度差越大威力越高
- 疾风刺 2能25 → 先手3连击
- 回旋风暴 3能100 → 稳定魔攻

**增益**：
- 羽化加速 2能 → 威力+20+迅捷入场
- 风起 4能 → 风起印记先手+20%威力
- 暴风眼 2能 → 连击+100%（翻倍）
- 魔法增效 0能 → 魔攻+70%

**控场**：
- 羽刃 2能75 → 应对状态强制换人

## 七、推荐配招方向

### 路线A：翼系迅捷体系流（炮台，核心玩法）
- 水刃 + 龙卷风 + 疾风连袭 + 魔法增效
- 水刃飓风迅捷115威力（与翠顶夫人搭配），疾风连袭重放迅捷技，龙卷风1.5倍威力

### 路线B：高速爆发流（收割）
- 扇风 + 暴风眼 + 疾风刺 + 力量增效
- 极速266先手+扇风+50%威力，暴风眼连击翻倍，疾风刺先手3连击

### 路线C：联防+输出流（拦截+炮台）
- 羽刃 + 水刃 + 回旋风暴 + 魔法增效
- 羽刃应对状态强制换人，水刃迅捷115威力，魔法增效+70%魔攻后回旋风暴爆发
