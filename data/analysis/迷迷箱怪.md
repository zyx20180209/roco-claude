## 四、特性「虚假宝箱」

> 自己力竭时，敌方获得攻防+20%。

力竭（被击败）时给敌方攻防+20%：

- **死亡惩罚机制**：迷迷箱怪被击败时，对手获得攻防+20%增益
- **与公平鸽衡量联动**：对手获得+20%增益后，衡量特性精灵可以复制
- **战术博弈**：对手不敢轻易击败迷迷箱怪（否则给自己队友送增益）
- **与晒太阳联动**：对手获得增益后，己方晒太阳驱散
- **核心价值**：威慑对手不敢轻易击败，延长生存时间；被击败时也能给对手制造麻烦
- **局限性**：对手可以选择不击败（让迷迷箱怪一直在场）；速度80偏慢；火/水/幽3条弱点

## 五、传动系统说明

迷迷箱怪拥有独特的「传动」机制：
- 带有「传动X」的技能，回合开始时向下移动X个位置
- 技能位置影响威力/能耗/效果（1号位/3号位有特殊加成）
- 核心传动技：械斗（1号位+60威力）、啮合传递（1/3号位+物攻+60%）、离子震荡（3号位+40威力）、磁暴（1/3号位+30威力）、齿轮切开（1/3号位-2能耗）

## 六、技能池亮点

**传动位置联动核心**：
- 械斗 1能45 → 1号位威力+60=105，传动1（每回合下移1位）
- 啮合传递 1能 → 速度+80，1/3号位额外物攻+60%，传动1
- 离子震荡 3能90 → 3号位威力+40=130，传动1
- 磁暴 2能70 → 1/3号位威力+30=100，传动1
- 齿轮切开 5能130 → 1/3号位能耗-2=3能130，传动1
- 钢钻 4能 → 两侧技能威力和×1/3，传动1

**高威力输出**：
- 后发制人 3能155 → 先手-1（后手时威力高）
- 压扁 5能155 → 顶级物攻
- 音爆 4能130 → 稳定魔攻
- 双星 3能100 → 稳定物攻
- 四维降解 7能100 → 敌方每1层印记-1能耗（星陨叠层后低费）

**星陨控场**：
- 星链 3能 → 2连击每次赋予1层星陨（共2层）
- 冥想 3能 → 减伤80%+应对2层星陨
- 错乱 2能65 → 应对状态3层星陨
- 四维降解 7能100 → 每层印记-1能耗（星陨叠层后低费）

**防御续航**：
- 能量守恒 2能 → 减伤80%+应对两侧技能永久-1能耗（核心降费）
- 无畏之心 5能 → 减伤100%+回血（能耗永久+2）
- 硬化 2能 → 减伤90%（上回合攻击后-2能耗）
- 刺盾 2能 → 减伤70%+应对物攻-70%
- 吓退 2能 → 减伤60%+应对敌方脱离

**增益/控场**：
- 力量增效 1能 → 物攻+100%
- 棘刺 2能 → 换人-6%血
- 退化 2能 → 敌方萌化
- 震击 3能80 → 敌方连击-3

## 七、推荐配招方向

- 双防153顶级+能量守恒应对降费+硬化减伤90%，后发制人155威力，棘刺换人惩罚
