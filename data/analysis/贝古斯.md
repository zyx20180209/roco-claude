## 四、特性「贪心算法」

> 1号位技能获得传动，且使用后使敌方获得6层灼烧。

1号位技能自动获得传动效果，使用后附加6层灼烧：

- **传动机制**：1号位技能每回合向下移动，到达特定位置触发加成（如离子震荡3号位+40威力）
- **灼烧附加**：每次使用1号位技能=敌方+6层灼烧，配合充分燃烧翻倍极限叠层
- **与杠杆置换联动**：杠杆置换0能回2能+交换两侧技能位置，灵活调整1号位技能
- **与轴承支撑联动**：轴承支撑传动1+两侧技能-1能耗，配合贪心算法传动叠加
- **与天火联动**：天火10层灼烧（应对防御30层）+贪心算法每次+6层=极限灼烧
- **与充分燃烧联动**：充分燃烧灼烧翻倍+触发1次伤害，贪心算法叠层后翻倍爆发
- **核心价值**：传动机制让技能位置管理成为战术核心，灼烧持续叠加
- **局限性**：速度55极慢被先手压制；水系3x弱点致命；需要管理技能位置

## 五、核心战术：传动位置管理+灼烧叠层

**灼烧叠层流**：
1. 天火（3能）→ 敌方10层灼烧（应对防御30层）
2. 贪心算法1号位技能 → 每次使用+6层灼烧
3. 充分燃烧（3能）→ 灼烧翻倍+触发1次伤害

**传动爆发流**：
1. 杠杆置换（0能）→ 回2能+调整技能位置
2. 离子震荡（3能90）→ 3号位时+40威力=130威力
3. 磁暴（2能70）→ 1/3号位时+30威力=100威力

## 六、技能池亮点

**传动核心（贪心算法联动）**：
- 杠杆置换 0能 → 回2能+交换两侧技能（0能调位）
- 轴承支撑 3能 → 两侧-1能耗+传动1（能耗降低）
- 啮合传递 1能 → 速度+80（1/3号位+物攻+60%）+传动1
- 联动装置 0能 → 两侧+20威力（应对防御+30）

**灼烧叠层**：
- 天火 3能 → 10层灼烧（应对防御30层）
- 引燃 2能 → 10层灼烧
- 充分燃烧 3能 → 灼烧翻倍+触发1次伤害

- 烈焰风暴 4能75 → 6层灼烧

**高威力输出**：
- 燃尽 4能155 → 顶级火系魔攻（敌方低血时威力高）
- 离子震荡 3能90 → 3号位+40=130威力
- 四维降解 7能100 → 敌方每层印记-1能耗
- 磁暴 2能70 → 1/3号位+30=100威力

**防御续航**：
