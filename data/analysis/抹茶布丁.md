## 四、特性「茶多酚」

> 离场后，更换入场的精灵回复20%生命且免疫寄生。

离场后下个入场精灵回血20%+免疫寄生：

- **中转回血**：抹茶布丁离场后队友入场立即回20%血，等效切换+治疗
- **与泡沫幻影联动**：泡沫幻影减伤80%+应对自脱离，触发茶多酚让队友回血20%
- **免疫寄生**：下个入场精灵免疫寄生（如孢子/纠缠等），破除寄生体系压制
- **战术节奏**：抹茶布丁作为中转位，进场释放能耗工具后主动撤下让队友回血
- **核心价值**：队伍级别回血+免疫寄生工具，支援通用性强
- **局限性**：需要主动离场才触发（泡沫幻影/换人）；自身攻击面偏弱；5条2x弱点偏多

## 五、核心战术：中转支援+能耗降低

**支援流程**：
1. 蓄水（1能）→ 下次技能-6能耗
2. 水炮/天洪（5-7能）→ 蓄水后低费高威力释放
3. 泡沫幻影（2能）→ 减伤80%+应对自脱离
4. 茶多酚触发→ 队友入场回20%血+免疫寄生

**水炮成长流**：
1. 水炮（5能110威力）→ 每次使用永久-1能耗（成长型）
2. 第3次使用时仅需2能110威力，性价比极高
3. 配合蓄水-6能耗可免费释放

## 六、技能池亮点

**高威力输出**：
- 天洪 7能150 → 应对状态永久-6能耗（成长爆发）
- 水炮 5能110 → 每次使用永久-1能耗（成长）
- 水刃 4能115 → 应对状态永久-4能耗
- 叶绿光束 4能120 → 稳定草系魔攻
- 棘突 3能100 → 草系魔攻
- 光能聚集 7能100 → 每次使用其他草系技永久+60威力

**低费高效**：
- 肥皂泡 1能60 → 低费水系魔攻
- 泡沫 1能60 → 低费水系物攻
- 潮涌 2能80 → 低费水系物攻
- 阻断 2能80 → 应对打断

**能耗降低**：
- 蓄水 1能 → 下次技能-6能耗（核心）
- 洗礼 1能 → 驱自己减益+全技能-1能耗
- 水环 2能 → 减伤60%+应对全技能-2能耗

**防御续航**：
- 泡沫幻影 2能 → 减伤80%+应对自脱离（触发茶多酚）
- 纤维化 2能 → 减伤80%+应对物防+70%
- 蜡质膜 2能 → 减伤80%+应对回3能
- 捧杀 4能 → 减伤90%+应对敌方萌化

**增益**：
- 花炮 2能 → 2连击每次魔攻+60%（滚雪球）

## 七、推荐配招方向

### 路线A：中转支援流（中转核心）
- 蓄水 + 水炮 + 泡沫幻影 + 洗礼
- 蓄水-6能耗→水炮110威力+永久-1能耗，泡沫幻影应对自脱离触发茶多酚，洗礼驱减益

### 路线B：花炮爆发流（炮台）
- 花炮 + 叶绿光束 + 肥皂泡 + 泡沫幻影
- 花炮2连击每次+60%魔攻滚雪球，叶绿光束120威力爆发，肥皂泡低费输出，泡沫幻影触发茶多酚

### 路线C：能耗控场流（工兵）
- 水环 + 蓄水 + 天洪 + 纤维化
- 水环应对全技能-2能耗，蓄水-6能耗，天洪7能150威力（应对永久-6），纤维化防御
