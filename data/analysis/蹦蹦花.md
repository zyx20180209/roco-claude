# 蹦蹦花 PVP 对战分析（四形态）

## 一、共同基础

**属性**：毒/草双属性
**特性**：生物碱——使用草系技能时，敌方获得2层中毒
**4形态种族值差异极小**（有效种族473-478）

## 三、属性抗性（毒/草双属性）

- **2x弱点**：火、冰、翼、恶、幻（5条偏多）
- **半抗**：草、水、光、电、武、萌（6条）
- **免疫**：无

## 四、特性「生物碱」

> 使用草系技能时，敌方获得2层中毒

每次用草系技能自动触发2层中毒，无需额外出手：

- 配合刺藤/藤绞/叶绿光束等草系输出技 → 每次攻击+2层中毒
- 配合毒孢子5层/剧毒3层/腐化 → 快速叠加中毒层数
- 配合毒液渗透（中毒层越多能耗越低）：5层中毒→毒液渗透0能120威力
- 配合毒雾（增益转化中毒）+落井下毒（减益翻倍）：组合爆炸性
- **局限性**：对毒系精灵无效（互相免疫）；中毒为百分比DOT，对高HP对手有效但对低HP对手反而不如直接爆发

## 五、技能池亮点

**草+毒双系打击**：
- 刺藤 3能45×2 → 草系物攻，2连击触发生物碱2层中毒
- 叶绿光束 4能120 → 草系魔攻
- 仙人掌刺击 6能150 → 高威力草系物攻
- 毒液渗透 5能120 → 中毒层越多能耗越低
- 针刺射击 3能80 → 敌方换人时回7能（反中转）

**中毒叠加**：
- 毒孢子 3能 → 5层中毒
- 剧毒 2能 → 3层中毒（应对防御8层）
- 腐化 1能 → 敌方每层中毒双攻-30%

**续航**：
- 根吸收 2能 → 回血15%+4能
- 蜡质膜 2能 → 减伤80%+回3能
- 汲取 1能30 → 100%吸血

**功能**：
- 落井下毒 6能 → 敌方减益层数翻倍
- 毒雾 7能 → 敌方增益转化中毒
- 晒太阳 1能 → 驱散敌方增益
- 移花接木 2能 → 回血15%+脱离

## 六、推荐配招方向

### 路线A：生物碱破盾流（破盾）
- 刺藤 + 毒孢子 + 毒液渗透 + 腐化
- 刺藤触发生物碱每次+2层中毒，毒孢子5层+腐化每层双攻-30%，毒液渗透低费终结

### 路线B：毒雾反增益流（工兵+破盾）
- 毒雾 + 落井下毒 + 毒液渗透 + 晒太阳
- 毒雾转化增益为中毒，落井下毒减益翻倍，毒液渗透收割

### 路线C：草系续航流（拦截+炮台）
- 叶绿光束 + 蜡质膜 + 根吸收 + 孢子
- 叶绿光束生物碱触发中毒，蜡质膜减伤80%回能，根吸收续航

## 八、擅长应对的情况

- 高耐久拦截位（中毒DOT绕过减伤）
- 水/地系对手（草系2x克制）
- 依赖增益积累的对手（毒雾转中毒）
- 换人频繁的对手（针刺射击回7能反中转）

## 九、棘手的对手特点

- 毒系精灵（免疫中毒）
- 火/翼/恶/幻系先手压制（5条2x弱点）
- 带洗礼/自清技能的对手（清除中毒）
- 物防73偏脆，高物攻威胁大
