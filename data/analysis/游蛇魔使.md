# 游蛇魔使 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 3维 | 双攻110/104均衡+思维之盾应对后下次技能能耗-5+破防敌方双防-130%+提气威力+40 |
| 耐久 | 3维 | HP105+物防103+特防103均衡+听桥减伤60%+应对反伤 |
| 功能性 | 3维 | 思维之盾应对降费+破防双防-130%+盐水浴全技能-2能耗+落雨天气 |

**主定位**：应对降费型爆发（思维之盾应对后能耗-5+高费大招低费释放）
**战术角色**：炮台 + 拦截

## 二、种族值与速度

- HP105 / 攻110 / 特攻104 / 防103 / 特防103 / 速105（总630）
- **有效种族526**（去除特攻104）
- **极速240.2**（63%分位，中高速）/ 满速208.5
- 启动速度：⭐⭐⭐⭐ 快速（思维之盾应对触发，1能技能即可出手）

## 三、属性抗性（水/武双属性）

- **2x弱点**：草、电、翼、萌、幻（5条偏多）
- **半抗**：火、地、虫、恶、机械（5条）
- **免疫**：无

## 四、特性「思维之盾」

> 应对成功后，下次行动技能能耗-5

每次应对成功后下次技能能耗-5：

- 一拳（5能140）→ 应对后0能140威力
- 打鼾（6能165）→ 应对后1能165威力
- 涌泉（6能60）→ 应对后1能+能耗每-1威力+10（=5能耗-5=0能+50威力=110威力）
- 水波术（6能90）→ 应对后1能90威力+每回合威力永久+20
- 润泽（7能魔攻+190%）→ 应对后2能使用
- **核心价值**：应对后下次攻击几乎0能耗，高费大招可以低费释放
- **局限性**：需要对手主动攻击触发；5条2x弱点偏多

## 五、技能池亮点

**思维之盾联动高费技**：
- 一拳 5能140 → 应对后0能140威力
- 打鼾 6能165 → 应对后1能165威力
- 涌泉 6能60 → 能耗每-1威力+10（思维之盾后威力暴涨）
- 水波术 6能90 → 回合结束威力永久+20（应对后低费使用）
- 润泽 7能 → 魔攻+190%（应对后2能使用）

**破盾/增益**：
- 破防 3能 → 敌方双防-130%（应对防御技能冷却2回合）
- 提气 4能 → 全技能威力+40（换人+50）
- 预备势 1能 → 物攻+80%（应对防御敌方物防-80%）
- 力量增效 1能 → 物攻+100%
- 化劲 2能 → 全技能威力+40

**防御反击**：
- 听桥 4能 → 减伤60%+应对攻击反伤（威力=被应对技能威力）
- 叠势 3能25×2 → 每应对永久+2连击
- 泡沫幻影 2能 → 减伤80%+脱离

**能量工具**：
- 盐水浴 2能 → 全技能-2能耗（应对防御-3）
- 打湿 4能 → 湿润印记全技能-1能耗
- 落雨 5能 → 雨天水系+50%威力

## 六、推荐配招方向

### 路线A：思维之盾爆发流（炮台）
- 一拳 + 听桥 + 破防 + 力量增效
- 听桥应对触发思维之盾，一拳0能140威力，破防双防-130%

### 路线B：打鼾应对流（炮台+拦截）
- 打鼾 + 叠势 + 听桥 + 提气
- 叠势应对叠连击+思维之盾，打鼾应对后1能165威力，提气+40威力

### 路线C：水波术发育流（炮台）
- 水波术 + 听桥 + 落雨 + 盐水浴
- 水波术回合结束永久+20威力+应对后低费，落雨水系+50%威力

## 八、擅长应对的情况

- 依赖攻击的对手（思维之盾应对触发高费低释）
- 依赖高耐久的对手（破防双防-130%破盾）
- 水系队（雨天+50%威力）
- 消耗战节奏（水波术永久+20威力）

## 九、棘手的对手特点

- 草/电/翼/萌/幻系先手压制（5条2x弱点）
- 不攻击的对手（思维之盾无触发机会）
- 速度105极速240被更高速精灵压制
- HP105偏脆，高物攻先手威胁大
