## 五、特性「好象坏象」

> 状态技能应对1次后，回满状态，变为棋骑后。

棋祈督使用的状态技被对方应对1次后触发回满+变身：

- **反向触发机制**：与棋骑士/棋齐垒不同，棋祈督的"被破解"反而成为优势——对方应对到状态技触发变身
- **主动触发套路**：自己先用状态技（力量增效/均势/破绽等），对方若用防御/应对技应对该状态技→触发特性
- **对手不应对则享受增益**：对手不应对状态技→棋祈督白拿增益；对手应对状态技→棋祈督回满变身
- **双赢情境**：逼迫对手做出选择，无论选哪边棋祈督都有利
- **变身棋骑后**：变身后形态具备渗透特性（队友武/地技能使用×+5%攻防入场）
- **核心价值**：状态技变成"进可攻退可变身"的双保险战术工具
- **局限性**：需要对方主动应对状态技才触发；7条2x弱点极多；速度85偏慢

## 七、技能池亮点

**状态技（触发好象坏象核心）**：
- 力量增效 1能 → 物攻+100%
- 均势 2能 → 物防+140%+速度-30
- 石肤术 2能 → 物防+160%+魔防-60%
- 破绽 2能 → 敌方双防-70%+应对防御物攻+70%
- 预备势 1能 → 物攻+80%+应对防御敌方物防-80%
- 蓄势待发 1能 → 蓄势印记全攻击技+30%威力
- 泥浆铠甲 3能 → 物攻物防+60%+应对防御增益翻倍
- 化劲 2能 → 全技能威力+40
- 提气 4能 → 全技能威力+40（换人+50）
- 漫反射 2能 → 每系别至多一个技能威力+35

**天气/环境工具**：
- 放晴 3能 → 光系永久+50%（应对防御+100%）
- 沙涌 3能 → 沙暴地系能耗减半（8回合）
- 流沙 3能 → 敌方3回合无法换人+应对防御双防-60%

**高威力输出**：
- 岩脉崩毁 4能120 → 地系固定4能120威力（高性价比）
- 岩土暴击 7能140 → 被攻击永久-1能耗
- 迫近攻击 3能90 → 每次使用永久+45威力
- 影袭 3能100 → 稳定武系
- 天光 4能95 → 系别随天气（沙暴=地系，晴天=光系）

**应对防御**：
- 听桥 4能 → 减伤60%+应对反伤
- 防御反击 2能 → 减伤80%+应对全技能威力+40
- 叠势 3能25×N → 每应对永久+2连击

## 八、推荐配招方向

### 路线A：状态陷阱流（工兵核心）
- 破绽 + 预备势 + 岩脉崩毁 + 力量增效
- 破绽/预备势状态技逼迫对手选择（不应对=白拿减益+增益；应对=好象坏象回满变身），岩脉崩毁4能120威力爆发

### 路线B：状态诱骗变身流（工兵）
- 均势 + 石肤术 + 听桥 + 岩土暴击
- 均势/石肤术堆防（被应对触发好象坏象），听桥应对反伤，岩土暴击被攻击永久-1能耗

### 路线C：沙暴地系流（炮台）
- 沙涌 + 岩脉崩毁 + 天光 + 力量增效
- 沙涌沙暴地系能耗减半，岩脉崩毁固定4能120威力（沙暴下更低费），天光沙暴=地系
