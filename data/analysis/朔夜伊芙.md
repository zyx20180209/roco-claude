# 朔夜伊芙 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|

**战术角色**：收割 + 工兵（交换类技能博弈）

- **2x弱点**：光、虫、武、萌（4条）

- **免疫**：无

## 四、特性「嫁祸」

> 自己每失去25%生命，连击数+2

核心机制：血量越低连击数越高：

- 失去25%血（血量75%）→ 连击+2
- 失去50%血（血量50%）→ 连击+4
- 失去75%血（血量25%）→ 连击+6
- 满能量贪婪100%吸血 → 吸血回复后又能再次触发嫁祸的负向循环
- **配合撕咬**（3能30×3，血量<50%改+2=5连击）：低血时撕咬5连击×30=150等效威力
- **配合黑手/叠势**：连击类技能叠加嫁祸连击后爆发
- **核心价值**：越残血越强，形成反杀循环
- **局限性**：需要先承受伤害才能触发；特防85偏中等

## 五、叠加型技能发育分析

**识别**：撒娇（自己获得萌化威力永久+20）+ 叠势（每成功应对1次永久+2连击）

- **撒娇**：初始30威力，每次使用萌化+20威力。使用3次→90威力+3连击
- **叠势**：初始2连击，每应对+2连击。应对3次→8连击
- **队伍依赖度**：低。嫁祸+吸血自给自足。

## 六、技能池亮点

**嫁祸联动核心**：
- 撕咬 3能30×3 → 血量<50%改5连击（低血时等效威力150）
- 黑手 3能45×2 → 嫁祸后连击数+N
- 叠势 3能25×2 → 每应对+2连击
- 月光合奏 3能30×N → 双方每层萌化连击+1

**吸血续航**：
- 贪婪 2能 → 100%吸血
- 蝙蝠 2能65 → 吸血100%（2能高性价比）
- 等价交换 3能 → 减伤90%+应对50%吸血
- 甜心续航 3能 → 萌化+附加回血40%+4能

**交换类功能**（博弈工具）：
- 欺诈契约 3能 → 与敌方交换增益和减益（强力反制强化队）
- 隐藏条款 8能 → 与敌方交换携带的技能（高风险高回报）
- 伪造账单 1能 → 若敌方本回合回血改为失去2倍+先手+1（反续航）

**控场**：
- 咆哮 1能 → 敌方物攻-130%
- 晒太阳 1能 → 驱散敌方增益
- 恶意逃离 1能 → 脱离+应对防御能耗+4

## 七、推荐配招方向

### 路线A：嫁祸反杀流（收割）
- 撕咬 + 贪婪 + 蝙蝠 + 力量增效
- 贪婪100%吸血维持嫁祸循环，撕咬低血5连击爆发，力量增效+100%物攻

### 路线B：交换控场流（工兵+收割）
- 欺诈契约 + 撕咬 + 咆哮 + 伪造账单
- 欺诈契约交换敌方增益减益，伪造账单反续航，撕咬终结

### 路线C：叠势应对流（收割+拦截）
- 叠势 + 等价交换 + 撕咬 + 贪婪
- 等价交换应对90%减伤+50%吸血，叠势应对+2连击，撕咬低血爆发
