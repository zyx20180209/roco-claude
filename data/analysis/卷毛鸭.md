# 卷毛鸭 PVP 对战分析

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 4维 | 物攻98+得寸进尺雨天双攻+100%+力量增效物攻+100%+崩拳应对状态物攻+100%+爆冲应对状态5倍威力 |

**战术角色**：收割 + 炮台

## 二、种族值与速度

- HP116 / 攻98 / 特攻29 / 防83 / 特防63 / 速115（总504）
- **有效种族475**（去除特攻29）
- **极速253.4**（82%分位，高速）/ 满速219.5
- 启动速度：⭐⭐⭐⭐ 快速（落雨1回合触发得寸进尺后，疾风刺/崩拳即可爆发）

## 三、属性抗性（武/草双属性）

- **3x弱点**：翼（极危险）
- **2x弱点**：火、冰、毒、萌、幻（5条偏多）
- **半抗**：水、光、电、恶（4条）
- **1/3抗**：地（1条）
- **免疫**：无

## 四、特性「得寸进尺」

> 天气为雨天，或处于其他水系环境中时，获得双攻+100%。

雨天环境下双攻直接翻倍：

- **落雨联动**（自带落雨5能）：自己打落雨触发得寸进尺+100%
- **队友水系支援**：水系队友携带落雨/潮汐等水环境，卷毛鸭被动享受+100%
- **与力量增效叠加**：得寸进尺+100%+力量增效+100%=物攻+200%
- **与崩拳应对叠加**：得寸进尺+崩拳应对状态+100%+力量增效=物攻+300%
- **核心价值**：雨天环境下物攻98（66%分位）暴涨至+100%/+200%/+300%，配合极速253先手
- **局限性**：依赖雨天或水环境（不自带环境需队友支持或落雨占技能栏）；翼3x弱点+5条2x弱点偏多

## 五、技能池亮点

**雨天触发**：
- 落雨 5能 → 雨天8回合（触发得寸进尺）
- 水刃 5能115 → 应对状态永久-4能耗（雨天+50%威力）

**高威力输出**：
- 一拳 5能140 → 雨天+得寸进尺爆发
- 水刃 5能115 → 雨天+50%威力=172.5
- 顶端优势 3能100 → 稳定草系
- 爆冲 3能65 → 应对状态5倍=325威力
- 偷袭 3能85 → 应对状态3倍=255威力

**增益**：
- 力量增效 1能 → 物攻+100%（与得寸进尺叠加）
- 崩拳 2能65 → 应对状态物攻+100%（三重叠加）
- 盛开 1能 → 威力+30（应对防御+60）
- 热身运动 2能 → 连击+3

**应对爆发**：
- 爆冲 3能65 → 应对状态5倍=325威力
- 无影脚 3能85 → 应对状态2倍=170
- 散手 3能35×6 → 应对状态6连击（210等效威力）
- 抽枝 4能90 → 应对状态回50%血+5能

**先手/迅捷**：
- 先发制人 2能55 → 先手+1
- 疾风刺 2能25×3 → 先手3连击
- 针刺射击 3能80 → 换人时回7能

**续航**：
- 根吸收 3能 → 回血15%+4能
- 应激反应 2能 → 回血25%（应对防御50%）

## 六、推荐配招方向

### 路线A：雨天爆发流（炮台）
- 落雨 + 一拳 + 力量增效 + 水刃
- 落雨触发得寸进尺+100%，力量增效+100%=+200%，一拳140威力，水刃雨天+50%=172.5威力

### 路线B：队友雨天流（收割）
- 一拳 + 崩拳 + 力量增效 + 疾风刺
- 队友提供雨天环境，崩拳应对物攻+100%+得寸进尺+100%+力量增效+100%=+300%，一拳秒杀

### 路线C：应对爆发流（炮台）
- 爆冲 + 散手 + 抽枝 + 崩拳
- 爆冲应对状态5倍325威力，散手应对6连击，抽枝应对回血+5能，崩拳应对+物攻

## 八、擅长应对的情况

- 雨天/水天气队伍（得寸进尺被动触发）
- 水/地/电/恶系对手（武草双系克制）
- 低速高耐久对手（雨天+200%物攻爆发击穿）
- 换人频繁的对手（针刺射击回7能）

## 九、棘手的对手特点

- 翼系克星（3x弱点必死）
- 火/冰/毒/萌/幻系先手压制（5条2x弱点）
- 非雨天环境（得寸进尺失效）
- 驱散增益/天气的对手（雨天被清除）

## 十、队伍需求

### 精灵缺的
- 翼系克星清理（3x弱点必须处理）
- 冰/萌系克星清理
- 雨天支援（无队友雨天时需落雨占技能栏）

### 精灵给的
- 得寸进尺雨天双攻+100%（雨天队核心收割位）
- 极速253高速先手（82%分位）
- 武草双系打击面
- 应对状态技池完整（爆冲/无影脚/散手/抽枝）
- 水刃雨天+50%威力爆发

### 体系归属
- 雨天爆发队收割位
- 武草双系主题队
- 水系队合作位
