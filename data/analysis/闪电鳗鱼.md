## 四、特性「生物电」

> 携带的电系技能获得迸发：能耗-2。

所有携带的电系技能获得迸发效果（能耗-2）：

- **电弧联动**（3能80威力，迸发+40威力）：生物电迸发-2=1能80威力，再触发迸发+40=1能120威力
- **交叉闪电联动**（3能100威力）：生物电迸发-2=1能100威力
- **双联脉冲联动**（4能50，迸发次数+1）：生物电迸发-2=2能50威力+次数+1
- **雷暴联动**（1能55，迸发获得所有生效过的迸发效果）：生物电迸发-2=雷暴-1能，同时积累迸发种类
- **电离爆破联动**（3能，敌方速度-40）：生物电迸发-2=1能速度-40控场
- **核心价值**：所有电系技能能耗-2，配合迸发体系形成低费高效输出
- **局限性**：草/地2x弱点；HP85偏低；迸发需要积累

## 五、技能池亮点

**生物电迸发核心**：
- 交叉闪电 3能100 → 生物电迸发-2=1能100威力
- 电弧 3能80 → 生物电迸发-2=1能80威力+迸发+40=1能120威力
- 双联脉冲 4能50 → 生物电迸发-2=2能50+迸发次数+1
- 电离爆破 3能 → 生物电迸发-2=1能速度-40+3连击
- 雷暴 1能55 → 迸发获得所有生效迸发效果（生物电-2+电弧+40+双联脉冲次数+1）

**高威力输出**：
- 水光冲击 5能140 → 顶级魔攻
- 音爆 4能130 → 稳定魔攻
- 消毒法 4能115 → 驱散5层增益+魔攻

**防御续航**：
- 嗜痛 2能 → 减伤80%+应对双攻+40%
- 水泡盾 2能 → 减伤80%+应对魔攻+70%
- 硬化 2能 → 减伤90%（上回合攻击后-2能耗）
- 电磁偏转 3能 → 减伤70%+应对下回合次数+1
- 集中 2能 → 减伤80%+应对回合结束返场
- 血气 2能 → 减伤60%+应对致命伤保留1血

**控场**：
- 激怒 3能 → 能耗+3×3回合
- 耀眼 1能 → 敌方连击-4
- 远程访问 2能 → 强制敌方返场
- 强制重启 3能90 → 应对状态敌方返场

**资源**：
- 蓄水 1能 → 下次技能-6能耗
