# 锤头鹳 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 2维 | 双攻67/65偏弱；快锤低能耗技能迅捷+盐水浴全技能-2能耗 |
| 耐久 | 2维 | HP111+物防91；特防72偏脆 |
| 功能性 | 4维 | 快锤低能耗迅捷+飞羽0能驱散+盐水浴能耗-2+操控/聒噪能耗压制+恶意逃离中转 |

**主定位**：低能耗迅捷工兵（快锤使所有能耗<3技能获得迅捷+盐水浴降费）
**战术角色**：工兵 + 中转

## 二、种族值与速度

- HP111 / 攻67 / 特攻65 / 防91 / 特防72 / 速105（总511）
- **有效种族446**（去除特攻65）
- **极速240.2**（63%分位，中高速）/ 满速208.5
- 启动速度：⭐⭐⭐⭐⭐ 即时（快锤被动即时，鸣叫0能即可出手且迅捷）

## 三、属性抗性（水/翼双属性）

- **3x弱点**：电（致命）
- **2x弱点**：冰
- **半抗**：火、虫、武、机械
- **免疫**：无

## 四、特性「快锤」

> 携带的能耗小于3的技能，获得迅捷

构筑时带低能耗技能即触发：

- **0能技能**：鸣叫（40威力）、飞羽（驱散）、魔法增效（魔攻+70%）→ 全部迅捷
- **1能技能**：鹰爪（60威力）、蓄水（-6能耗）、咆哮（物攻-130%）→ 全部迅捷
- **2能技能**：风矢（80威力）、潮涌（80威力）、疾风刺（先手3连击）→ 全部迅捷
- **配合盐水浴**（全技能能耗-2）：原本3能的技能降至1能，也获得迅捷
- **核心价值**：换人入场即可使用迅捷技能，节奏极快
- **局限性**：低能耗技能威力有限；电系3x弱点致命

## 五、叠加型技能发育分析

**识别**：水炮（永久-1能耗）+ 乘胜追击（每次使用连击数永久+1）

- **水炮**：初始5能，每次使用-1。叠加3次→2能，触发快锤迅捷+110威力。
- **乘胜追击**：初始1连击，每次+1。叠加5次→6连击，配合迅捷先手爆发。
- **盐水浴加速**：盐水浴全技能-2能耗，水炮从5能直接降至3能，再使用1次即触发快锤。
- **队伍依赖度**：低。快锤被动即时，发育期自给自足。

## 六、技能池亮点

**快锤迅捷核心（能耗<3）**：
- 鸣叫 0能40 → 迅捷魔攻
- 飞羽 0能 → 迅捷+驱散敌方1种增益
- 魔法增效 0能 → 迅捷+魔攻+70%
- 鹰爪 1能60 → 迅捷物攻
- 咆哮 1能 → 迅捷+物攻-130%
- 风矢 2能80 → 迅捷物攻
- 潮涌 2能80 → 迅捷物攻
- 疾风刺 2能 → 迅捷+先手3连击

**盐水浴联动**：
- 盐水浴 2能 → 全技能能耗-2（应对防御-3），使3能技能降至1能获得迅捷

**控场**：
- 操控 1能 → 迅捷+敌方本回合技能能耗+7
- 精神扰乱 0能 → 迅捷+敌方能耗+1
- 聒噪 3能 → 敌方攻击技能耗+3持续3回合
- 恶意逃离 1能 → 迅捷+脱离+应对防御能耗+4

## 七、推荐配招方向

### 路线A：快锤迅捷压制流（工兵+收割）
- 飞羽 + 风矢 + 咆哮 + 恶意逃离
- 全部迅捷，飞羽驱散，咆哮-130%物攻，恶意逃离中转+能耗+4

### 路线B：盐水浴降费流（工兵）
- 盐水浴 + 操控 + 飞羽 + 风矢
- 盐水浴全技能-2能耗，操控迅捷+能耗+7，飞羽驱散，风矢迅捷输出

### 路线C：水炮发育流（炮台）
- 水炮 + 盐水浴 + 飞羽 + 魔法增效
- 盐水浴降费后水炮快速触发快锤迅捷，魔法增效+70%魔攻后水炮爆发
