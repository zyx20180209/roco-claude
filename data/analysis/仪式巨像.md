# 仪式巨像 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 4维 | 特攻95（52%）+观星每层星陨地系+15%+超新星馈赠叠加星陨+多维击打星陨连击 |
| 耐久 | 3维 | 双防115/119均衡+硬化/不动如山减伤90%+均势物防+140% |
| 功能性 | 3维 | 星轨裂变/超新星馈赠铺星陨+流沙禁换+沙涌天气+四维降解印记降费 |

**主定位**：星陨体系核心炮台（观星每层星陨地系技能+15%威力+星陨铺设工具最完整）
**战术角色**：炮台 + 工兵（星陨体系）

## 二、种族值与速度

- HP106 / 攻90 / 特攻95 / 防115 / 特防119 / 速85（总610）
- **有效种族520**（去除物攻90）
- **极速213.8**（30%分位，中低速）/ 满速186.5
- 启动速度：⭐⭐⭐⭐ 快速（观星被动即时，偷师0能即可出手）

## 三、属性抗性（地/幻双属性）

- **2x弱点**：草、水、冰、虫、幽、机械（6条偏多）
- **半抗**：普通、火、电、毒、翼、幻（6条）
- **免疫**：无

## 四、特性「观星」

> 敌方每有1层星图印记，自己的地系技能威力+15%

星陨层数越多，地系技能威力越高：

- 2层星陨 → 地系+30%
- 4层星陨 → 地系+60%
- 6层星陨 → 地系+90%
- 10层星陨 → 地系+150%
- **配合超新星馈赠**（每次使用星陨印记层数+1）：叠加后星陨层数快速增长
- **配合多维击打**（每层星陨连击+1）：星陨层数越多连击越高
- **配合四维降解**（每层印记能耗-1）：星陨层数越多四维降解越便宜
- **核心价值**：星陨体系中地系技能威力极高，陨石3能100→+150%=250威力
- **局限性**：需要铺设星陨才能触发；速度85偏慢；6条弱点偏多

## 五、叠加型技能发育分析

**识别**：超新星馈赠（每次使用赋予的星陨印记层数+1）

- **超新星馈赠**：初始2层星陨印记，每次使用+1。使用3次→5层，使用5次→7层。
- **观星联动**：7层星陨→地系技能+105%威力，陨石100→205威力。
- **四维降解联动**：7层印记→四维降解7能降至0能100威力。
- **队伍依赖度**：低。双防均衡+硬化/不动如山自保，发育期基本自给自足。

## 六、技能池亮点

**星陨铺设（最完整的星陨体系）**：
- 超新星馈赠 2能 → 2层星陨印记（每次使用+1，叠加后越来越多）
- 星轨裂变 1能 → 2层星陨（低费铺垫）
- 二律背反 4能 → 3层星陨（应对防御翻倍）
- 超维投射 4能 → 4层星陨
- 心灵洞悉 7能 → 星陨印记=敌方现有印记层数
- 空间压迫 3能70 → 1层星陨+物攻输出

**观星联动地系输出**：
- 陨石 3能100 → 观星+N%威力（星陨叠满后250+威力）
- 热砂 2能80 → 低费地系
- 落石 1能55 → 最低费地系触发观星

**多维击打联动**：
- 多维击打 4能15×N → 每层星陨连击+1（10层星陨=10连击）

**四维降解联动**：
- 四维降解 7能100 → 每层印记能耗-1（7层印记后0能100威力）

**防御**：
- 硬化 2能 → 减伤90%（攻击后-2能耗）
- 不动如山 2能 → 减伤90%

**控场**：
- 流沙 2能 → 禁换3回合（应对防御双防-60%）
- 沙涌 7能 → 沙暴地系能耗减半

## 七、推荐配招方向

### 路线A：超新星馈赠发育流（炮台）
- 超新星馈赠 + 陨石 + 多维击打 + 硬化
- 超新星馈赠叠加星陨层数，陨石观星+N%威力，多维击打星陨连击爆发

### 路线B：星陨爆发流（炮台+工兵）
- 星轨裂变 + 二律背反 + 陨石 + 四维降解
- 星轨裂变+二律背反快速铺星陨，陨石高威力，四维降解印记降费

### 路线C：沙暴地系流（炮台）
- 沙涌 + 陨石 + 超新星馈赠 + 不动如山
- 沙暴地系能耗减半，陨石3能→1.5能+观星加成

## 九、擅长应对的情况

- 电/火/毒/机械系对手（地系克制）
- 依赖增益积累的对手（晒太阳驱散）
- 消耗战节奏（超新星馈赠叠加星陨）
- 需要星陨体系支援的队伍

## 十、棘手的对手特点

- 水/冰/幽/机械系先手压制（6条2x弱点）
- 速度85偏慢，被高速精灵先手压制
- 带自清技能的对手（清除星陨印记）
- 高特攻爆发（特防119偏中）

## 十一、队伍需求

### 精灵缺的
- 水/冰/幽/机械系克星清理
- 速度补位（极速214偏慢）

### 精灵给的
- 星陨体系最完整铺设（超新星馈赠叠加+星轨裂变+二律背反+心灵洞悉）
- 观星地系技能+15%/层（陨石叠满后250+威力）
- 多维击打星陨连击爆发
- 四维降解印记降费
- 流沙禁换控场

### 体系归属
- 星陨体系绝对核心（不可替代）
- 地/幻系主题队
- 沙暴地系体系

## 十二、首领化形态：祭礼巨像

**种族值**：HP106 / 攻105 / 特攻111 / 防115 / 特防119 / 速85
**有效种族**：536（去除物攻105）
**特性「坠星」**：敌方每有1层星陨印记，自己技能威力+15%（不限地系）

### 观星 vs 坠星

| 对比 | 仪式巨像（观星） | 祭礼巨像（坠星） |
|------|------|------|
| 加成范围 | **仅地系技能** | **所有技能** |
| 加成幅度 | +15%/层 | +15%/层（相同） |
| 特攻 | 95 | **111** |
| 物攻 | 90 | **105** |
| 有效种族 | 520 | **536** |

**坠星的质变**：从"仅地系+15%"升级为"所有技能+15%"，陨石/多维击打/四维降解/坍缩等全部受益，输出天花板大幅提升。
