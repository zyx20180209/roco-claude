## 四、特性「共鸣」

> 携带的「虫鸣」技能威力+20

携带虫鸣即触发：

- 虫鸣（2能15×N）→ 共鸣后每连击威力+20（实际每连击35威力）
- **虫鸣机制**：队伍中每有1只精灵携带虫鸣，本次技能连击数+1
- 配合虫系队全员携带虫鸣：3只精灵各带虫鸣→虫鸣3连击+共鸣+20=3×35=105威力
- **核心价值**：虫鸣体系中共鸣提升虫鸣威力，配合多只精灵携带虫鸣后连击爆发
- **局限性**：需要虫系队友配合；HP93偏脆；火/翼系2x弱点

## 五、技能池亮点

**虫鸣联动核心**：
- 虫鸣 2能15×N → 共鸣+20威力=35/连击，队伍每带1个虫鸣+1连击
- 虫网 0能30 → 0能回1能，低费骚扰
- 虫蛊 3能100 → 稳定虫系魔攻
- 翅刃 4能95 → 驱散敌方所有印记（应对状态偷取印记）

**奉献联动**：
- 假寐 2能 → 回2能+奉献能耗-2（虫系队联动）
- 飞断 1能20 → 奉献触发+20威力

**能耗压制**：
- 聒噪 3能 → 敌方攻击技能耗+3持续3回合
- 摇篮曲 5能 → 敌方能耗+3+眩晕

**输出**：
- 触底强击 3能65 → 能耗耗尽后+110=175威力
- 魔能爆 0能 → 消耗所有能量，能量越多伤害越高
- 追打 3能75 → 应对状态3连击

**功能**：
- 倾泻 3能70 → 未被应对驱散双方印记
- 棘刺 2能 → 棘刺印记换人-6%血

## 六、推荐配招方向

### 路线A：虫鸣联动流（工兵+炮台，核心玩法）
- 虫鸣 + 虫蛊 + 假寐 + 翅刃
- 虫鸣共鸣+20威力，假寐奉献能耗-2，翅刃驱散印记

### 路线B：能耗压制流（工兵）
- 聒噪 + 摇篮曲 + 虫鸣 + 假寐
- 聒噪+摇篮曲彻底封锁能耗，虫鸣联动，假寐奉献

### 路线C：触底强击爆发流（炮台）
- 触底强击 + 魔能爆 + 虫鸣 + 假寐
- 触底强击能耗耗尽175威力，魔能爆消耗所有能量爆发
