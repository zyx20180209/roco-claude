**主定位**：冰火双系DOT破盾（灵魂灼伤冰系技能附带4层灼烧+火系技能附带2层冻结，双向DOT）
**战术角色**：破盾 + 工兵

## 二、种族值与速度

- HP115 / 攻87 / 特攻87 / 防84 / 特防107 / 速100（总580）
- **有效种族493**（去除特攻87）

- **3x弱点**：地（极危险）
- **2x弱点**：火、武
- **半抗**：草、光、虫、萌（4条）
- **免疫**：无

## 四、特性「灵魂灼伤」

> 冰系技能能使敌方获得4层灼烧，火系技能能使敌方获得2层冻结

每次使用冰/火系技能自动触发双向DOT：

- **冰系技能→4层灼烧**：暴风雪/冷风/打雪仗/霜降/冰点/冰墙等每次使用+4层灼烧
- **火系技能→2层冻结**：火焰切割/炎枪/易燃物质/爆裂飞弹等每次使用+2层冻结
- **双向DOT**：冰系技能叠灼烧（火系DOT），火系技能叠冻结（冰系DOT）
- 配合充分燃烧（灼烧翻倍+触发1次灼烧伤害）：灼烧层数翻倍爆发
- 配合冬至（暴风雪每回合+2层冻结）：持续触发灵魂灼伤+4层灼烧
- **局限性**：地系3x弱点致命；双攻87偏中等

## 五、技能池亮点

**灵魂灼伤联动**：
- 冷风 1能60 → 冰系+4层灼烧（低费高效）
- 暴风雪 3能85 → 冰系+4层灼烧+1层冻结
- 霜降 1能 → 4层冻结+灵魂灼伤（冰系技能触发）
- 冰点 2能 → 5层冻结（应对防御10层）
- 火焰切割 3能100 → 火系+2层冻结
- 炎枪 3能100 → 火系+2层冻结
- 易燃物质 3能30×2 → 每连击+2层灼烧+灵魂灼伤+2层冻结

**灼烧爆发**：
- 充分燃烧 3能 → 灼烧翻倍+触发1次灼烧伤害（爆发神技）
- 火焰护盾 2能 → 减伤70%+应对触发6层灼烧
- 引燃 2能 → 10层灼烧
- 天火 3能 → 10层灼烧（应对防御30层）

**清场/控场**：
- 焚烧烙印 3能 → 驱散双方所有印记+每层+5层灼烧
- 晒太阳 1能 → 驱散敌方增益
- 热身 1能 → 下次攻击翻倍（应对防御4倍）
- 冬至 7能 → 暴风雪天气每回合+2层冻结

## 六、推荐配招方向

### 路线A：灵魂灼伤双DOT流（破盾）
- 冷风 + 炎枪 + 充分燃烧 + 火焰护盾
- 冷风冰系+4层灼烧，炎枪火系+2层冻结，充分燃烧灼烧翻倍爆发

### 路线B：冰系冻结+灼烧流（破盾+工兵）
- 霜降 + 冰点 + 充分燃烧 + 火焰切割
- 霜降4层冻结，冰点10层冻结，充分燃烧灼烧翻倍，火焰切割+2层冻结

### 路线C：天火爆发流（破盾）
- 天火 + 充分燃烧 + 冷风 + 焚烧烙印
- 天火应对防御30层灼烧，充分燃烧翻倍，冷风+4层灼烧，焚烧烙印清印记
