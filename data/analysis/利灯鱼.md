## 四、特性「对流」

> 自己的能耗增加变为能耗降低；能耗降低变为能耗增加。

所有针对自己的能耗变化效果反转：

- **反摇篮曲**：摇篮曲给敌方能耗+3，对利灯鱼变为能耗-3（极品反制）
- **反精神扰乱**：精神扰乱能耗+1，对利灯鱼变为能耗-1
- **反能耗控场体系**：所有能耗增加技能对利灯鱼反而降费
- **与打湿联动**：打湿给自己湿润印记（全技能-1能耗），对流不影响自己主动获得的降费
- **与落雷联动**：落雷每次入场永久+20威力，多次入场后威力极高
- **核心价值**：天然克制能耗控场体系，对手越控能耗利灯鱼越强
- **局限性**：物防71偏低；草/地2x弱点；速度105中等被先手压制

## 五、核心战术：反能耗爆发

**反控场流**：
1. 对手使用摇篮曲（能耗+3）→ 对流反转=利灯鱼能耗-3
2. 润泽（7能）→ 魔攻+190%（对流后可能低费）
3. 水光冲击（5能140威力）→ 魔攻+190%爆发

**落雷成长流**：
1. 加大功率（4能）→ 自己脱离+队友回8能
2. 重入场→落雷永久+20威力
3. 多次入场后落雷威力极高

## 六、技能池亮点

**高威力输出**：
- 水光冲击 5能140 → 顶级水系魔攻
- 激流 4能120 → 稳定水系魔攻
- 落雷 4能100 → 每次入场永久+20威力（成长）
- 超导 3能95 → 迸发能耗-1（电系）
- 双联脉冲 4能50 → 迸发使用次数+1

**增益**：
- 润泽 7能 → 魔攻+190%（极限爆发）
- 打湿 4能 → 湿润印记全技能-1能耗
- 魔法增效 0能 → 魔攻+70%（0能增益）

**资源**：
- 加大功率 4能 → 自己脱离+队友回8能（极品资源技）
- 涌泉 6能60 → 能耗每-1威力+10（降费后爆发）

**控场**：
- 摇篮曲 5能 → 敌方全技能+3能耗+打断+眩晕
- 强制重启 3能90 → 应对状态回合结束敌方返场
- 伪造账单 1能 → 敌方回血变失血×2（先手+1）

**天气**：
- 落雨 5能 → 雨天水系+50%（8回合）

## 七、推荐配招方向

### 路线A：反控场爆发流（炮台核心）
- 润泽 + 水光冲击 + 落雷 + 加大功率
- 对流反能耗控场，润泽魔攻+190%，水光冲击140威力爆发，落雷每次入场成长，加大功率脱离回队友8能

### 路线B：雨天爆发流（炮台）
- 落雨 + 激流 + 水光冲击 + 打湿
- 落雨水系+50%，打湿全技能-1能耗，激流/水光冲击雨天爆发

### 路线C：控场反制流（拦截）
- 摇篮曲 + 强制重启 + 超导 + 伪造账单
- 摇篮曲对手能耗+3（对流自己-3），强制重启应对返场，伪造账单反回血
