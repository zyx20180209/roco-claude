# 乌拉塔 PVP 对战分析

**战术角色**：收割 + 炮台

## 四、特性「恶魔的晚宴」

> 主动击败敌方精灵时，自己获得双攻+50%。

每次主动击败敌方精灵后双攻+50%（永久叠加）：

- **连续击败滚雪球**：击败1次=+50%，击败2次=+100%，击败3次=+150%
- **与怒火联动**（双攻+120%）：怒火+120%+恶魔的晚宴+150%=双攻+270%
- **与力量增效联动**（物攻+100%）：三重叠加=双攻+370%
- **与流星火雨联动**（每次击败永久+75威力）：击败3次=流星火雨75+225=300威力
- **与趁火打劫联动**（每次击败永久+2连击）：击败3次=趁火打劫35×8连击=280等效威力
- **核心价值**：每次击败都变强，连续击败后双攻极高
- **局限性**：速度55极慢；水/光/地/武4条弱点；特防70偏低；需要主动击败

## 五、叠加型技能发育分析

**流星火雨**（每次击败永久+75威力）：
- 初始75威力，击败1次=150，击败2次=225，击败3次=300威力
- 配合恶魔的晚宴双攻叠加，后期爆发极强

**趁火打劫**（每次击败永久+2连击）：
- 初始2连击，击败1次=4连击，击败2次=6连击，击败3次=8连击
- 35×8=280等效威力，配合恶魔的晚宴双攻叠加

## 六、技能池亮点

**击败叠加核心**：
- 流星火雨 3能75 → 每次击败永久+75威力（击败3次=300威力）
- 趁火打劫 3能35×N → 每次击败永久+2连击（击败3次=8连击）
- 恶魔的晚宴（被动）→ 每次击败双攻+50%

**高威力输出**：
- 极限撕裂 4能135 → 高威力（生命>50%时使用后双攻-50%）
- 火焰切割 3能100 → 稳定物攻
- 跌落 3能120 → 应对状态物攻+50%（否则-50%）
- 气势一击 3能80 → 上回合应对成功后+180=260威力

**增益**：
- 怒火 1能 → 双攻+120%+双防-40%
- 力量增效 1能 → 物攻+100%
- 伺机而动 1能 → 下次攻击+70威力
- 贪婪 2能 → 100%吸血

**应对爆发**：
- 闪燃 1能40 → 应对状态4倍=160威力
- 崩拳 2能65 → 应对状态物攻+100%
- 气势一击 3能80 → 上回合应对后+180=260威力

**控场/脱离**：
- 恶意逃离 1能 → 脱离+应对防御攻击技能+4能耗
- 当头棒喝 3能80 → 换人时+100=180威力
- 力量吞噬 4能 → 敌方威力-20+自己+20
