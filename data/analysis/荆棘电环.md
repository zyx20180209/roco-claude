## 四、特性「防过载保护」

> 每次行动后脱离。

每次使用技能后都强制脱离：

- **强制脱离机制**：无论用什么技能（攻击/状态/防御），行动结束必定脱离
- **避免被针对**：对手先手打不到；每次只能留1回合，极难被秒
- **与落雷联动（每次入场永久+20威力）**：入场+行动+脱离+再入场=落雷滚雪球核心机制
- **与操控联动（1能能耗+7）**：操控1能极低费，打完立刻脱离
- **与魔能爆联动（消耗所有能量）**：一次能量大爆发后脱离返回聚能
- **与欺诈契约联动（交换增益减益）**：交换后立刻脱离，对手无法还手
- **与勾魂/惊吓盒子联动（偷能/减能）**：偷能后脱离，对手下回合能量更少
- **核心价值**：打完就跑的游击战术，每次入场都带来一次性高价值操作
- **局限性**：无法连续战斗；依赖多次入场循环；魔力（全队4点）限制脱离次数；需要队友配合

## 五、核心战术：入场+操作+脱离循环

**典型流程**：
1. 入场→操控（1能锁对手能耗+7×3回合）→脱离
2. 队友战斗→对手能量不足→荆棘电环再入场
3. 入场→落雷（100+20=120威力滚雪球）→脱离
4. 再次循环→落雷140/160/180...威力滚雪球

## 六、技能池亮点

**入场滚雪球核心**：
- 落雷 4能100 → 每次入场永久+20威力（防过载脱离循环=反复+20）

**能耗控场（1能低费打完就跑）**：
- 操控 1能 → 敌方本回合技能能耗+7×3回合（极低费控场神技）
- 精神扰乱 0能 → 全技能能耗+1（应对防御+3）
- 勾魂 1能 → 偷取敌方3能量

**控场/减益**：
- 咆哮 1能 → 敌方物攻-130%
- 晒太阳 1能 → 驱散敌方所有增益
- 麻痹 2能 → 敌方先手-1+应对防御双攻-70%
- 欺诈契约 3能 → 交换增益和减益
- 冰点 2能 → 敌方5层冻结（应对防御10层）
- 速冻 4能 → 2层减速印记

**应对/中断**：
- 强制重启 3能90 → 应对状态敌方返场（配合己方另一个精灵回场保护）
- 阻断 2能80 → 应对状态打断

**资源管理**：
- 加大功率 4能 → 主动脱离+替换入场精灵回8能（队友能量支援）
- 魔能爆 0能1 → 消耗所有能量爆发（脱离前清空能量）

**输出**：
- 交叉闪电 3能100 → 稳定物攻
- 灵媒 3能100 → 稳定魔攻
- 电弧 3能80 → 迸发+40威力=120
- 先发制人 2能55 → 先手+1

- 落雷每入场永久+20威力（多次循环后120/140/160），操控锁场，晒太阳驱增益
