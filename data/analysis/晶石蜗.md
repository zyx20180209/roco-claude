## 三、属性抗性（光/地双属性）

- **2x弱点**：水、冰（地系弱点）
- **半抗**：普通、火、电、毒、翼（地系半抗）
- **免疫**：无

## 四、特性「偏振」

> 受到自己携带技能系列的攻击伤害-40%

构筑时选择什么属性的技能，就对该属性攻击减伤40%：

- 携带光系技能 → 光系攻击-40%
- 携带地系技能 → 地系攻击-40%
- 携带光+地技能 → 光系和地系攻击均-40%
- **核心价值**：主动定制抗性，针对热门属性构筑
- **局限性**：只减伤40%，不是免疫；速度65极速187极慢

## 五、共同技能池亮点

**输出**：
- 闪光冲击 光系 → 主力输出
- 脉冲光线 光系 → 光系输出
- 跺地 地系 → 地系输出
- 地刺 3能95 → 应对状态打断

**防御**：
- 壁垒 2能 → 减伤90%+防御技CD-1
- 淤泥表皮 2能 → 减伤80%+敌方连击数-3
- 漫反射/镜像反射 → 减伤+反射

**增益/控场**：
- 魔法增效 0能 → 魔攻+70%
- 蓄势待发 4能 → 攻击印记+全技能威力+30%
- 锐利眼神 → 破盾

## 七、叠加型技能发育分析（星彩榴石）

**识别**：吹火（每次使用威力永久+20）

- **发育曲线**：吹火每次使用威力+20，叠加5次后威力大幅提升。
- **偏振联动**：携带火系技能→偏振抗火系攻击-40%，发育期自保能力强。
- **队伍依赖度**：低。偏振自保+壁垒减伤90%，发育期基本自给自足。

## 八、推荐配招方向

### 西瓜碧玺：冰系控场流
- 冰墙 + 冰雹 + 闪光冲击 + 壁垒
- 冰墙应对触发冻结，冰雹应对状态能耗+3，偏振抗冰系

### 星彩榴石：吹火发育流
- 吹火 + 火焰护盾 + 闪光冲击 + 壁垒
- 吹火永久叠加，火焰护盾应对触发灼烧，偏振抗火系

### 火山琉璃：偷能拦截流
- 报复 + 地刺 + 闪光冲击 + 壁垒
- 报复应对触发偷3能，地刺打断，偏振抗幽系

### 蓝锥矿：星陨控场流
- 冥想 + 空间压迫 + 能量守恒 + 闪光冲击
- 冥想应对触发星陨，能量守恒降低两侧技能能耗

### 烧蓝黄金：电系输出流
- 超导 + 电磁偏转 + 闪光冲击 + 壁垒
- 超导迸发递减，电磁偏转+1使用次数，偏振抗电系
