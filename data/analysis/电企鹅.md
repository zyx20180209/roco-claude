## 四、特性「超负荷」

> 攻击技能获得迸发:敌方获得全技能能耗+1

所有攻击技能都附带迸发效果——对手全技能能耗+1：

- 电流 0能40 → 迸发敌方+1能耗
- 超导加速 2能70 → 迸发+自己速度+30
- 感电 2能60 → 迸发+每离场使用次数+1
- 落雷 4能100 → 迸发+每次入场永久+20
- 冰锋横扫 4能0 → 迸发+威力等于敌方技能能耗×10
- **核心价值**：持续给对手叠加能耗+1，配合聒噪（+3能耗）/摇篮曲（+3能耗）形成能耗封锁
- **冰锋横扫联动**：超负荷+聒噪后敌方技能能耗大幅提升，冰锋横扫威力爆发
- **局限性**：地系3x弱点致命；HP94+物防73偏脆

## 五、叠加型技能发育分析

**识别**：落雷（每次入场永久+20威力）+ 感电（每次离场使用次数永久+1）

- **落雷**：初始100威力，每次入场+20。入场3次→160威力。
- **感电**：每次离场使用次数+1，中转循环中叠加。
- **队伍依赖度**：中等（中转流需要配合过载回路自返场）。

## 六、技能池亮点

**超负荷联动输出**：
- 电流 0能40 → 低费迸发+1能耗
- 超导加速 2能70 → 迸发+速度+30（极速266→296）
- 冰锋横扫 4能0 → 能耗×10，超负荷后威力翻倍
- 落雷 4能100 → 入场+20发育
- 极寒领域 6能105 → 冻结时+60威力

**冰系控场**：
- 霜冻 1能 → 敌方魔防-100%
- 寒风吹 3能70 → 敌方魔防-50%
- 冰点 2能 → 敌方5层冻结（应对防御再5层）
- 雾气环绕 1能 → 回能=敌方技能能耗的一半

**能耗压制**：
- 超负荷（被动）→ 攻击技能迸发+1能耗
- 聒噪 3能 → 敌方攻击技能耗+3持续3回合
- 摇篮曲 5能 → 敌方能耗+3+眩晕

**功能**：
- 过载回路 1能 → 自己返场（触发感电/落雷）
- 强制重启 3能90 → 应对状态使敌方返场
- 杠杆置换 0能 → 回2能+交换两侧技能位置
- 取念 0能 → 复制敌方技能-2能耗

## 七、推荐配招方向

### 路线A：超负荷能耗压制流（炮台+工兵）

### 路线B：冰冻爆发流（炮台）

- 冰点5层冻结，霜冻魔防-100%，极寒领域威力+60爆发

### 路线C：中转发育流（工兵+炮台）
- 过载回路 + 落雷 + 感电 + 超导加速
- 过载回路返场触发感电使用次数+1+落雷威力+20
