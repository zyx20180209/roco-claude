# 嘟嘟锅 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 3维 | 特攻102（62%）+毒液渗透中毒层越多能耗越低+以毒攻毒每层中毒魔攻+30%+毒雾转化增益为中毒 |

**主定位**：毒系破盾专精（复方汤剂+多重中毒叠加+能量压制）
**战术角色**：破盾 + 工兵（中毒DOT+能耗压制）

## 二、种族值与速度

- HP137 / 攻36 / 特攻102 / 防89 / 特防104 / 速105（总573）

## 三、属性抗性（毒系）

- **2x弱点**：地、恶、幻（3条）
- **半抗**：草、毒、虫、武、萌（5条）
- **免疫**：无

## 四、特性「复方汤剂」

> 回合结束时，中毒效果触发次数+1

每回合结束中毒层结算次数+1：

- 正常中毒每回合扣血1次，复方汤剂使扣血次数+1，即每层中毒造成的伤害翻倍
- 配合毒孢子（5层）/剧毒（3-8层）/腐蚀酸液（2层）：中毒层数可快速叠加
- 配合以毒攻毒（每层中毒魔攻+30%）：5层中毒→魔攻+150%
- 配合毒液渗透（中毒层越多能耗越低）：5层中毒→毒液渗透从5能降至0能120威力
- **毒系免疫中毒**：嘟嘟锅自身免疫中毒DOT
- **局限性**：对毒系精灵无效（互相免疫）；地/恶/幻系2x弱点是硬伤

## 五、技能池亮点

**中毒叠加核心**：
- 毒孢子 3能 → 敌方5层中毒
- 剧毒 2能 → 敌方3层中毒（应对防御8层）
- 腐蚀酸液 2能35 → 敌方2层中毒+魔伤
- 疫病吐息 3能 → 1层中毒印记（场地层面，换人后保留）

**毒系联动输出**：
- 毒液渗透 5能120 → 敌方每层中毒能耗-1（叠5层后0能120威力）
- 以毒攻毒 1能 → 每层中毒魔攻+30%（5层后+150%）
- 瘴气喷射 3能100 → 稳定魔攻
- 毒泡泡 0能40 → 低费消耗

**功能**：
- 落井下毒 6能 → 敌方减益层数翻倍（与中毒联动爆炸性倍增）
- 毒雾 7能 → 将敌方所有增益转化成中毒
- 晒太阳 1能 → 驱散敌方增益
- 取念 0能 → 复制敌方技能-2能耗

**控场**：
- 精神扰乱 0能 → 敌方能耗+1
- 激怒 3能 → 敌方能耗+3持续3回合
- 摇篮曲 5能 → 敌方能耗+3+眩晕

## 六、推荐配招方向

### 路线A：复方汤剂破盾流（破盾）
- 毒孢子 + 以毒攻毒 + 毒液渗透 + 晒太阳
- 毒孢子铺5层中毒→复方汤剂使扣血翻倍→以毒攻毒魔攻+150%→毒液渗透0能120威力

### 路线B：剧毒爆发流（破盾）
- 剧毒 + 毒液渗透 + 以毒攻毒 + 腐蚀酸液
- 剧毒应对防御叠8层中毒，毒液渗透降费终结

### 路线C：毒雾转化流（工兵+破盾）
- 毒雾 + 落井下毒 + 毒液渗透 + 晒太阳
- 毒雾转化敌方增益为中毒，落井下毒减益翻倍，毒液渗透终结
