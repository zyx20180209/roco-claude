# 权杖-V PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 2维 | 双攻105/97均衡中等+传动机械技能循环+联动装置两侧+20/30威力 |
| 耐久 | 5维 | HP103+物防136（95%）+特防136（95%）+无畏之心减伤100%+机械10条抗性+加固/鼓劲 |

**战术角色**：拦截（赖场消耗）

## 二、种族值与速度

- HP103 / 攻105 / 特攻97 / 防136 / 特防136 / 速75（总652）
- **有效种族555**（去除特攻97）
- **极速200.6**（19%分位，极慢）/ 满速175.5

## 三、属性抗性（机械系）

- **2x弱点**：火、水、武（3条）
- **半抗**：普通、草、冰、龙、毒、虫、翼、萌、机械、幻（**10条半抗**，全游戏最多）
- **免疫**：无

## 四、特性「机械要式」

> 自己携带的技能每回合位置变化时，该技能能耗-1

核心机制：机械系传动机制让技能每回合自动向下移动位置，触发机械要式能耗-1：

- **传动技能**：械斗/钢铁洪流/离子震荡/磁暴/啮合传递/轴承支撑（传动1-2）
- 每回合技能位置变化 → 能耗-1
- 配合杠杆置换（交换两侧技能位置+回2能）主动触发位置变化
- 配合联动装置（两侧技能威力+20/30）：传动+联动装置形成威力滚雪球
- 配合轴承支撑（两侧技能能耗-1）：传动+轴承支撑双重降费
- **核心价值**：机械系技能随时间越变越便宜+越强
- **局限性**：需要传动技能才能触发；速度75极慢

## 五、社区共识

**社区定位**：赖场消耗型联防

**典型配招**：啮合传递 + 钢铁洪流 + 无畏之心 + 补盲技能

**核心价值**：依靠双防136+机械10条抗性长时间赖场，无畏之心100%减伤+转治疗，啮合传递补速+叠攻，钢铁洪流主输出。

**关键弱点**：怕百分比叠加状态——中毒、冰冻、灼烧等。如果对方不能上百分比状态，权杖-V可以长期赖场输出。

## 六、技能池亮点

**机械传动核心**：
- 钢铁洪流 3能70 → 1号位时威力+90=160（传动2）
- 械斗 1能45 → 1号位+60=105威力（传动1）
- 离子震荡 3能90 → 3号位时+40=130（传动1）
- 磁暴 2能70 → 1号或3号位+30=100（传动1）

**机械工具**：
- 杠杆置换 0能 → 回2能+交换两侧技能位置（主动触发机械要式）
- 联动装置 0能 → 两侧技能威力永久+20（应对防御+30）
- 轴承支撑 3能 → 两侧技能能耗-1+传动1（自降费+传动）
- 啮合传递 1能 → 速度+80+1/3号位+60%物攻（攻速双修）

**顶级防御**：
- 无畏之心 5能 → 减伤100%+回血（应对触发）
- 加固 2能 → 物防+140%
- 鼓劲 3能 → 魔防+170%

**工具**：
- 棘刺 2能 → 棘刺印记换人-6%血
- 彗星 0能240 → 消耗全部生命换爆发

## 七、推荐配招方向

## 九、擅长应对的情况
