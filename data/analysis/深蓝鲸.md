## 四、特性「倾轧」

> 携带的技能受能耗变化效果的影响翻倍。

所有能耗变化效果翻倍：

- **蓄水联动**（下次技能-6能耗）：倾轧翻倍=-12能耗（天洪7能→-5能，实际0能）
- **水炮联动**（每次使用永久-1能耗）：倾轧翻倍=每次使用永久-2能耗（5次=0能）
- **天洪联动**（应对状态永久-6能耗）：倾轧翻倍=应对后永久-12能耗（天洪7能→-5能）
- **水环联动**（应对-2能耗）：倾轧翻倍=应对后-4能耗
- **雪替身联动**（应对回能=被应对技能能耗×2）：倾轧翻倍=回能×4
- **涌泉联动**（能耗每-1威力+10）：倾轧让能耗降得更快=涌泉威力更高
- **核心价值**：所有降费效果翻倍，高费大招极速降至0能
- **局限性**：速度50极慢；草/电2x弱点；需要多回合积累

## 五、核心战术：倾轧降费极限

**降费流程**：
1. 蓄水（-6能耗）→ 倾轧翻倍=-12能耗 → 天洪7能→-5能（实际0能）
2. 水炮每次使用-2能耗（倾轧翻倍）→ 5次=0能110威力
3. 天洪应对-12能耗 → 天洪永久0能150威力
4. 水环应对-4能耗 → 全技能-4能耗

## 六、技能池亮点

**倾轧降费核心**：
- 蓄水 1能 → 下次技能-6能耗（倾轧=-12能耗）
- 水炮 5能110 → 每次使用永久-1能耗（倾轧=-2能耗/次，5次=0能）
- 天洪 7能150 → 应对状态永久-6能耗（倾轧=-12能耗，天洪0能）
- 水环 2能 → 减伤60%+应对-2能耗（倾轧=-4能耗）
- 雪替身 2能 → 减伤70%+应对回能=被应对技能能耗×2（倾轧×4）

**高威力输出（降费后）**：
- 天洪 7能150 → 倾轧降费后0能150威力
- 音爆 4能130 → 稳定魔攻
- 激流 4能120 → 稳定魔攻
- 水炮 5能110 → 倾轧降费后0能110威力
- 涌泉 6能60 → 能耗每-1威力+10（倾轧降费后威力极高）
- 魔能爆 0能1 → 消耗所有能量爆发

**防御续航**：
- 水泡盾 2能 → 减伤80%+应对魔攻+70%
- 防御反击 3能 → 减伤80%+应对全技能威力+40
- 冰墙 2能 → 减伤80%+应对2层冻结
- 有效预防 1能 → 减伤50%+应对先手+1
- 防反 2能 → 减伤60%+应对双攻+40%

**增益**：
- 魔法增效 0能 → 魔攻+70%
- 落雨 5能 → 雨天水系+50%威力

## 七、推荐配招方向

### 路线A：倾轧降费爆发流（炮台核心）
