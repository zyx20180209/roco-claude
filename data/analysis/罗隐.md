- **3x弱点**：武（极危险）

- **半抗**：普通、火、电、毒、翼、幽、恶（7条）
- **免疫**：无

## 四、特性「石头大餐」

> 能量不足时，消耗5%生命，代替1能量

能量耗尽时自动消耗5%生命代替1能量：

- 配合高费技能（地震10能/岩土暴击8能/彼岸之手10能）：能量不足时用生命续战
- 配合贪婪（100%吸血）：攻击回血抵消生命消耗，形成自给自足循环
- 配合蝙蝠（吸血100%）：低费吸血维持生命
- **核心价值**：永远不会因能量耗尽而无法出手，高费大招可以持续释放
- **局限性**：生命消耗过快时会被秒；特防70极脆；速度75极慢

## 五、叠加型技能发育分析

**识别**：岩土暴击（8能，每被攻击1次永久-1能耗）+ 彼岸之手（10能，每失去10%生命能耗-1）

- **岩土暴击**：初始8能，每被攻击-1。配合石头大餐吸引攻击，被攻击8次后趋近0能140威力。
- **彼岸之手**：初始10能，每失去10%生命-1。石头大餐消耗生命同时触发彼岸之手递减，形成联动。
- **发育阶段风险**：特防70极脆，魔攻爆发可绕过物防直接秒杀。
- **队伍依赖度**：低。石头大餐自给续战，发育期基本自给自足。

## 六、技能池亮点

**顶级物攻输出**：
- 地震 10能190 → 石头大餐续能后释放
- 岩土暴击 8能140 → 被攻击递减后低费
- 彼岸之手 10能150 → 失血递减后低费

**吸血续航**：
- 贪婪 2能 → 100%吸血（抵消石头大餐生命消耗）
- 蝙蝠 2能65 → 吸血100%
- 暗突袭 4能70 → 吸血50%+应对状态翻倍

**增益**：
- 力量增效 1能 → 物攻+100%
- 泥浆铠甲 2能 → 物攻物防+60%（应对防御翻倍）
- 怒火 1能 → 双攻+120%双防-40%
- 力量吞噬 4能 → 敌方威力-20+自己威力+20

**防御**：
- 均势 3能 → 物防+140%+速度-30

- 地陷 5能80 → 物防+70%（应对状态翻倍）
