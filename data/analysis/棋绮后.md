# 棋绮后 PVP 对战分析

## 一、形态说明

棋绮后白子/黑子数值完全一致（总508），仅外观差异。
**首领化形态**：棋契陛下（不可独立入队）
**棋骑系列关系**：棋绮后是棋骑士/棋齐垒/棋祈督触发变身特性后的"棋骑后"共同形态，但也作为独立精灵可入队。

## 二、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 3维 | 双攻84均衡+渗透队友武/地技能×+5%攻防入场+岩脉崩毁固定4能120+抛石永久-5能耗 |
| 耐久 | 2维 | HP93+物防82+石肤术物防+160% |
| 功能性 | 4维 | 渗透队伍构筑支援+摇篮曲能耗+3+放晴光系永久+50%+破绽双防-70% |

**主定位**：队伍构筑转化型中转（渗透队友武/地技能积累×+5%攻防入场+均衡面板+4能120固定威力）
**战术角色**：中转 + 炮台

## 三、种族值与速度

- HP93 / 攻84 / 特攻84 / 防82 / 特防75 / 速90（总508）
- **有效种族424**（双攻均衡取84）

- 启动速度：⭐⭐⭐⭐ 快速（渗透入场即结算，队友武/地技能越多威力越高）

## 四、属性抗性（地/武双属性）

- **2x弱点**：草、水、冰、武、萌、机械、幻（7条极多）
- **半抗**：普通、火、地、电、毒、虫、恶（7条）
- **免疫**：无

## 五、特性「渗透」

> 己方精灵每使用1次武系或地系技能，自己入场时获得攻防+5%。

队伍武/地技能使用次数转化为入场时攻防加成：

- **队伍构筑要求**：地武双系主题队，队友大量使用武/地技能
- **数值成长**：队友每使用1次武/地技能=棋绮后入场攻防+5%
- **层数示例**：队友使用10次武/地技能=+50%攻防；20次=+100%攻防
- **中转位价值**：作为2-3号位出场时，前面队友已积累大量层数
- **同时+攻+防**：攻防各+5%/层，双向加成（物攻+物防/魔攻+魔防？描述未明确）
- **核心价值**：棋骑系列主题队收尾位，队友越多武/地技能棋绮后越强
- **局限性**：完全依赖队伍构筑；非棋骑/地武系队伍时特性失效；7条2x弱点极多

## 六、棋骑系列关系

棋绮后是棋骑士/棋齐垒/棋祈督变身特性触发后的"棋骑后"形态：

| 精灵 | 变身特性 | 战术角色 |
|------|---------|---------|
| 棋骑士 | 腾挪（攻击应对1次）| 输出/先手 |
| 棋齐垒 | 保卫（防御应对2次）| 拦截 |
| 棋祈督 | 好象坏象（状态被应对1次）| 工兵 |
| **棋绮后**| **渗透（队友武/地技能→攻防+5%/次）**| **中转/炮台**|

棋绮后特性是"被动继承"，与前三者的"主动应对变身"不同，更适合作为队伍构筑的收尾位。

## 七、技能池亮点

**高威力输出**：
- 岩脉崩毁 4能120 → 固定4能120威力（高性价比，渗透+50%后=180威力等效）

- 影袭 3能100 → 稳定武系

**增益**：

- 破绽 2能 → 敌方双防-70%+应对防御物攻+70%

- 蓄势待发 1能 → 蓄势印记全攻击技+30%威力
- 石肤术 2能 → 物防+160%+魔防-60%

- 化劲 2能 → 全技能威力+40
- 提气 4能 → 全技能威力+40（换人+50）

**控场**：
- 摇篮曲 5能 → 能耗+3+应对防御打断+眩晕
- 晒太阳 1能 → 驱散敌方增益

**天气**：
- 放晴 3能 → 光系永久+50%（应对防御+100%）

**应对防御**：

**续航**：

## 八、推荐配招方向

### 路线A：渗透收尾流（中转+炮台）
- 岩脉崩毁 + 力量增效 + 破绽 + 抛石
- 渗透入场攻防+N%，力量增效+100%，破绽双防-70%，岩脉崩毁固定4能120威力（渗透加成后等效更高）

### 路线B：增益爆发流（炮台）
- 力量增效 + 破绽 + 石肤术 + 一拳
- 渗透入场+攻防+力量增效+破绽减防+石肤术+160%物防联防，一拳140威力爆发

### 路线C：天气联动流（炮台）
- 放晴 + 天光 + 力量增效 + 岩脉崩毁
- 放晴光系永久+50%，天光随天气切换系别，岩脉崩毁稳定4能120威力

## 十、擅长应对的情况

- 地武系主题队（渗透核心联动）
- 火/电/毒/机械系对手（地系克制）
- 普通/冰/恶系对手（武系克制）
- 消耗战节奏（岩脉崩毁4能120威力持续输出）

## 十一、棘手的对手特点

- 草/水/冰/武/萌/机械/幻系克星（7条2x弱点）
- 非棋骑/地武系队伍（渗透失效）
- 高速先手精灵（面板偏脆无保护）
- 驱散增益的对手（渗透攻防被清）

## 十二、队伍需求

### 精灵缺的
- 冰/水/萌/幻系克星清理（7条弱点多）
- 队友武/地技能支持（渗透核心需求）

### 精灵给的
- 渗透队友武/地技能转化入场攻防（棋骑系主题收尾位）
- 岩脉崩毁固定4能120威力（高性价比输出）
- 摇篮曲能耗+3控场
- 放晴光系永久+50%天气支援
- 破绽双防-70%破盾工具

### 体系归属
- 棋骑系主题队中转/收尾位（核心定位）
- 地武双系主题队
- 构筑依赖型炮台

## 十三、首领化对比

**棋契陛下**（首领形态）：棋骑士/棋齐垒/棋祈督/棋绮后首领化后的共同形态，不能独立入队，需通过首领化机制获取。
