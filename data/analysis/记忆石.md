# 记忆石 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 3维 | 物攻108（63%）+不移纯伤技+30%威力+力量增效+100%物攻+蓄势印记全技能威力+30% |
| 耐久 | 3维 | HP112+物防141（97%顶级）+均势物防+140%+泥浆铠甲物防+60% |
| 功能性 | 1维 | 沙暴天气+地系联动；功能技较少 |

**主定位**：纯物攻炮台（不移纯伤技+30%+物防141顶级联防，以简单粗暴的高威力地系技能为主）
**战术角色**：炮台 + 拦截

## 二、种族值与速度

- HP112 / 攻108 / 特攻97 / 防141 / 特防79 / 速70（总607）
- **有效种族510**（去除特攻97）
- **极速194.0**（15%分位，极慢）/ 满速170.0
- 启动速度：⭐⭐⭐ 中速（不移被动即时，但速度70极慢）

## 三、属性抗性（地系）

- **2x弱点**：草、水、冰、武、机械（5条）
- **半抗**：普通、火、电、毒、翼（5条）
- **免疫**：无

## 四、特性「不移」

> 携带的无额外效果的攻击技能，威力+30%

**关键**：技能描述必须是**无额外效果**的纯伤害技能才能触发+30%：

- **触发**（无额外效果纯伤技）：
  - 跺地 2能80 → **104威力**
  - 扬沙 1能60 → **78威力**
  - 陨石 3能100 → **130威力**
  - 践踏 4能130 → **169威力**
  - 地震 10能190 → **247威力**
- **不触发**（有额外效果的）：
  - 裂石（应对状态-80%物防）
  - 地陷（应对状态翻倍）
  - 蓄势待发（印记）
  - 偷袭/抽枝/光合作用等带条件效果的
- **核心价值**：简单粗暴的纯伤害技能+30%威力，配合物攻108高输出
- **局限性**：限制技能选择（带额外效果的技能不触发）；速度70极慢

## 五、叠加型技能发育分析

**识别**：抛石（30能，每回合结束能耗永久-5）

- **抛石**：初始30能（超高），每回合结束永久-5。需6回合才趋近0能140威力。
- **发育风险**：30能起步太高，实战几乎不可能直接释放，必须依赖沙涌（沙暴地系能耗减半）或回合结束累积。
- **结论**：抛石发育路线极难实操，实战主要靠跺地/陨石/地震稳定输出。

## 六、技能池亮点

**不移触发的纯伤技**：
- 跺地 2能80 → 104威力（+30%）
- 陨石 3能100 → 130威力（+30%）
- 地震 10能190 → 247威力（+30%，需要沙涌降费）
- 扬沙 1能60 → 78威力（低费稳定）

**增益**：
- 力量增效 1能 → 物攻+100%
- 蓄势待发 4能 → 攻击印记全技能威力+30%+能耗+1
- 泥浆铠甲 2能 → 物攻物防+60%（应对防御翻倍）
- 均势 3能 → 物防+140%+速度-30

**防御**：
- 地陷 5能80 → 物防+70%（应对状态翻倍+物防额外+70%）
- 均势 → 物防极高

**天气联动**：
- 沙涌 7能 → 沙暴天气（地系技能能耗减半）
- 地震10能→5能（沙暴后），触发不移后威力247

## 七、推荐配招方向

### 路线A：不移炮台流（炮台）
- 跺地 + 陨石 + 力量增效 + 蓄势待发
- 力量增效+100%物攻+蓄势+30%+不移+30%，跺地等效威力极高

### 路线B：沙暴地震流（炮台）
- 沙涌 + 地震 + 力量增效 + 蓄势待发
- 沙涌沙暴地系能耗减半（地震10能→5能），地震247威力+各种加成

### 路线C：均势联防流（拦截）
- 均势 + 泥浆铠甲 + 跺地 + 地陷
- 均势物防+140%+泥浆铠甲+60%，物防达顶级联防
