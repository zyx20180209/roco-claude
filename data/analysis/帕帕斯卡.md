## 四、特性「翼轴」

> 1号位技能获得迅捷和传动1。

1号位技能自动获得迅捷+传动1（每回合下移1位）：

- **俯冲猛击联动**（5能140威力）：1号位俯冲猛击迅捷140威力先手
- **疾风刺联动**（2能25×3，先手3连击）：1号位疾风刺迅捷+先手3连击
- **龙卷风联动**（5能70，自带迅捷+应对1.5倍）：1号位龙卷风双重迅捷
- **能量刃联动**（3能80，应对永久+90威力）：1号位能量刃迅捷+应对叠加
- **传动1效果**：1号位技能每回合下移1位，下回合到2号位失去迅捷
- **杠杆置换联动**（0能交换两侧技能）：手动将目标技能移回1号位=再次获得迅捷
- **核心价值**：1号位技能迅捷先手，配合传动循环和杠杆置换持续获得迅捷
- **局限性**：物攻68偏低；火/水/电3条弱点；需要管理技能位置

## 五、技能池亮点

**翼轴1号位迅捷核心**：
- 俯冲猛击 5能140 → 1号位迅捷140威力先手
- 疾风刺 2能25×3 → 1号位迅捷先手3连击
- 龙卷风 5能70 → 1号位双重迅捷+应对1.5倍
- 能量刃 3能80 → 1号位迅捷+应对永久+90威力
- 风矢 2能80 → 1号位迅捷80威力

**传动位置联动**：
- 杠杆置换 0能 → 回2能+交换两侧技能（手动调整1号位）
- 轴承支撑 3能 → 主动-1能耗，被动两侧-1能耗，传动1
- 齿轮扭矩 3能80 → 每回合位置变化永久+20威力
- 钢钻 4能 → 两侧技能威力和×1/3，传动1
- 传感器 1能20×2 → 1/3号位连击+1，传动1

**防御续航**：
- 加固 2能 → 物防+140%
- 鼓劲 3能 → 魔防+170%
- 羽化加速 2能 → 威力+20+迅捷入场

**控场**：
- 精神扰乱 0能 → 能耗+1（应对防御+3）
- 过载回路 1能 → 返场+下回合次数+1

**应对**：
- 偷袭 3能85 → 应对状态3倍=255威力
