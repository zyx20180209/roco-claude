# 花衣蝶 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 1维 | 双攻67/72偏弱；飞断/啃咬受奉献影响输出有限 |
| 耐久 | 2维 | HP122（77%）+特防94；物防84偏中 |
| 功能性 | 5维 | 花精灵每回合随机奉献+虫群智慧2次奉献+虫茧奉献+飞断奉献+虫结阵应对奉献 |

**主定位**：虫系队奉献生产核心（花精灵每回合被动产奉献+虫群智慧批量产奉献）
**战术角色**：工兵（虫系队奉献引擎）

## 二、种族值与速度

- HP122 / 攻67 / 特攻72 / 防84 / 特防94 / 速100（总539）
- **有效种族472**（去除物攻67）
- **极速233.6**（54%分位，中速）/ 满速203.0
- 启动速度：⭐⭐⭐⭐⭐ 即时（花精灵回合结束被动，0能技能即可出手）

## 三、属性抗性（草/虫双属性）

- **3x弱点**：火、翼（双克3x，致命）
- **2x弱点**：冰、毒、虫
- **半抗**：草、水、光、地、电、武
- **免疫**：无

## 四、特性「花精灵」

> 回合结束时，己方队伍获得1次随机奉献

每回合结束自动生产1次随机奉献，无需出手：

- 持续在场即持续产奉献，是虫系队最稳定的奉献生产源
- 配合虫群智慧（3能，2次奉献）：单回合可产3次奉献
- 配合其他奉献生产技（飞断/虫茧/捆缚/虫群过境）：奉献积累速度极快
- **核心价值**：让虫系队友的奉献联动技连续触发（飞断+20威力/虫茧+10%吸血/捆缚+2中毒/虫群过境+1连击等）
- **局限性**：奉献是**随机**的（不特定类型）；本身输出弱依赖队友

## 五、核心战术：虫系奉献引擎

**花衣蝶的奉献产量（每回合）**：
- **花精灵被动**：1次随机奉献
- **虫群智慧**：2次随机奉献
- **虫茧/飞断/捆缚**：主动触发1次特定奉献
- **虫结阵**：应对攻击时触发1次随机奉献

**最大奉献积累**：单回合理论上可产5-6次奉献（花精灵+虫群智慧+主动技+虫结阵）

**虫系队联动技受奉献影响**（此部分技能与花魁蜂后/巨噬针鼹/女王蜂等共享）：
- 飞断 → +20威力
- 虫茧 → +10%吸血
- 捆缚 → 敌方+2中毒
- 假寐 → 能耗-2
- 虫群过境 → 连击数+1
- 虫群 → 受奉献影响（叠多次后爆发）
- 啃咬 → 受奉献影响（但能耗永久+1，慎用）

## 六、技能池亮点

**奉献生产核心**：
- 花精灵（被动）→ 每回合1次随机奉献
- 虫群智慧 3能 → 2次随机奉献
- 虫结阵 2能 → 减伤80%+应对触发1次奉献
- 飞断 1能20 → 奉献触发+20威力
- 虫茧 1能 → 回血20%+奉献触发+10%吸血

**输出（有限）**：
- 啃咬 0能40 → 受奉献影响（但能耗+1累积）
- 飞叶 0能40 → 低费草系
- 花香 1能60 → 低费草系魔攻

**功能**：
- 移花接木 2能 → 回血+脱离
- 晒太阳 1能 → 驱散敌方增益
- 食腐 2能 → 驱散敌方印记+回血
- 精神扰乱 0能 → 敌方能耗+1

**幻系技能石**：
- 星轨裂变 1能 → 2层星陨
- 二律背反 4能 → 3层星陨（应对防御翻倍）
- 四维降解 7能100 → 敌方每层印记能耗-1

## 七、推荐配招方向

### 路线A：虫系队奉献引擎流（工兵，核心玩法）
- 虫群智慧 + 虫茧 + 飞断 + 虫结阵
- 花精灵被动+虫群智慧2次+虫茧奉献+虫结阵应对，每回合疯狂产奉献

### 路线B：奉献+星陨复合流（工兵）
- 虫群智慧 + 二律背反 + 星轨裂变 + 虫茧
- 虫群智慧产奉献支援队友，二律背反星陨+应对翻倍，星轨裂变铺垫

### 路线C：奉献+控场流（工兵+清场）
- 虫群智慧 + 晒太阳 + 食腐 + 精神扰乱
- 奉献生产+晒太阳驱散+食腐清印记+精神扰乱骚扰

## 九、擅长应对的情况

- 虫系主题队（奉献引擎核心支援）
- 消耗战节奏（持续奉献累积优势）
- 印记体系（食腐驱散+回血）
- 需要支援队友输出的场面（奉献+20威力/+10%吸血等）

## 十、棘手的对手特点

- **火/翼系克星（3x致命）**——几乎必死
- 冰/毒/虫系先手压制（3条2x弱点）
- 无增益机制的对手（花衣蝶本身输出弱）
- 高速爆发秒杀（速度100中等但面板偏脆）

## 十一、队伍需求

### 精灵缺的
- **火/翼系克星清理**（双3x弱点必须先处理）
- 虫系队友（发挥奉献核心价值）
- 输出补位（自身面板偏弱）

### 精灵给的
- 花精灵+虫群智慧每回合批量产奉献（虫系队核心支援）
- 虫茧/飞断/虫结阵多重奉献生产
- 幻系星陨辅助（二律背反/星轨裂变）
- 晒太阳驱散+食腐清印记

### 体系归属
- 虫系专队奉献引擎核心（不可替代）
- 消耗战辅助位
