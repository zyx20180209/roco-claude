## 四、特性「警惕」

> 回合结束时，若自己能量为0则逃离。

回合结束时能量为0自动脱离：

- **魔能爆联动**（消耗所有能量爆发）：魔能爆消耗所有能量→回合结束能量0→警惕触发脱离
- **主动清空能量**：使用高费技清空能量后，警惕自动脱离，不消耗魔力
- **与抛石联动**（永久-5能耗）：抛石自递减后低费，配合警惕脱离循环
- **与石锁联动**（锁场3回合）：石锁锁场后警惕脱离，对手无法换人
- **核心价值**：能量耗尽后自动脱离，不消耗魔力，保留队伍资源
- **局限性**：需要主动清空能量才触发；5条2x弱点偏多；特防70偏低

## 五、技能池亮点

**警惕触发核心**：
- 魔能爆 0能1 → 消耗所有能量爆发（警惕触发脱离）
- 抛石 30能140 → 永久-5能耗（自递减后低费，清空能量触发警惕）

**高威力输出**：
- 岩脉崩毁 4能120 → 固定4能120威力
- 抛石 30能140 → 永久-5能耗（叠加后低费）
- 地刺 3能95 → 应对状态打断
- 裂石 3能95 → 应对状态物防-80%
- 偷袭 3能85 → 应对状态3倍=255威力

**控场**：
- 石锁 3能50 → 锁场3回合（警惕脱离后对手无法换人）
- 精神扰乱 0能 → 能耗+1（应对防御+3）
- 震击 3能80 → 敌方连击-3
- 晒太阳 1能 → 驱散敌方增益

**增益**：
- 泥浆铠甲 2能 → 物攻物防+60%+应对防御增益翻倍
- 三连破 1能 → 物攻+30%+3连击
- 蓄势待发 4能 → 蓄势印记全攻击技+30%威力

**天气**：
- 沙涌 7能 → 沙暴地系能耗减半（8回合）

**奉献**：
- 飞断 1能20 → 奉献威力+20

## 六、推荐配招方向

### 路线A：警惕脱离流（中转核心）
- 魔能爆 + 石锁 + 岩脉崩毁 + 精神扰乱
- 石锁锁场3回合，精神扰乱能耗+1，岩脉崩毁4能120威力，魔能爆消耗所有能量爆发后警惕脱离

### 路线B：抛石降费流（炮台）

- 沙涌地系能耗减半，抛石永久-5能耗，泥浆铠甲物攻物防+60%，岩脉崩毁4能120威力

### 路线C：锁场控场流（工兵）
- 石锁 + 精神扰乱 + 地刺 + 魔能爆
- 石锁锁场3回合，精神扰乱能耗+1，地刺应对打断，魔能爆清空能量警惕脱离
