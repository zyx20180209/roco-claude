## 四、特性「起飞加速」

> 本场战斗首次使用的技能获得迅捷。

本场战斗第一次使用的技能获得迅捷：

- **首技能选择**：可选择任意技能作为首次使用，获得迅捷
- **激怒联动**（3能，能耗+3×3回合）：首次激怒迅捷，立刻锁场
- **破防联动**（3能，双防-130%）：首次破防迅捷，立刻破盾
- **锐利眼神联动**（2能，双防-120%）：首次锐利眼神迅捷，立刻破盾
- **飞羽联动**（0能，迅捷+驱散增益）：飞羽本身自带迅捷，首次使用双重迅捷
- **穿膛联动**（2能65，敌方≤2能量时5倍=325威力）：首次穿膛迅捷，先手5倍爆发
- **核心价值**：首技能迅捷让任意技能都能先手，配合极速253形成绝对先手
- **局限性**：只有首次使用有效（每场战斗1次）；双攻63偏低；冰/电2x弱点

## 五、技能池亮点

**起飞加速首技能迅捷**：
- 激怒 3能 → 首次迅捷锁场（能耗+3×3回合）
- 破防 3能 → 首次迅捷破盾（双防-130%）
- 锐利眼神 2能 → 首次迅捷破盾（双防-120%）
- 穿膛 2能65 → 首次迅捷（敌方≤2能量时5倍=325威力）
- 风隐 1能 → 首次迅捷双方脱离（先手-1）

**自带迅捷技**：
- 飞羽 0能 → 自带迅捷+驱散敌方1种增益
- 翼击 3能50 → 自带迅捷
- 风墙 2能 → 减伤50%+迅捷+应对攻击

**控场**：
- 激怒 3能 → 能耗+3×3回合
- 精神扰乱 0能 → 全技能能耗+1（应对防御+3）
- 破防 3能 → 双防-130%（应对防御冷却2回合）
- 锐利眼神 2能 → 双防-120%
- 晒太阳 1能 → 驱散敌方所有增益
- 风隐 1能 → 双方脱离+先手-1

**防御续航**：
- 羽翼庇护 3能 → 减伤70%+应对连击+3
- 风墙 2能 → 减伤50%+迅捷
- 吓退 2能 → 减伤60%+应对敌方脱离
- 应激反应 2能 → 回血25%（应对防御50%）

**输出**：
- 水幕冲击 3能100 → 稳定物攻
- 气泡 3能100 → 稳定魔攻
- 扇风 3能75 → 先手+50%威力（极速253必先手）
- 闪击 4能60 → 速度差越大威力越高
