# 幽冥眼 PVP 对战分析（睁眼/闭眼形态）

| 方向 | 支持维度 | 主要支持点 |

**战术角色**：拦截 + 工兵（能量压制）+ 破盾（背袭0能20倍）

- **2x弱点**：光、幽、恶（3条）
- **半抗**：普通、毒、虫、武（4条）
- **免疫**：无

## 四、特性「惊吓」

> 能量等于0的精灵，无法对自己造成伤害

核心机制：对手能量耗尽时完全无法攻击幽冥眼：

- 配合勾魂（偷3能）+恶作剧（偷3能）+报复（应对偷3能）：快速将对手能量压至0
- 配合背袭（敌方能量=0时20倍伤害）：0能40×20=800威力一击必杀
- 配合穿膛（敌方能量≤2时5倍伤害）：325威力终结
- 配合降灵印记（换人入场-1能）：对手换人后能量更快归零
- **局限性**：对手可以聚能恢复能量；光/幽/恶系2x弱点；需要持续偷能才能维持压制

## 五、技能池亮点

**能量压制核心**：
- 勾魂 1能 → 偷3能
- 恶作剧 1能 → 敌方失去3能（应对防御6能）
- 报复 2能 → 减伤70%+应对触发偷3能
- 降灵 2能 → 降灵印记，换人入场-1能
- 操控 1能 → 敌方本回合技能能耗+7
- 聒噪 3能 → 敌方攻击技能耗+3持续3回合

**终结技**：
- 背袭 2能40 → 敌方能量=0时20倍=800威力
- 穿膛 2能65 → 敌方能量≤2时5倍=325威力
- 幽灵爆发 5能140 → 高威力魔攻

**增益**：
- 嘲弄 2能 → 魔攻+90%（敌方换人时额外速度+70）

**防御**：
- 报复 2能 → 减伤70%+偷3能
- 吓退 2能 → 减伤60%+强制换人

## 六、推荐配招方向

### 路线A：惊吓封锁流（拦截+破盾）
- 勾魂 + 恶作剧 + 背袭 + 降灵
- 勾魂+恶作剧快速压能至0，背袭800威力终结，降灵封锁换人

### 路线B：报复偷能流（拦截+工兵）
- 报复 + 勾魂 + 穿膛 + 降灵
- 报复应对触发偷3能，勾魂偷3能，穿膛低能5倍325威力

### 路线C：能耗封锁流（工兵）
- 操控 + 聒噪 + 幽灵爆发 + 嘲弄
- 操控+聒噪彻底封锁能耗，嘲弄魔攻+90%后幽灵爆发终结
