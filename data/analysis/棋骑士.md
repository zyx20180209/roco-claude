# 棋骑士 PVP 对战分析

## 一、形态说明

棋骑士存在白子/黑子两个形态，数值略有差异：
- **白子**（主形态）：HP90/攻126/特攻45/防88/特防69/速125（总543）
- **黑子**：HP83/攻121/特攻43/防84/特防66/速125（总522）
- 本文基于白子分析，技能池与机制黑子通用

## 二、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 4维 | 物攻126（85%）+极速267先手+腾挪应对变身+气沉丹田物攻+130%+迫近攻击永久+45威力 |
| 耐久 | 3维 | HP90+腾挪应对后回满状态（核心回复）+石肤术物防+160% |
| 功能性 | 3维 | 腾挪应对变身+气沉丹田回血60%+应对-3能耗+破绽双防-70%+截拳应对打断回能 |

**主定位**：应对变身型拦截（腾挪应对1次回满状态+变身棋骑后+极速267顶级先手）
**战术角色**：拦截 + 炮台

## 三、种族值与速度

- 白子：HP90 / 攻126 / 特攻45 / 防88 / 特防69 / 速125（总543）
- **有效种族498**（去除特攻45）
- **极速266.6**（95%分位，顶级高速）/ 满速230.5
- 启动速度：⭐⭐⭐⭐⭐ 即时（极速267顶级先手+腾挪应对即变身）

## 四、属性抗性（地/武双属性）

- **2x弱点**：草、水、冰、武、萌、机械、幻（7条极多）
- **半抗**：普通、火、地、电、毒、虫、恶（7条）
- **免疫**：无

## 五、特性「腾挪」

> 攻击技能应对1次后，回满状态，变为棋骑后。

应对1次攻击后回满状态+变身：

- **回满状态**：应对1次后生命/能量全部回满，相当于完全刷新
- **变身棋骑后**：变身后形态具备不同技能池/特性（渗透：队友武/地技能使用×+5%攻防入场）
- **应对+防御联动**：搭配防御/有效预防/石肤术等应对防御技触发
- **核心价值**：一次应对=完全重置，相当于一次"免死"+变身继续战斗
- **局限性**：只能触发1次（变身后失效）；需要被攻击；7条2x弱点极多；棋骑后面板偏辅助

## 六、技能池亮点

**应对触发腾挪**：
- 防御 0能 → 减伤70%+应对攻击（最便宜触发）
- 石肤术 2能 → 物防+160%+魔防-60%（物攻应对）
- 均势 2能 → 物防+140%+速度-30
- 地陷 4能80 → 应对状态翻倍+物防+70%

**高威力输出**：
- 光刃 7能120 → 稳定物攻
- 岩土暴击 7能140 → 被攻击永久-1能耗（腾挪联动积累）
- 迫近攻击 3能90 → 每次使用永久+45威力（叠加发育）
- 影袭 3能100 → 稳定武系
- 跺地 3能80 → 稳定地系
- 扫尾 3能90 → 稳定物攻

**增益**：
- 气沉丹田 10能 → 回血60%+物攻+130%+应对-3能耗（腾挪后使用威力更大）
- 力量增效 1能 → 物攻+100%
- 破绽 2能 → 敌方双防-70%+应对防御物攻+70%
- 化劲 2能 → 全技能威力+40
- 提气 4能 → 全技能威力+40（换人+50）

**应对控场**：
- 截拳 3能90 → 应对状态打断+回能
- 地刺 3能95 → 应对状态打断
- 触底强击 4能65 → 能量耗尽后威力+110=175

**先手**：
- 先发制人 2能55 → 先手+1
- 快速移动 2能 → 速度+80（应对防御+160）

## 七、推荐配招方向

### 路线A：腾挪应对回血流（拦截）
- 石肤术 + 岩土暴击 + 力量增效 + 光刃
- 石肤术物防+160%应对1次触发腾挪回满，岩土暴击被攻击永久-1能耗，光刃120威力爆发

### 路线B：破绽爆发流（炮台）
- 破绽 + 光刃 + 力量增效 + 先发制人
- 破绽双防-70%+应对防御物攻+70%，力量增效+100%，光刃120威力+先发制人先手

### 路线C：迫近叠加流（炮台）
- 迫近攻击 + 力量增效 + 气沉丹田 + 先发制人
- 迫近攻击每使用永久+45威力滚雪球，气沉丹田腾挪后回血+物攻+130%爆发
