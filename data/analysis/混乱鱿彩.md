## 四、特性「涂鸦」

> 使用非本系技能时威力+50%。

使用非幽系/非恶系技能时威力+50%：

- **能量刃联动**（武系，应对永久+90威力）：涂鸦+50%+应对叠加，能量刃80×1.5=120威力
- **坟场搏击联动**（幽系，敌方每1能量-10%威力）：坟场搏击是幽系，不触发涂鸦
- **双星联动**（幻系100威力）：涂鸦+50%=150威力
- **超级糖果联动**（萌系100威力+萌化+60）：涂鸦+50%=150+萌化+60=210威力
- **诋毁联动**（恶系80威力）：恶系不触发涂鸦
- **幻象联动**（幽系60威力）：幽系不触发涂鸦
- **核心价值**：非幽/恶系技能+50%威力，覆盖武/萌/幻/普通等多系别
- **局限性**：光系3x弱点致命；速度70偏慢；幽/恶系本系技能不触发

## 五、技能池亮点

**涂鸦联动（非幽/恶系技能）**：
- 双星 3能100 → 幻系+涂鸦=150威力
- 超级糖果 3能100 → 萌系+涂鸦=150威力+萌化+60=210威力
- 能量刃 3能80 → 武系+涂鸦=120威力+应对永久+90威力
- 气势一击 3能80 → 武系+涂鸦=120威力（上回合应对后+180=300威力）
- 坟场搏击 4能180 → 幽系（不触发涂鸦，但本身威力高）

**能量控场**：
- 恶作剧 1能 → 敌方-3能（应对防御-6能）
- 勾魂 1能 → 偷取敌方3能量
- 精神扰乱 0能 → 能耗+1（应对防御+3）
- 降灵 2能 → 降灵印记换人-1能

**星陨叠层**：
- 超新星馈赠 2能 → 2层星陨印记（每使用+1层，叠加后极多）
- 二律背反 4能 → 3层星陨（应对防御翻倍）
- 星链 3能 → 2连击每次1层星陨

**防御续航**：
- 虚化 2能 → 减伤80%+应对魔防+70%
- 彼岸之手 10能150 → 每失去10%生命-1能耗

**增益**：
- 力量增效 1能 → 物攻+100%
- 主场优势 3能 → 攻击印记全技能+10%威力
- 力量吞噬 4能 → 敌方威力-20+自己+20

**控场**：
- 欺诈契约 3能 → 交换增益和减益
- 暗箱操作 1能 → 自己双攻双防-100%（应对防御改为敌方-100%）
- 伪造账单 1能 → 反回血+先手+1

## 六、推荐配招方向

### 路线A：涂鸦爆发流（炮台核心）
- 能量刃 + 双星 + 力量增效 + 恶作剧
- 能量刃武系+涂鸦=120威力+应对永久+90，双星幻系+涂鸦=150威力，力量增效+100%，恶作剧-3能

### 路线B：超级糖果萌化流（炮台）
- 超级糖果 + 双星 + 力量增效 + 勾魂
- 超级糖果萌系+涂鸦=210威力，双星幻系+涂鸦=150威力，力量增效+100%，勾魂偷3能

### 路线C：星陨控场流（工兵）
- 超新星馈赠 + 二律背反 + 坟场搏击 + 恶作剧
- 超新星馈赠叠加星陨印记，二律背反应对翻倍，坟场搏击180威力，恶作剧-3能
