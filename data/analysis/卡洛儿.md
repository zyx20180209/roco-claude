# 卡洛儿 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 3维 | 特攻107+守护者队友弱化层数转入场-1能耗+水刃应对永久-4能耗+水波术回合结束永久+20威力 |
| 耐久 | 4维 | HP114+特防124（85%）+捧杀减伤90%+无畏之心减伤100%+泡沫幻影脱离 |
| 功能性 | 4维 | 守护者弱化转能耗+生日蛋糕驱减益增益翻倍+甜心续航萌化回40%生命+4能+水环应对全技能-2能耗 |

**主定位**：队友弱化转化型中转爆发（守护者弱化层数×-1能耗入场+特防124+萌化续航机制）
**战术角色**：中转 + 炮台

## 二、种族值与速度

- HP114 / 攻102 / 特攻107 / 防82 / 特防124 / 速90（总619）
- **有效种族517**（去除物攻102）
- **极速220.4**（30%分位，中等）/ 满速192.0
- 启动速度：⭐⭐⭐⭐⭐ 即时（守护者入场即结算，立刻低费释放大招）

## 三、属性抗性（水/萌双属性）

- **2x弱点**：草、电、毒、恶（4条）
- **半抗**：火、虫、武（3条）
- **免疫**：无

## 四、特性「守护者」

> 己方其他精灵每有1层弱化，自己入场时全技能能耗-1。

队友累积的弱化层数转化为入场时的降费：

- **应对防御队联动**：队友被精神扰乱/激怒等buff后入场，卡洛儿全技能-N能耗
- **被减益队友换入**：队友被减益换下，卡洛儿入场享受降费
- **大招低费释放**：水刃（5能）/爆米花爆破（7能）/音爆（6能）低费释放
- **与水刃联动**：水刃应对状态永久-4能耗+守护者入场-N，叠加几乎0能
- **与涌泉联动**：涌泉能耗每-1威力+10，守护者-N能耗=威力+N×10
- **核心价值**：中转位入场低费爆发，是"先吃减益后上卡洛儿"的逆向收割
- **局限性**：依赖队友累积弱化层数；速度90中等；4条2x弱点偏多

## 五、技能池亮点

**守护者联动高费技**：
- 爆米花爆破 7能140 → 守护者-N能耗后低费大威力
- 音爆 6能130 → 守护者低费爆发
- 水刃 5能115 → 应对状态永久-4能耗+守护者叠加
- 水波术 6能90 → 回合结束永久+20威力
- 涌泉 6能60 → 能耗每-1威力+10（守护者威力滚雪球）

**萌化续航**：
- 甜心续航 3能 → 双方萌化：回40%生命+4能
- 捧杀 2能 → 减伤90%+应对敌方萌化（触发甜心续航效果）
- 撒娇 2能25 → 萌化威力永久+20
- 月光合奏 2能30×N → 双方萌化层数+1连击

**增益**：
- 生日蛋糕 4能 → 驱散减益+增益翻倍（神技）
- 魔法增效 0能 → 魔攻+70%
- 潮汐 3能 → 减伤60%+应对湿润印记-1能耗
- 落雨 5能 → 雨天水系+50%威力

**防御续航**：
- 泡沫幻影 2能 → 减伤80%+脱离
- 无畏之心 4能 → 减伤100%+回血（能耗永久+2）
- 水环 3能 → 减伤60%+应对全技能-2能耗
- 捧杀 2能 → 减伤90%+应对萌化

**破罐破摔**（减益增伤）：
- 破罐破摔 3能80 → 自己有减益时威力+60=140
- 洗礼 2能 → 驱散自己减益+全技能-1能耗

## 六、推荐配招方向

## 八、擅长应对的情况

- 应对防御流队伍（守护者队友弱化积累核心联动）
- 火/地/机械系对手（水系克制）
- 武/恶/龙系对手（萌系克制）
- 消耗战节奏（甜心续航回40%+水波术永久+20）

## 九、棘手的对手特点

- 草/电系先手压制（2x弱点）

- 速度90极速220被高速先手压制
- 驱散减益的对手（队友弱化被清除）
