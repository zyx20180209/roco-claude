# 绒仙子 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 3维 | 缤纷星光非本系血脉威力+100%+放晴光系永久+50%+漫反射每系别技能+35+贮藏0能技双攻+50% |
| 耐久 | 3维 | 特防133（91%）+镜像反射减伤70%+应激反应回血+虫结阵减伤80%+应对奉献 |

**战术角色**：拦截 + 工兵

- **免疫**：无

## 四、特性「缤纷星光」

> 攻击时，若敌方血脉是非本系的系列血脉，技能威力+100%。

敌方血脉为非光/虫系时攻击威力+100%：

- **触发条件**：敌方携带非光系/虫系血脉时，绒仙子攻击威力翻倍
- **与放晴联动**：放晴光系永久+50%，缤纷星光+100%=光系技能威力+150%
- **与漫反射联动**：漫反射每系别技能+35，缤纷星光+100%后等效更高
- **与虹光冲击联动**：虹光冲击100威力+缤纷星光+100%=200威力等效
- **核心价值**：对携带非光/虫血脉的对手（绝大多数精灵）威力翻倍，覆盖面极广
- **局限性**：物攻/特攻73偏低，即使+100%也仅中等；物防81偏低；依赖血脉判断

## 五、核心战术：放晴+缤纷星光爆发

**光系爆发流**：
1. 放晴（0能）→ 光系永久+50%（应对防御+100%）
2. 漫反射（1能）→ 每系别技能+35
3. 虹光冲击（3能100）→ 缤纷星光+100%+放晴+50%=250威力等效

**控场消耗流**：
1. 精神扰乱（0能）→ 敌方全技能+1能耗（应对防御+3）
2. 摇篮曲（5能）→ 敌方全技能+3能耗+打断+眩晕
3. 捆缚（2能）→ 敌方+2层中毒+奉献

## 六、技能池亮点

**核心输出（缤纷星光加成）**：
- 虹光冲击 3能100 → 光系魔攻（缤纷星光+100%=200等效）
- 光刃 4能120 → 光系物攻（缤纷星光+100%=240等效）
- 光球 2能80 → 低费光系魔攻
- 脉冲光线 0能40 → 0能光系魔攻

**增益**：
- 放晴 0能 → 光系永久+50%（应对防御+100%）
- 漫反射 1能 → 每系别技能+35
- 贮藏 2能 → 双攻+50%（每携带0能技额外+50%）

**防御**：
- 镜像反射 1能 → 减伤70%+应对变为被应对技能
- 虫结阵 2能 → 减伤80%+应对队伍随机奉献

**控场**：
- 精神扰乱 0能 → 敌方全技能+1能耗（应对防御+3）
- 摇篮曲 5能 → 敌方全技能+3能耗+打断+眩晕
- 捆缚 2能 → 敌方+2层中毒+奉献
- 退化 2能 → 敌方+1层萌化
- 晒太阳 1能 → 驱散敌方增益

**资源**：
- 虫网 0能30 → 0能攻击+回1能
- 虫群智慧 3能 → 队伍2次随机奉献

## 九、擅长应对的情况

- 携带非光/虫血脉的对手（缤纷星光威力翻倍，覆盖绝大多数精灵）
- 幽/恶系对手（光系克制）
- 能量消耗战（精神扰乱+摇篮曲双重能耗控场）
- 增益发育型对手（晒太阳驱增益）
