# 白金独角兽 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 4维 | 特攻161（97%顶级）+目空非光系+25%+折射攻速双修+过曝永久叠加 |
| 耐久 | 1维 | HP99+特防92偏中；物防77偏脆 |
| 功能性 | 3维 | 折射多系别触发效果+漫反射每系+35威力+晒太阳驱散+倾泻驱散印记 |

**主定位**：顶级特攻主C（目空非光系+25%+折射攻速双修+特攻97%分位）
**战术角色**：收割 + 炮台

## 二、种族值与速度

- HP99 / 攻61 / 特攻161 / 防77 / 特防92 / 速105（总595）
- **有效种族534**（去除物攻61）
- **极速240.2**（63%分位，中高速）/ 满速208.5
- 启动速度：⭐⭐⭐⭐⭐ 即时（目空被动即时，魔法增效0能即可出手）

## 三、属性抗性（光系）

- **2x弱点**：草、幽（仅2条）
- **半抗**：恶、幻（2条）
- **免疫**：无

## 四、特性「目空」

> 携带的非光系技能，威力+25%

构筑时带非光系技能即触发：

- 许愿星（普通3能110）→ **137.5威力**
- 叶绿光束（草4能120）→ **150威力**
- 气泡（水3能100）→ **125威力**
- 超导（电3能95）→ **118.75威力**
- 回旋风暴（翼3能100）→ **125威力**
- **核心价值**：所有非光系技能+25%，配合多系别技能池覆盖面极广
- **局限性**：光系技能不触发（虹光冲击/天光/过曝不受益）；物防77偏脆

## 五、叠加型技能发育分析

**识别**：过曝（每使用1个其他系别技能，本技能威力永久+30）

- **过曝**：初始60威力，每使用其他系别技能+30。使用3次→150威力。
- **注意**：过曝是光系技能，目空不加成；但过曝叠加后威力极高。
- **折射联动**：折射携带不同系别技能触发不同效果（水-1能耗/电速度+50/草回血/火灼烧）。

## 六、技能池亮点

**目空联动非光系输出**：
- 许愿星 3能110 → **137.5威力**（目空+25%）
- 叶绿光束 4能120 → **150威力**（目空+25%）
- 气泡 3能100 → **125威力**（目空+25%）
- 超导 3能95 → **118.75威力**（目空+25%+迸发-1能耗）
- 回旋风暴 3能100 → **125威力**（目空+25%）

**折射攻速双修**：
- 折射 4能50 → 携带电系技能时速度+50（极速240→290+）
- 折射 4能50 → 携带水系技能时能耗-1
- 折射 4能50 → 携带草系技能时回血15%
- 折射 4能50 → 携带火系技能时敌方4层灼烧

**光系输出**：
- 虹光冲击 3能100 → 稳定光系
- 天光 3能95 → 系别随天气变化
- 过曝 3能60 → 每使用其他系别技能永久+30

**增益/功能**：
- 魔法增效 0能 → 魔攻+70%
- 漫反射 1能 → 每系别至多1个技能+35威力
- 晒太阳 1能 → 驱散敌方增益
- 倾泻 3能70 → 未被应对驱散双方印记
