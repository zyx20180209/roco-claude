# 伊兰亚龙 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 5维 | 特攻190（99%顶级）+嫉妒蓄力状态可用任意技能+升龙咆哮蓄力200威力+龙吟蓄力双攻+100%+架势下次无蓄力 |

**主定位**：蓄力爆发型炮台（嫉妒蓄力状态可使用任意携带技能+特攻190顶级+升龙咆哮200威力）
**战术角色**：炮台

## 二、种族值与速度

- HP112 / 攻91 / 特攻190 / 防69 / 特防99 / 速90（总651）
- **有效种族560**（去除物攻91）
- **极速220.4**（30%分位，中等）/ 满速192.0
- 启动速度：⭐⭐⭐⭐ 快速（架势/龙息环爆可跳过蓄力，嫉妒蓄力期间仍可行动）

## 三、属性抗性（龙系）

- **2x弱点**：冰、龙、萌（3条）
- **半抗**：草、火、水、电、翼（5条）
- **免疫**：无

## 四、特性「嫉妒」

> 蓄力状态下，可以使用任一携带技能。

蓄力期间不被锁定，可自由使用任意携带技能：

- **升龙咆哮联动**：蓄力期间使用龙吟（双攻+100%）→ 下回合升龙咆哮200威力+双攻+100%
- **龙吟联动**：蓄力期间使用魔法增效/氧输送叠加 → 下回合爆发
- **怨力打击联动**：蓄力期间使用其他技能同时等待被攻击触发（敌方技能威力×3）
- **架势联动**：架势后升龙咆哮直接200威力无蓄力
- **龙息环爆联动**：应对触发后升龙咆哮直接200威力
- **核心价值**：蓄力期间不浪费行动，双重行动效率极高
- **局限性**：速度90中等；物防69偏低；冰/龙/萌3条2x弱点

## 五、技能池亮点

**蓄力核心**：
- 升龙咆哮 3能200 → 蓄力，顶级魔攻
- 龙吟 3能 → 蓄力，双攻+100%+速度+60
- 怨力打击 3能1 → 蓄力，被攻击=敌方技能威力×3

**跳过蓄力**：
- 架势 2能 → 回血20%+下次技能无蓄力
- 龙息环爆 1能55 → 应对状态下次技能无蓄力

**高威力输出**：
- 升龙咆哮 3能200 → 顶级威力
- 湮灭 5能155 → 稳定高威力
- 叶绿光束 4能120 → 草系魔攻
- 大爆炸 3能100 → 稳定魔攻
- 超导 3能95 → 迸发-1能耗

**增益**：
- 魔法增效 0能 → 魔攻+70%
- 氧输送 2能 → 回4能+魔攻+70%
- 龙威 3能 → 龙噬印记5能技双攻+30%

**应对爆发**：
- 突袭 2能70 → 应对状态3倍=210威力
- 阻断 2能80 → 应对状态打断
- 强制重启 3能90 → 应对状态回合结束敌方返场

**控场**：
- 精神扰乱 0能 → 全技能能耗+1（应对防御+3）
- 星轨裂变 1能 → 敌方2层星陨

**续航**：
- 应激反应 2能 → 回血25%（应对防御50%）
- 移花接木 2能 → 回血15%+脱离

## 六、推荐配招方向

### 路线A：蓄力双行动爆发流（炮台核心）
- 升龙咆哮 + 龙吟 + 架势 + 魔法增效
- 架势下次无蓄力→升龙咆哮直接200威力；龙吟蓄力期间嫉妒使用魔法增效→下回合双攻+100%+魔攻+70%

### 路线B：怨力打击反击流（炮台）
- 怨力打击 + 升龙咆哮 + 龙息环爆 + 氧输送
- 怨力打击蓄力期间嫉妒使用氧输送，被攻击=敌方技能威力×3；龙息环爆应对后升龙咆哮无蓄力200威力

### 路线C：湮灭稳定流（炮台）
- 湮灭 + 魔法增效 + 氧输送 + 架势
- 架势无蓄力+魔法增效+氧输送叠加魔攻，湮灭155威力稳定输出
