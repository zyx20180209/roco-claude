# 獠牙猪 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 3维 | 物攻119（78%）+冻土冰系技能地系+10%/个+力量增效+100%+岩脉崩毁4能120 |
| 耐久 | 3维 | HP123（78%）+物防110（72%）+特防101+石肤术物防+160% |
| 功能性 | 3维 | 冰晶坠能耗+1+冰雹应对+3能+石锁禁换+瞬间零度+3能耗 |

**主定位**：冰地双系物攻炮台（冻土强化地系技能+冰地双系打击面+石锁控场）
**战术角色**：炮台 + 拦截 + 工兵

## 二、种族值与速度

- HP123 / 攻119 / 特攻41 / 防110 / 特防101 / 速85（总579）
- **有效种族538**（去除特攻41）
- **极速213.8**（30%分位，中低速）/ 满速186.5
- 启动速度：⭐⭐⭐⭐ 快速（冻土被动即时，冰系/地系技能即可出手）

## 三、属性抗性（冰/地双属性）

- **3x弱点**：武、机械（双3x，致命）
- **2x弱点**：草、地
- **半抗**：普通、光、电、毒、翼
- **免疫**：无

## 四、特性「冻土」

> 每携带1个冰系技能进入战斗，地系技能威力+10%

构筑时携带的冰系技能越多，地系技能威力越高：

- 携带1个冰系技能 → 地系+10%
- 携带2个冰系技能 → 地系+20%
- 携带3个冰系技能 → 地系+30%
- **最优构筑**：携带3个冰系+1个地系核心：岩脉崩毁4能120→+30%=156威力，地震10能190→+30%=247威力
- **局限性**：需要占用技能栏带冰系技能；武/机械系双3x致命

## 五、技能池亮点

**冻土联动地系输出**：
- 岩脉崩毁 4能120 → 固定4能耗+30%=156威力
- 地震 10能190 → +30%=247威力（需冰系铺垫能量）
- 地刺 3能95 → +30%=123.5威力+应对状态打断
- 石锁 3能50 → +30%=65威力+禁换3回合

**冰系联动**：
- 滚雪球 3能55 → 2层冻结，应对状态+2层冻结+威力翻倍（应对后110威力）
- 暴风雪 3能85 → +1层冻结
- 冰雹 4能105 → 应对状态敌方+3能耗
- 冰晶坠 4能90 → 敌方+1能耗
- 瞬间零度 0能 → 本回合敌方技能能耗+3

**控场**：
- 石锁 3能50 → 禁换3回合
- 雪球 1能 → 敌方速度-90
- 霜冻 1能 → 敌方魔防-100%
- 聒噪 3能 → 敌方攻击技能耗+3持续3回合

**增益/防御**：
- 力量增效 1能 → 物攻+100%
- 石肤术 3能 → 物防+160%+魔防-60%
- 均势 3能 → 物防+140%+速度-30

## 六、推荐配招方向

### 路线A：冻土冰地双系流（炮台）
- 岩脉崩毁 + 暴风雪 + 冰晶坠 + 力量增效
- 携带3个冰系（暴风雪+冰晶坠+力量增效位）触发冻土+30%，岩脉崩毁156威力+力量增效爆发

### 路线B：冰系能耗压制流（工兵+炮台）
- 冰雹 + 冰晶坠 + 瞬间零度 + 岩脉崩毁
- 冰雹应对+3能耗，冰晶坠+1能耗，瞬间零度+3能耗，岩脉崩毁+冻土爆发

### 路线C：石锁控场流（拦截+工兵）
- 石锁 + 滚雪球 + 地刺 + 石肤术
- 石锁禁换，滚雪球冻结爆发，地刺应对打断，石肤术防+160%

## 八、擅长应对的情况

- 草/地/龙/翼系对手（冰系克制）
- 电/火/毒/机械系对手（地系克制）
- 依赖中转节奏（石锁禁换3回合）
- 高能耗技能池（冰晶坠+冰雹+瞬间零度能耗压制）

## 九、棘手的对手特点

- 武/机械系克星（双3x弱点必死）
- 草/地系先手压制（2x弱点）
- 速度85偏慢，被高速精灵先手
- 纯魔攻对手（冻土只强化地系，纯魔攻玩法受限）

- 石锁禁换控场
