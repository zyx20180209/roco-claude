# 海豹船长 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 3维 | 双攻113/109均衡+身经百炼队伍应对后入场水武技能+20%/次+水波术回合结束永久+20威力 |
| 耐久 | 3维 | HP90+物防111（72%）+防御反击减伤80%+应对全技能威力+40+听桥应对反伤 |
| 功能性 | 3维 | 身经百炼队伍应对层数转化入场威力+雪替身应对回能+水环应对全技能-2能耗 |

**主定位**：应对队伍体系中转爆发型炮台（身经百炼队伍应对层数×+20%威力+极速234+水武双系打击面）
**战术角色**：炮台 + 中转

## 二、种族值与速度

- HP90 / 攻113 / 特攻109 / 防111 / 特防77 / 速100（总600）
- **有效种族491**（双攻均衡取113）
- **极速233.6**（54%分位，中速）/ 满速203.0
- 启动速度：⭐⭐⭐⭐ 快速（身经百炼入场即结算，队伍前面应对越多威力越高）

## 三、属性抗性（武/水双属性）

- **2x弱点**：草、电、翼、萌、幻（5条偏多）
- **半抗**：火、地、虫、恶、机械（5条）
- **免疫**：无

## 四、特性「身经百炼」

> 己方精灵每应对1次，自己入场时水系和武系技能威力+20%。

队伍应对层数累积到入场时的水武系威力加成：

- **应对队伍联动**：队伍前面的精灵（如针叶巡林/游蛇魔使等应对型）每应对1次，海豹船长入场时水武技能+20%威力
- **中转位价值**：应对流队伍中作为2-3号位出场时积累3-5层=+60%~+100%威力
- **水武双系覆盖**：水系和武系技能都享受加成，覆盖面广
- **与水波术联动**：水波术回合结束永久+20威力，叠加身经百炼+20%×N层形成滚雪球
- **核心价值**：应对队伍专用收割位，前排应对越多后排爆发越强
- **局限性**：依赖队伍应对前置积累；自己本身面板一般；5条2x弱点偏多

## 五、技能池亮点

**身经百炼联动高威力**：
- 一拳 5能140 → 武系高威力
- 水刃 5能115 → 水系应对状态永久-4能耗
- 水幕冲击 6能100 → 水系稳定物攻
- 水波术 6能90 → 水系回合结束永久+20威力
- 气泡 5能100 → 水系稳定魔攻

**应对防御**：
- 防御反击 2能 → 减伤80%+应对全技能威力+40（自身也应对积累）
- 听桥 4能 → 减伤60%+应对反伤
- 水环 3能 → 减伤60%+应对全技能-2能耗
- 雪替身 2能 → 减伤70%+应对回能（被应对技能能耗×2）
- 冰墙 3能 → 减伤80%+应对2层冻结
- 泡沫幻影 2能 → 减伤80%+脱离
- 硬门 2能 → 应对打断+90威力物伤

**增益**：
- 力量增效 1能 → 物攻+100%
- 魔法增效 0能 → 魔攻+70%
- 化劲 2能 → 全技能威力+40
- 蓄水 3能 → 下次技能能耗-6

**应对爆发**：
- 叠势 3能25×N → 每应对永久+2连击

## 六、推荐配招方向

### 路线A：身经百炼水波术滚雪球（炮台）
- 水波术 + 力量增效 + 防御反击 + 一拳
- 身经百炼入场水武+20%×N，力量增效+100%物攻，水波术回合结束永久+20威力滚雪球，一拳140终结

### 路线B：水武双系收割流（炮台）
- 水刃 + 一拳 + 魔法增效 + 水环
- 身经百炼入场水武技能+20%×N层，水刃应对永久-4能耗，一拳140威力，水环应对-2能耗

### 路线C：应对积累流（中转）
- 防御反击 + 听桥 + 叠势 + 水刃
- 防御反击+听桥应对积累身经百炼层数给后续队友+自身叠势连击，水刃应对永久-4能耗

## 八、擅长应对的情况

- 应对流队伍（身经百炼核心联动）
- 火/地/虫/恶/机械系对手（水武双系克制+半抗）
- 消耗战节奏（水波术永久+20威力滚雪球）
- 换人频繁的对手（应对积累层数）

- 速度100极速234被高速先手压制
- HP90+特防77偏脆，特攻先手威胁大
