## 四、特性「扩散侵蚀」

> 使用水系技能后，敌方获得中毒，获得层数等于中毒印记层数的2倍。

使用水系技能后，敌方获得中毒层数=当前中毒印记层数×2：

- **疫病吐息联动**（1层中毒印记）：先放疫病吐息1层印记，水系技能后=1×2=2层中毒
- **多层印记叠加**：若有3层中毒印记，水系技能后=3×2=6层中毒
- **以毒攻毒联动**：6层中毒=魔攻+180%
- **不可接触联动**：6层中毒=减伤+60%（基础50%+60%=实际减伤极高）
- **毒液渗透联动**：6层中毒=能耗-6=毒液渗透-1能120威力
- **鸩毒联动**：6层中毒=威力+60=75+60=135威力（应对状态+120=195威力）
- **核心价值**：水系技能随手触发大量中毒，配合中毒联动技形成爆发
- **局限性**：需要先有中毒印记才能放大；地/电/恶/幻4条弱点；特攻85偏低

## 五、技能池亮点

**扩散侵蚀触发源（水系技能）**：
- 激流 4能120 → 水系魔攻+触发扩散侵蚀
- 水炮 5能110 → 水系+每次使用永久-1能耗+触发扩散侵蚀
- 天洪 7能150 → 水系+应对永久-6能耗+触发扩散侵蚀
- 甩水 0能30 → 水系+回1能+触发扩散侵蚀（最低费触发）
- 飞吻 2能80 → 水系魔攻+触发扩散侵蚀

**中毒印记叠层**：
- 疫病吐息 3能 → 1层中毒印记（扩散侵蚀放大核心）
- 腐蚀酸液 2能35 → 2层中毒（非印记，直接中毒）

**中毒联动爆发**：
- 以毒攻毒 1能 → 中毒每层+30%魔攻（6层=+180%）
- 毒液渗透 5能120 → 中毒每层-1能耗（6层=0能120威力）
- 鸩毒 3能75 → 中毒每层+10威力（应对状态+20）（6层=135/195威力）
- 不可接触 1能 → 减伤50%+中毒每层+10%减伤（6层=减伤110%）

**防御续航**：
- 不可接触 1能 → 减伤50%+中毒每层+10%减伤
- 泡沫幻影 2能 → 减伤80%+脱离
- 水环 2能 → 减伤60%+应对全技能-2能耗
- 吓退 2能 → 减伤60%+应对敌方脱离
- 捧杀 4能 → 减伤90%+应对敌方萌化

**增益**：
- 润泽 7能 → 魔攻+190%
- 洗礼 1能 → 驱减益+全技能-1能耗
