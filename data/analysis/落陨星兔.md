# 落陨星兔 PVP 对战分析

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|

| 耐久 | 4维 | 特防141（95%）+陨落回合结束效果-1（反孢子/灼烧/中毒）+虚化减伤80%+应对魔防+70%+嗜痛减伤80% |

**主定位**：反持续效果型高速拦截炮台（陨落双方回合结束效果-1+特防141顶级+极速273顶级先手）
**战术角色**：拦截 + 炮台

## 二、种族值与速度

- HP93 / 攻61 / 特攻125 / 防87 / 特防141 / 速130（总637）
- **有效种族576**（去除物攻61）

## 三、属性抗性（幻/幽双属性）

- **3x弱点**：幽（极危险）
- **2x弱点**：光、恶（2条）
- **半抗**：普通、毒、幻（3条）
- **1/3抗**：武（1条）
- **免疫**：无

## 四、特性「陨落」

> 在场时，双方回合结束时触发的效果，触发次数-1。

在场时所有回合结束效果减少1次触发（最低0次=完全抵消）：

- **反孢子寄生**：孢子寄生每回合触发1次，陨落在场=触发0次（完全免疫）
- **反灼烧**：灼烧每回合触发1次，陨落在场=触发0次（完全免疫）
- **反中毒**：中毒每回合触发1次，陨落在场=触发0次（完全免疫）
- **反水波术成长**：水波术回合结束+20威力，陨落在场=不成长
- **与粉耳星兔对立**：粉耳星兔双向光速+1次，落陨星兔陨落-1次，两者完全克制
- **双方效果**：陨落影响双方，自己的回合结束效果也-1（需注意）
- **核心价值**：天然克制孢子/灼烧/中毒/水波术等持续效果体系
- **局限性**：幽系3x弱点致命；双方效果（自己也受影响）；HP93偏低

## 五、核心战术：反持续效果+高速先手

**反持续效果流**：
1. 陨落在场→孢子/灼烧/中毒完全抵消
2. 恶作剧（1能）→ 偷敌方3能（应对防御6能）
3. 嘲弄（2能）→ 魔攻+90%
4. 多维击打（4能）→ 星陨层数=连击数爆发

**能量清零流**：
1. 恶作剧（1能）→ 偷3能（应对防御6能）
2. 降灵（2能）→ 换人-1能印记
3. 操控（1能）→ 本回合技能+7能耗

## 六、技能池亮点

**核心控场**：
- 恶作剧 1能 → 偷敌方3能（应对防御6能）
- 降灵 2能 → 换人-1能印记
- 操控 1能 → 本回合技能+7能耗×3回合
- 精神扰乱 0能 → 全技能+1能耗

**星陨叠层**：
- 多维击打 4能15×N → 星陨层数=连击数
- 二律背反 4能 → 敌方+3层星陨（应对防御翻倍）
- 超维投射 4能 → 敌方+4层星陨
- 星轨裂变 1能 → 敌方+2层星陨
- 心灵洞悉 7能 → 印记层数转星陨

**高威力输出**：
- 大爆炸 3能100 → 幻系魔攻
- 坍缩 3能85 → 击败后魔攻+70%
- 灵光 3能25×3 → 敌方换人连击翻倍

**增益**：
- 嘲弄 2能 → 魔攻+90%（敌方换人+速度+70）
- 伺机而动 1能 → 下次攻击+70威力

**防御**：
- 虚化 2能 → 减伤80%+应对魔防+70%
- 嗜痛 2能 → 减伤80%+应对双攻+40%
- 有效预防 1能 → 减伤50%+应对先手+1
