## 四、特性「蒸汽膨胀」

> 己方精灵每使用1次火系技能，自己入场时获得全技能威力+10。

队友火系技能使用次数转化为入场时全技能威力加成：

- **火系队伍联动**：队友使用火焰冲锋/山火/引燃等火系技能，烈火守护入场时全技能+10/次
- **层数示例**：队友使用10次火系技能=全技能+100威力；20次=+200威力
- **与山火联动**（每使用其他火系技永久翻倍）：山火初始15威力，每次使用其他火系技翻倍，配合蒸汽膨胀+N威力
- **与热身联动**（下次攻击翻倍）：蒸汽膨胀+N威力后热身翻倍=极限爆发
- **与怒火联动**（双攻+120%）：蒸汽膨胀+N威力+怒火+120%双攻
- **核心价值**：火系主题队的中转收割位，入场即高威力爆发
- **局限性**：速度65极慢；依赖火系队伍积累；水/地2x弱点

## 五、技能池亮点

**蒸汽膨胀联动输出**：
- 火云车 5能140 → 顶级物攻（蒸汽膨胀+N威力后极限爆发）
- 山火 3能15 → 每使用其他火系技永久翻倍（叠加后极高威力）
- 闪燃 1能40 → 应对状态4倍=160威力（蒸汽膨胀+N威力后更高）
- 偷袭 3能85 → 应对状态3倍=255威力

**翻倍机制**：
- 热身 1能 → 下次攻击翻倍（应对防御4倍）
- 闪燃 1能40 → 应对状态4倍
- 山火 3能15 → 每使用其他火系技永久翻倍

**增益**：
- 怒火 1能 → 双攻+120%+双防-40%
- 力量增效 1能 → 物攻+100%
- 化劲 2能 → 全技能威力+40
- 热身运动 2能 → 连击+3

**控场**：
- 引燃 2能 → 10层灼烧（DOT）
- 锐利眼神 2能 → 敌方双防-120%（破盾）
- 伪造账单 1能 → 反回血+先手+1

**防御续航**：
- 有效预防 1能 → 减伤50%+应对先手+1
- 彼岸之手 10能150 → 每失去10%生命-1能耗（低血量低费）

**应对**：
- 撕裂 3能85 → 应对状态吸血100%
- 斩断 2能75 → 应对状态打断

## 六、推荐配招方向

### 路线A：蒸汽膨胀爆发流（中转）
