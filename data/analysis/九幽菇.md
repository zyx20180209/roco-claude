## 四、特性「多人宿舍」

> 自己的能量可以超过能量上限。

能量不受10点上限限制，可无限积累：

- **徒长联动**（回10能）：每次使用徒长回10能，多人宿舍允许超上限积累
- **蜡质膜联动**（应对回3能）：每次被攻击应对回3能，持续积累
- **魔能爆联动**（消耗所有能量，越多伤害越高）：积累20-30能后魔能爆爆发，威力极高
- **富养化联动**（场下精灵回3能）：支援队友同时自己也可积累
- **核心价值**：徒长+蜡质膜持续积累超上限能量，魔能爆一次性释放极限伤害
- **局限性**：速度70偏慢；冰系3x弱点致命；魔能爆需要多回合准备期

## 五、技能池亮点

**能量积累核心**：
- 徒长 2能 → 回10能（超上限积累）
- 蜡质膜 2能 → 减伤80%+应对回3能（防守积累）
- 魔能爆 1能 → 消耗所有能量，越多伤害越高（爆发终结技）

**高威力输出**：
- 仙人掌刺击 7能150 → 顶级物攻
- 抛石 8能140 → 回合结束永久-5能耗
- 棘突 5能100 → 稳定魔攻
- 陨石 5能100 → 稳定魔攻
- 垂死反击 4能80 → 血量越低威力越高（HP149基础高）

**控场**：
- 流沙 3能 → 锁场3回合+应对防御双防-60%
- 毒雾 2能 → 敌方所有增益转中毒
- 毒孢子 2能 → 5层中毒
- 芳香诱引 2能 → 连击+3+应对防御打断+眩晕

**防御续航**：
- 淤泥表皮 2能 → 减伤80%+应对敌方连击-3
- 应激反应 2能 → 回血25%（应对防御50%）
- 泥浆铠甲 3能 → 物攻物防+60%+应对防御增益翻倍

**队伍支援**：
- 富养化 3能 → 场下每个精灵回3能

## 六、推荐配招方向

### 路线A：能量超载爆发流（炮台）
- 徒长 + 蜡质膜 + 魔能爆 + 仙人掌刺击
- 徒长回10能+蜡质膜应对回3能积累超上限，魔能爆消耗所有能量极限爆发，仙人掌刺击150威力备用

### 路线B：毒控爆发流（工兵+炮台）
- 毒雾 + 毒孢子 + 流沙 + 魔能爆
- 毒雾将敌方增益转中毒，毒孢子5层中毒，流沙锁场，魔能爆积累后爆发

### 路线C：拦截续航流（拦截）
- 蜡质膜 + 淤泥表皮 + 应激反应 + 垂死反击
- 蜡质膜+淤泥表皮双重减伤80%，应激反应回血，垂死反击低血量高威力
