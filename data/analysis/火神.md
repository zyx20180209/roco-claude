# 火神 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 4维 | 攻139（92%）+极速273（97%）+助燃特性每次火系技能双攻+20%+山火/吹火永久增长 |
| 耐久 | 1维 | HP117（68%）但特防72（17%）偏脆 |
| 功能性 | 2维 | 晒太阳驱散+精神扰乱+锐利眼神破盾 |

**主定位**：攻击型主C（高速物攻+助燃滚雪球）
**战术角色**：收割 + 炮台 + 清场

## 二、种族值与速度

- HP 117 / 攻139 / 特攻61 / 防94 / 特防72 / 速130（总613）
- **有效种族552**（去除特攻61）
- **极速273.2**（97%分位，顶级高速）
- 启动速度：⭐⭐⭐⭐ 快速（怒火1能即可爆发）

## 三、属性抗性

- **2x弱点**：水、地（仅2条，极少）
- **半抗**：草、冰、虫、萌、机械（5条）

## 四、特性「助燃」

> 使用火系技能后，获得双攻+20%

每次使用火系技能自动叠加双攻+20%，配合山火（每用其他火系技能威力永久翻倍）形成双重滚雪球。

## 五、推荐配招

### 路线A：怒火爆发流（收割+炮台）
- 怒火 + 火焰切割 + 火云车 + 晒太阳

### 路线B：山火滚雪球流（炮台长线）
- 吹火 + 山火 + 火焰切割 + 力量增效

### 路线C：热身核爆流（爆发型）
- 热身 + 火云车 + 怒火 + 晒太阳
- 热身应对防御×4倍→火云车140×4=560等效威力

## 六、叠加型技能发育分析

### 吹火流（路线B核心）

- **发育曲线**：吹火初始威力50，每次使用永久+20。叠加3次→110，叠加5次→150，叠加8次→210。配合助燃特性每次火系技能双攻+20%，8次出手后双攻约+160%，等效威力倍增显著。
- **发育阶段风险**：发育期（前3-5次出手）火神虽有极速273先手优势，但特防72极脆，遭遇高特攻对手或水/地系克星时被迫下场。永久buff不消失，但出手机会浪费，发育节奏被打断。
- **队伍依赖度**：高。吹火流需要中转精灵为火神创造安全出手窗口——对手换上克星时，中转精灵接招并将克星清理或逼退，再送火神回场继续叠加。
- **发育路线 vs 即时路线**：
  - 怒火流（路线A）：即时启动，怒火1能即可爆发，无需发育，天花板受限于单次爆发
  - 吹火流（路线B）：需要5-8次安全出手，天花板更高（吹火+山火双重永久成长），后期几乎无解

**队伍配合推论**：吹火流火神需要1-2只中转精灵（有脱离/吓退技能）负责在克星出现时接招换场，为火神保留出手机会；同时需要水系/地系克星清理位先行压制。
