## 四、特性「煤渣草」

> 在场时，所有灼烧的衰减变为增长。

燃薪虫在场时，灼烧层数不再衰减而是每回合增长：

- **灼烧持续叠加**：正常灼烧每回合衰减一半，煤渣草在场时反转为增长（每回合×2）
- **与引燃联动**：引燃10层灼烧→煤渣草在场每回合翻倍=第2回合20层→第3回合40层
- **与充分燃烧联动**：充分燃烧使灼烧翻倍+触发1次伤害，煤渣草增长后再翻倍=极限叠层
- **与炙热波动联动**：炙热波动应对状态灼烧翻倍，煤渣草增长后再翻倍
- **与火焰护盾联动**：火焰护盾应对6层灼烧，煤渣草在场持续增长
- **核心价值**：灼烧从消耗型变为滚雪球型，长期战灼烧伤害极高
- **局限性**：火系精灵免疫灼烧；特防73偏低；需要在场才触发（换人失效）

## 五、核心战术：灼烧滚雪球

**灼烧叠层流**：
1. 引燃（2能）→ 敌方10层灼烧
2. 煤渣草在场→ 第2回合20层→第3回合40层（指数增长）
3. 充分燃烧（3能）→ 灼烧翻倍+触发1次伤害（极限爆发）

**山火成长流**：
1. 火焰冲锋（1能60）→ 使用1次火系技
2. 山火（3能15）→ 每使用1次其他火系技永久威力翻倍（第2次=30，第3次=60...）
3. 多次使用后山火威力极高

## 六、技能池亮点

**灼烧核心（煤渣草联动）**：
- 引燃 2能 → 敌方10层灼烧（低费核心）
- 充分燃烧 3能 → 灼烧翻倍+触发1次伤害
- 炙热波动 3能55 → 4层灼烧（应对状态翻倍）
- 火焰护盾 2能 → 减伤70%+应对6层灼烧

**高威力输出**：
- 灼伤 4能120 → 稳定火系物攻
- 顶端优势 3能100 → 稳定草系物攻
- 山火 3能15 → 每使用其他火系技永久威力翻倍（成长）
- 抽枝 4能90 → 应对状态回50%血+5能
- 见招拆招 2能65 → 上回合状态技后+55威力=120

**防御续航**：
- 酶浓度调整 3能 → 减伤90%+应对回20%血
- 蜡质膜 2能 → 减伤80%+应对回3能
- 淬火 4能 → 减伤80%+应对下次威力翻倍
- 无畏之心 5能 → 减伤100%+回血
- 嗜痛 2能 → 减伤80%+应对双攻+40%
- 血气 2能 → 减伤60%+应对致命保留1血

**资源**：
- 移花接木 2能 → 回15%血+脱离（主动换人）
- 高温回火 2能40 → 魔伤+自己脱离
- 藤绞 4能80 → 物伤+回5能

## 七、推荐配招方向

### 路线A：灼烧滚雪球流（炮台核心）
- 引燃 + 充分燃烧 + 酶浓度调整 + 灼伤
- 引燃10层灼烧→煤渣草增长，充分燃烧翻倍爆发，酶浓度调整减伤90%+回血续航，灼伤120威力

### 路线B：山火成长流（炮台）
- 火焰冲锋 + 山火 + 引燃 + 蜡质膜
- 火焰冲锋触发山火成长，山火威力指数增长，引燃灼烧滚雪球，蜡质膜减伤80%+回3能

### 路线C：应对爆发流（拦截）
- 淬火 + 炙热波动 + 抽枝 + 酶浓度调整
- 淬火应对下次威力翻倍，炙热波动应对灼烧翻倍，抽枝应对回50%血+5能，酶浓度调整续航
