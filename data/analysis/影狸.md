# 影狸 PVP 对战分析

**主定位**：换人惩罚型破盾（下黑手换人入场5层中毒+极速273顶级先手）
**战术角色**：破盾 + 工兵

- HP97 / 攻109 / 特攻105 / 防105 / 特防70 / 速130（总616）

- **3x弱点**：恶（极危险）
- **2x弱点**：光、地、幽、幻（4条）
- **半抗**：普通、草、萌（3条）
- **免疫**：无

## 四、特性「下黑手」

> 敌方精灵离场后，更换入场的精灵获得5层中毒

每次对手换人自动触发，无需出手：

- 配合降灵印记（换人入场-1能）：对手换人即-1能+5层中毒
- 配合毒液渗透（中毒层越多能耗越低）：5层中毒→毒液渗透0能120威力
- 配合坟场搏击（敌方每有1能威力-10%）：偷能后坟场搏击威力接近满值
- 配合勾魂（偷3能）：换人-1能+偷3能=单次换人损失4能
- **与梦悠悠做噩梦的对比**：做噩梦换人-3能；下黑手换人+5层中毒（DOT破盾）
- **局限性**：恶系3x弱点致命；特防70极脆；对不换人的对手效果有限

## 五、技能池亮点

**换人惩罚+中毒破盾**：
- 下黑手（被动）→ 换人入场5层中毒
- 降灵 2能 → 降灵印记换人-1能（叠加下黑手）
- 毒液渗透 5能120 → 中毒层越多能耗越低（5层后0能）
- 剧毒 2能 → 3层中毒（应对防御8层）
- 毒囊 2能25 → 2层中毒（应对状态6层）
- 疫病吐息 3能 → 中毒印记（场地层面，换人保留）

**能量压制**：
- 勾魂 1能 → 偷3能
- 惊吓盒子 3能80 → 应对状态偷6能
- 坟场搏击 4能180 → 敌方低能时威力接近满值

**输出**：
- 坟场搏击 4能180 → 低能时高威力
- 幽灵爆发 5能140 → 高威力魔攻
- 幻象 1能60 → 低费物攻

**中转**：
- 恶意逃离 1能 → 脱离+应对防御能耗+4

## 六、推荐配招方向

### 路线A：换人惩罚破盾流（破盾+工兵）
- 降灵 + 勾魂 + 毒液渗透 + 坟场搏击
- 降灵+下黑手换人-1能+5层中毒，勾魂偷3能，毒液渗透0能终结

### 路线B：中毒叠加流（破盾）
- 剧毒 + 毒液渗透 + 疫病吐息 + 坟场搏击
- 剧毒应对防御8层+下黑手5层，毒液渗透0能，疫病吐息中毒印记场地保留

### 路线C：恶意逃离中转流（中转+工兵）
- 恶意逃离 + 降灵 + 毒液渗透 + 勾魂
- 恶意逃离脱离+能耗+4，降灵封锁换人，毒液渗透终结
