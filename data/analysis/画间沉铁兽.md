# 画间沉铁兽 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 4维 | 物攻160（99%顶级）+变形活画敌方每1增益+10%威力+力量增效物攻+100%+提气全技能+40威力+爆冲应对5倍威力 |
| 耐久 | 3维 | HP100+物防100（54%）+应激反应应对回50%血+有效预防减伤50% |
| 功能性 | 3维 | 星轨裂变/超维投射叠星陨+空间压迫叠星陨+截拳应对打断回能+快速移动提速 |

**主定位**：反增益爆发型物理炮台（变形活画敌方每1层增益+10%威力+物攻160顶级+极速240先手）
**战术角色**：炮台 + 收割

## 二、种族值与速度

- HP100 / 攻160 / 特攻67 / 防100 / 特防76 / 速105（总608）
- **有效种族541**（去除特攻67）
- **极速240.2**（63%分位，中高速）/ 满速208.5
- 启动速度：⭐⭐⭐⭐ 快速（物攻160顶级+力量增效1能即可爆发）

## 三、属性抗性（普通/武双属性）

- **3x弱点**：无
- **2x弱点**：武、翼、萌、幻（4条）
- **半抗**：地、虫、幽、恶（4条）
- **1/3抗**：无
- **免疫**：无

## 四、特性「变形活画」

> 攻击时，敌方每有1层增益，本次技能威力+10%。

敌方增益层数直接转化为威力加成：

- **反增益核心**：敌方叠增益越多，画间沉铁兽攻击威力越高（敌方10层增益=威力+100%）
- **与力量增效联动**：力量增效物攻+100%+变形活画威力+N%，双重叠加
- **与提气联动**：提气全技能+40威力，再叠变形活画威力加成
- **与星陨联动**：星轨裂变/超维投射叠星陨（减益），但变形活画针对增益，两者方向不同
- **空间压迫联动**：空间压迫70威力+叠1层星陨，兼顾输出和叠层
- **核心价值**：天然克制增益发育型对手（鼓气/营养液泡/渗透等），对手增益越多越危险
- **局限性**：依赖敌方有增益才能最大化；对无增益对手威力加成为0；特防76偏低

## 五、核心战术：反增益爆发

**对增益型对手**：
1. 等待对手叠增益（鼓气/营养液泡/渗透等）
2. 力量增效（1能）→ 物攻+100%
3. 影袭/压扁（3-5能）→ 变形活画威力+N%爆发

**爆冲应对流**：
1. 有效预防（1能）→ 减伤50%+应对先手+1
2. 爆冲（3能）→ 应对状态威力×5=325威力爆发

## 六、技能池亮点

**核心输出（变形活画加成）**：
- 影袭 3能100 → 稳定武系物攻（变形活画加成后极高）
- 压扁 5能155 → 顶级物攻
- 爆冲 3能65 → 应对状态威力×5=325（极限爆发）
- 截拳 3能90 → 应对打断+回能（武系）
- 回旋踢 3能80 → 敌方换人威力翻倍

**增益**：
- 力量增效 1能 → 物攻+100%（低费核心）
- 提气 4能 → 全技能+40威力（换人+50）
- 快速移动 1能 → 速度+80（应对防御+160）

**星陨叠层（辅助）**：
- 星轨裂变 1能 → 敌方+2层星陨
- 超维投射 4能 → 敌方+4层星陨
- 空间压迫 3能70 → 物伤+叠1层星陨

**防御续航**：
- 应激反应 2能 → 回25%血（应对防御回50%血）
- 有效预防 1能 → 减伤50%+应对先手+1
- 能量刃 3能80 → 每应对成功永久+90威力（成长型）

**控场**：
- 连续爪击 2能30×2 → 应对状态连击翻倍

## 七、推荐配招方向

### 路线A：反增益爆发流（炮台核心）
- 力量增效 + 影袭 + 压扁 + 应激反应
- 力量增效1能物攻+100%，影袭/压扁+变形活画对增益型对手爆发，应激反应应对回50%血续航

### 路线B：爆冲应对流（收割）
- 有效预防 + 爆冲 + 力量增效 + 截拳
- 有效预防减伤50%+应对先手+1，爆冲应对状态325威力爆发，截拳应对打断回能

### 路线C：星陨联动流（炮台）
- 星轨裂变 + 空间压迫 + 力量增效 + 影袭
- 星轨裂变叠星陨，空间压迫70威力+叠层，力量增效+变形活画双重爆发
