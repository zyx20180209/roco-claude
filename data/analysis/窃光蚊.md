## 四、特性「血型吸引」

> 敌方每携带1种系列的技能，自己攻击时威力+10。

敌方技能池系别种数×10=威力加成：

- **多系别构筑克制**：敌方携带4种系别技能=攻击威力+40；携带5种=+50
- **与过曝联动**：过曝每使用1个其他系别技能永久+30威力，血型吸引+40=过曝威力极高
- **与漫反射联动**：漫反射每系别技能+35，血型吸引+40=双重加成
- **与光刃联动**：光刃120威力+血型吸引+40=160威力等效
- **核心价值**：天然克制多系别构筑（绝大多数精灵携带3-4种系别技能），覆盖面广
- **局限性**：物防63极低脆弱；5条2x弱点偏多；依赖敌方系别种数

## 五、核心战术：血型吸引+过曝叠加

**过曝叠加流**：
1. 使用不同系别技能（恶系/虫系/毒系）→ 过曝每系别永久+30威力
2. 过曝（3能60+N×30）→ 血型吸引+40=极高威力爆发

**跌落应对流**：
1. 暗箱操作（1能）→ 减伤+应对敌方双攻双防-100%
2. 跌落（3能120）→ 应对状态物攻+50%=180威力+血型吸引+40=220等效

## 六、技能池亮点

**核心输出（血型吸引加成）**：
- 光刃 4能120 → 光系物攻（+血型吸引+40=160等效）
- 跌落 3能120 → 应对状态物攻+50%（+血型吸引+40=220等效）
- 过曝 3能60 → 每系别永久+30威力（成长型）
- 灾厄 1能150 → 应对状态对敌方150威力（低费爆发）
- 牵连 4能85 → 敌方每力竭精灵+30威力
- 虫击 3能90 → 应对状态威力×2=180+无视抗性

**增益**：
- 漫反射 1能 → 每系别技能+35
- 贪婪 2能 → 100%吸血
- 力量吞噬 4能 → 偷敌方威力-20+自己+20

**防御**：
- 暗箱操作 1能 → 应对防御敌方双攻双防-100%（极品反制）
- 蝙蝠 2能65 → 物伤+100%吸血

**控场**：
- 恶意逃离 1能 → 脱离+应对防御敌方攻击技能+4能耗
- 隐藏条款 8能 → 与敌方交换技能（奇袭）
- 埋伏 4能30×3 → 敌方换人连击+3

**毒系辅助**：
- 腐蚀酸液 2能35 → 魔伤+2层中毒
- 连续毒针 2能15×2 → 每次+1层中毒

## 七、推荐配招方向

### 路线A：血型吸引爆发流（炮台核心）
- 过曝 + 光刃 + 漫反射 + 暗箱操作
- 过曝每系别永久+30威力，漫反射+35，光刃+血型吸引+40=160+威力，暗箱操作应对敌方双攻双防-100%

### 路线B：跌落应对流（收割）
- 暗箱操作 + 跌落 + 灾厄 + 贪婪
- 暗箱操作应对敌方双攻双防-100%，跌落应对物攻+50%=220等效，灾厄1能150威力，贪婪吸血续航

### 路线C：力量吞噬控场流（工兵）
- 力量吞噬 + 恶意逃离 + 过曝 + 虫击
- 力量吞噬偷威力，恶意逃离应对+4能耗，过曝成长，虫击应对180威力无视抗性
