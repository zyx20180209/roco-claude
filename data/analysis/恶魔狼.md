# 恶魔狼 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 4维 | 物攻114（73%）+悼亡每力竭1只双攻+30%+怒火双攻+120%+撕咬低血+2连击 |
| 耐久 | 2维 | HP115+物防91；特防63极脆 |
| 功能性 | 2维 | 恶意逃离中转+力量吞噬敌方威力-20+晒太阳驱散 |

**主定位**：后期爆发型收割（悼亡随力竭积累双攻+30%/只+极速260顶级先手）
**战术角色**：收割 + 炮台

## 二、种族值与速度

- HP115 / 攻114 / 特攻45 / 防91 / 特防63 / 速120（总548）
- **有效种族503**（去除特攻45）
- **极速260.0**（88%分位，顶级高速）/ 满速225.0
- 启动速度：⭐⭐⭐⭐⭐ 即时（悼亡被动积累，极速260先手）

## 三、属性抗性（恶系）

- **2x弱点**：光、虫、武、萌（4条）
- **半抗**：幽、恶（2条）
- **免疫**：无

## 四、特性「悼亡」

> 己方队伍中每有1只力竭的精灵，自己获得双攻+30%

队伍力竭越多，恶魔狼越强：

- 己方力竭1只 → 双攻+30%
- 己方力竭2只 → 双攻+60%
- 己方力竭3只 → 双攻+90%
- **后期爆发**：队伍残局时恶魔狼作为最后一两只精灵，此时悼亡加成最大
- 配合怒火（双攻+120%）+悼亡+90% = 双攻+210%（接近双攻+100%+110%叠加）
- **核心战术**：恶魔狼是"守尸者"，前面精灵力竭为它铺路
- **局限性**：只有队伍残局时才发挥最大价值；特防63极脆

## 五、叠加型技能发育分析

**识别**：趁火打劫（击败敌方后连击数永久+2）+ 彼岸之手（每失去10%生命能耗-1）

- **趁火打劫**：初始2连击，每次击败+2。击败3只后→8连击
- **彼岸之手**：初始10能，每失去10%生命-1。配合撕咬/暗突袭吸血维持生命
- **队伍依赖度**：低。极速260先手+悼亡被动积累，残局收割自给自足。

## 六、技能池亮点

**悼亡联动高爆发**：
- 怒火 1能 → 双攻+120%（配合悼亡后总加成极高）
- 撕咬 3能30×3 → 低血时5连击
- 暗突袭 4能70 → 吸血50%+应对状态翻倍
- 趁火打劫 3能35×2 → 击败敌方连击永久+2

**吸血续航**：
- 贪婪 2能 → 100%吸血
- 蝙蝠 2能65 → 吸血100%
- 撕裂 3能85 → 应对状态吸血100%
- 龙之利爪 3能130 → 蓄力+吸血50%

**终结技**：
- 坟场搏击 4能180 → 敌方低能时威力接近满值
- 彼岸之手 10能150 → 失血递减后低费
- 吞噬 6能150 → 击败后回6能

**功能**：
- 力量吞噬 4能 → 敌方威力-20+自己威力+20
- 晒太阳 1能 → 驱散敌方增益
- 恶意逃离 1能 → 中转+能耗+4
- 伪造账单 1能 → 反续航（敌方回血改为失去2倍）

## 七、推荐配招方向

### 路线A：悼亡残局爆发流（收割）
- 怒火 + 撕咬 + 贪婪 + 暗突袭
- 怒火+120%+悼亡+60-90%+贪婪吸血，撕咬低血5连击爆发

### 路线B：吸血续航流（收割）
- 贪婪 + 撕咬 + 暗突袭 + 蝙蝠
- 贪婪+蝙蝠+暗突袭三重吸血，撕咬低血爆发

### 路线C：终结流（收割+炮台）
- 坟场搏击 + 彼岸之手 + 贪婪 + 怒火
- 悼亡+怒火后坟场搏击/彼岸之手高威力终结
