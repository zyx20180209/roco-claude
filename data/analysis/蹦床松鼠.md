# 蹦床松鼠 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 2维 | 双攻中等（84/79）+气势一击应对后+180威力+蓄能轰击普通系叠加 |
| 耐久 | 4维 | HP114+物防111（73%）+囤积每能量双防+10%+无畏之心减伤100% |
| 功能性 | 2维 | 耀眼敌方连击-4+吓退强制脱离+锐利眼神双防-120%破盾 |

**主定位**：耐久型拦截（囤积能量叠加双防+应对反击流）
**战术角色**：拦截 + 工兵

## 二、种族值与速度

- HP114 / 攻84 / 特攻79 / 防111 / 特防87 / 速80（总555）
- **有效种族476**（去除特攻79）
- **极速207.2**（24%分位，偏慢）/ 满速181.0
- 启动速度：⭐⭐⭐ 中速（囤积需要能量积累才触发最大效果）

## 三、属性抗性（普通系）

- **2x弱点**：武（仅1条，极少）
- **半抗**：幽（仅1条）
- **免疫**：无

## 四、特性「囤积」

> 每有1能量，获得双防+10%

能量越多，双防加成越高：

- 初始10能 → 双防+100%（物防111→222，特防87→174）
- 满能15能 → 双防+150%（物防111→277，特防87→217）
- **核心价值**：不出手时囤积能量就能变得极耐打，配合聚能（+5能/次）快速叠加
- **局限性**：一旦出手消耗能量，囤积效果下降；速度80偏慢；特防87偏脆

## 五、叠加型技能发育分析

**识别**：蓄能轰击（每使用1次其他普通系技能，本技能能耗永久-2）

- **发育曲线**：蓄能轰击初始6能，使用3次普通系技能后→0能130威力。
- **囤积联动**：不出手仅聚能，囤积叠加+蓄能轰击发育同步进行。
- **发育阶段风险**：速度80偏慢，但囤积满能后双防极高，发育期自保能力强。
- **队伍依赖度**：低。囤积+防御技能自保，发育期基本自给自足。

## 六、技能池亮点

**输出核心**：
- 气势一击 3能80 → 应对成功后+180威力=260威力
- 蓄能轰击 6能130 → 每使用1次其他普通系技能本技能能耗永久-2
- 追打 3能75 → 应对状态3连击

**防御核心**：
- 无畏之心 5能 → 减伤100%+回血（能耗永久+2）
- 防反 2能 → 减伤60%+应对触发双攻+40%
- 吓退 2能 → 减伤60%+强制换人
- 硬化 2能 → 减伤90%（攻击后-2能耗）

**控场**：
- 耀眼 1能 → 敌方连击-4，反制连击型
- 锐利眼神 2能 → 敌方双防-120%，破盾神技
- 休息回复 2能 → 回血30%

## 七、推荐配招方向

### 路线A：囤积应对流（拦截）
- 防反 + 气势一击 + 吓退 + 休息回复
- 囤积叠双防+150%，防反应对触发双攻+40%，气势一击应对后260威力

### 路线B：蓄能轰击发育流（炮台）
- 落星 + 音波弹 + 蓄能轰击 + 耀眼
- 落星0能+音波弹1能触发蓄能轰击递减，使用3次后蓄能轰击0能130威力

### 路线C：破盾拦截流（拦截+破盾）
- 锐利眼神 + 气势一击 + 防反 + 吓退
- 锐利眼神双防-120%破盾，气势一击应对+180威力终结

## 九、擅长应对的情况

- 低连击物攻对手（囤积叠加双防后几乎打不动）
- 依赖应对触发的对手（气势一击应对反制）
- 消耗战节奏（囤积持续叠加）
- 依赖连击破盾的对手（耀眼连击-4反制）

## 十、棘手的对手特点

- 武系克星（唯一2x弱点）
- 高特攻魔攻核心（特防87偏脆，即使囤积叠加后也有限）
- 高速精灵先手压制（速度80偏慢）
- 强制消能效果（偷能精灵削弱囤积）

## 十一、队伍需求

### 精灵给的

- 气势一击应对后260威力终结
- 锐利眼神双防-120%破盾（团队破盾工具）
- 无畏之心减伤100%自保
- 耀眼反制连击

### 体系归属
- 囤积拦截位（普通系主题队）
- 应对流反击队（气势一击）
- 低能耗消耗战队
