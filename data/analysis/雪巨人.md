# 雪巨人 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 4维 | 物攻174（100%顶级最高）+打雪仗能量回复+力量增效+100%+压扁5能155威力 |
| 耐久 | 4维 | HP162（98%）+物防139（96%）+特防104+冰墙/冰天雪地/冰蛋壳多重减伤 |
| 功能性 | 3维 | 打雪仗冰系体系联动+冰晶坠/冰雹/霜天能耗压制+冰墙冻结+雪球速度-90 |

**主定位**：冰系体系绝对核心（物攻全游戏最高+打雪仗队友冰系技能积累+顶级HP/物防联防）
**战术角色**：炮台 + 拦截（冰系能耗压制）

## 二、种族值与速度

- HP162 / 攻174 / 特攻79 / 防139 / 特防104 / 速60（总718）
- **有效种族639**（去除特攻79，全游戏顶级）
- **极速180.8**（6%分位，极慢）/ 满速159.0
- 启动速度：⭐⭐ 慢速（初始0能，依赖队友铺垫冰系技能回能）

## 三、属性抗性（冰系）

- **2x弱点**：火、地、武、机械（4条）
- **半抗**：水、光、冰（3条）
- **免疫**：无

## 四、特性「打雪仗」

> 初始能量为0，入场前己方精灵每放1次冰系技能，回复3能量

核心机制：雪巨人入场时能量为0，但队友每次使用1次冰系技能就预先回复3能量：

- 队友使用3次冰系技能 → 入场即有9能，可直接释放压扁（5能）或冰晶坠（4能）
- 队友使用4次冰系技能 → 入场即有12能（上限10），满能入场
- **冰系队友**：电企鹅/冰钻布鲁斯/卡瓦重雪山/燃薪虫等都有冰系技能
- **与布克棱岩地脉/彩蝶鲨水翼推进的对比**：同类机制，雪巨人需要冰系队友
- **局限性**：完全依赖队友铺垫，若队友无冰系技能则入场0能；速度60极慢

## 五、叠加型技能发育分析

**识别**：力量增效（+100%物攻）+ 冰雹（应对状态额外+3能耗）

- **力量增效+物攻174**：使用后物攻+100%，压扁155×物攻174×2+ = 极高伤害
- **冰雹4能105**：应对状态时额外使敌方+3能耗，配合冰晶坠（+1能耗）双重压制
- **队伍依赖度**：极高（打雪仗需要冰系队友积累）。

## 六、技能池亮点

**顶级物攻输出**：
- 压扁 5能155 → 高费高威力（打雪仗后可早释放）
- 一拳 5能140 → 武系高威力
- 冰雹 4能105 → 应对状态+3能耗压制
- 冰晶坠 4能90 → +1能耗压制
- 暴风雪 3能85 → +1层冻结
- 丢冰块 3能90 → 敌方速度-30

**冻结+能耗压制**：
- 霜天 4能 → 1层冻结+每层冻结敌方全技能能耗+1
- 冰墙 2能 → 减伤80%+应对触发2层冻结
- 冰天雪地 2能 → 减伤80%+被应对技能能耗+3
- 雪球 1能 → 敌方速度-90

**防御核心**：
- 雪替身 2能 → 减伤70%+应对回能=2倍被应对能耗
- 冰墙 2能 → 减伤80%+冻结
- 冰蛋壳 3能 → 减伤60%+2层减速印记
- 无畏之心 5能 → 减伤100%+回血

**控场**：
- 降灵 2能 → 降灵印记（换人-1能）

## 七、推荐配招方向

### 路线A：打雪仗物攻爆发流（炮台）
- 力量增效 + 压扁 + 冰雹 + 冰墙
- 冰系队友积累后入场满能，力量增效+100%+物攻174，压扁爆发

### 路线B：冰冻能耗压制流（拦截+工兵）
- 霜天 + 冰雹 + 冰晶坠 + 冰墙
- 霜天冻结+能耗+1/层，冰雹应对+3能耗，冰墙应对2层冻结，冰晶坠+1能耗

### 路线C：冰系联防流（拦截）
- 雪替身 + 冰墙 + 冰天雪地 + 无畏之心
- 雪替身回能，冰墙冻结，冰天雪地+3能耗，无畏之心终极减伤
