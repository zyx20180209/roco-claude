## 四、特性「圣火骑士」

> 应对成功后，下次攻击威力翻倍

每次应对成功后下次攻击威力翻倍：

- 配合淬火（应对攻击+下次攻击威力翻倍）：应对1次=**威力×4**（圣火骑士×淬火双重翻倍）
- 配合持续高温（应对状态+下次攻击威力翻倍）：应对1次=**威力×4**
- 配合叠势（每应对永久+2连击）：应对叠连击
- **核心价值**：应对后下次攻击威力直接翻倍，配合淬火/持续高温可达4倍
- **局限性**：需要对手主动攻击或状态技触发；速度80偏慢

## 五、核心战术：三重翻倍联动

**翻倍机制叠加**：
- **圣火骑士**（应对后下次威力×2）
- **淬火**（应对攻击后下次威力×2）
- **持续高温**（应对状态后下次威力×2）

**极限爆发**：如果同一回合内触发圣火骑士+淬火或持续高温，下次攻击威力**×4**（两个翻倍独立结算）。

## 六、技能池亮点

**应对翻倍核心**：
- 淬火 4能 → 减伤80%+应对攻击下次威力翻倍
- 持续高温 2能70 → 应对状态下次威力翻倍
- 圣火骑士（被动）→ 应对后下次翻倍
- 叠势 3能25×N → 每应对永久+2连击

**高威力输出**：
- 爆裂飞弹 7能160 → 应对后翻倍=320威力
- 燃尽 4能155 → 应对后翻倍=310威力
- 炎打 2能95 → 应对后翻倍=190威力（双防-40%代价）
- 阳火增辉 3能75 → 每击败敌方威力永久翻倍

**火系灼烧**：
- 炙热波动 3能55 → 4层灼烧（应对状态威力+灼烧翻倍）
- 引燃 2能 → 10层灼烧
- 焚毁 2能60 → 驱散敌方印记

**武系联动**：
- 气波 0能40 → 低费魔攻
- 斩断 2能75 → 应对状态打断
- 贯手（血脉）2能80 → 武系输出

**增益**：
- 怒火 1能 → 双攻+120%+双防-40%
- 魔法增效 0能 → 魔攻+70%
- 化劲 2能 → 全技能威力+40

## 七、推荐配招方向

### 路线A：三重翻倍爆发流（炮台）
- 淬火 + 持续高温 + 爆裂飞弹 + 炎打
- 淬火+圣火骑士双翻倍=×4，爆裂飞弹160→640威力，炎打95→380威力

### 路线B：灼烧+应对流（破盾+拦截）
- 炙热波动 + 淬火 + 阳火增辉 + 魔法增效
- 炙热波动4层灼烧（应对状态翻倍），淬火应对翻倍，阳火增辉击败后永久翻倍

### 路线C：叠势连击流（炮台）
- 叠势 + 持续高温 + 炎打 + 怒火
- 叠势应对+2连击，持续高温应对翻倍，怒火双攻+120%，炎打爆发
