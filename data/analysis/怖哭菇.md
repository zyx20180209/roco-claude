# 怖哭菇 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 2维 | 多维击打星陨连击+四维降解印记降费+错乱应对状态3层星陨 |
| 耐久 | 3维 | HP124（79%）+特防100（59%）+嗜痛/血气/无畏之心多重防御 |
| 功能性 | 5维 | 吸积盘每回合+2层星陨+星链铺星陨+冥想应对+2层+错乱应对+3层+心灵洞悉印记翻倍 |

**主定位**：星陨体系发起者（吸积盘每回合被动+2层星陨+星陨铺设技能最全）
**战术角色**：工兵（星陨体系核心）

## 二、种族值与速度

- HP124 / 攻63 / 特攻69 / 防50 / 特防100 / 速60（总466）
- **有效种族403**（去除物攻63，全游戏5%分位）
- **极速180.8**（6%分位，极慢）/ 满速159.0
- 启动速度：⭐⭐⭐⭐⭐ 即时（吸积盘被动即时，偷师0能即可出手）

## 三、属性抗性（幻系）

- **2x弱点**：虫、幽（2条）
- **半抗**：武、幻（2条）
- **免疫**：无

## 四、特性「吸积盘」

> 回合结束时，敌方获得2层星陨

每回合结束自动给对手+2层星陨，无需出手：

- 持续在场即持续积累，3回合后敌方6层星陨
- 配合多维击打（每层星陨连击+1）：6层星陨→多维击打6连击
- 配合四维降解（每层印记能耗-1）：6层星陨→四维降解7能降至1能
- 配合心灵洞悉（星陨印记=敌方现有印记层数）：6层星陨→心灵洞悉再+6层=12层
- **社区共识**：与龙息帕尔天作之合（龙息帕尔有星陨联动技能）
- **局限性**：面板极低（有效种族403）；物防50极脆；速度60极慢

## 五、技能池亮点

**星陨铺设（最被动的星陨体系）**：
- 吸积盘（被动）→ 每回合+2层星陨
- 星链 3能 → 2连击，每连击+1层星陨（共+2层）
- 冥想 3能 → 减伤80%+应对触发+2层星陨
- 错乱 2能65 → 应对状态+3层星陨
- 心灵洞悉 7能 → 星陨印记=敌方现有印记层数（翻倍）

**星陨联动输出**：
- 多维击打 4能15×N → 每层星陨连击+1
- 四维降解 7能100 → 每层印记能耗-1（6层后1能）
- 针状物 3能30×3 → 3连击

**防御续航**：
- 嗜痛 2能 → 减伤80%+每次受伤双攻+40%
- 血气 2能 → 减伤60%+致命伤害保留1血
- 无畏之心 5能 → 减伤100%+回血
- 酶浓度调整 3能 → 减伤90%+回血20%

**功能**：
- 耀眼 1能 → 敌方连击-4（反制连击型）
- 当头棒喝 3能80 → 换人+100威力

## 六、推荐配招方向

### 路线A：吸积盘发育流（工兵，核心玩法）
- 冥想 + 多维击打 + 四维降解 + 嗜痛
- 吸积盘被动积累+冥想应对+2层，多维击打星陨连击，四维降解印记降费

### 路线B：心灵洞悉爆发流（工兵）
- 星链 + 心灵洞悉 + 多维击打 + 冥想
- 星链铺星陨，心灵洞悉翻倍，多维击打高连击爆发

### 路线C：联防续航流（拦截）
- 冥想 + 无畏之心 + 酶浓度调整 + 多维击打
- 冥想+无畏之心+酶浓度调整三重续航，多维击打星陨连击
