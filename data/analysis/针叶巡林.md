# 针叶巡林 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 4维 | 物攻139（92%）+野性感官应对后先手+1+能量刃应对永久+90威力+多重应对状态技（无影脚/截拳/散手） |

**主定位**：应对反击型收割（野性感官应对后先手+1+应对状态技池+物攻92%分位）
**战术角色**：收割 + 炮台

## 二、种族值与速度

- HP105 / 攻139 / 特攻57 / 防128 / 特防100 / 速105（总634）
- **有效种族577**（去除特攻57）
- **极速240.2**（63%分位，中高速）/ 满速208.5
- 启动速度：⭐⭐⭐⭐ 快速（野性感官被动触发，1能技能即可出手）

## 三、属性抗性（草/武双属性）

- **3x弱点**：翼（极危险）
- **2x弱点**：火、冰、毒、萌、幻
- **半抗**：水、光、电、恶
- **免疫**：无

## 四、特性「野性感官」

> 应对成功后，下次行动先手+1

每次应对成功后下次行动获得先手+1：

- 配合能量刃（每应对成功永久+90威力）：每次应对都变强+获得先手
- 配合应对状态技（无影脚2倍/截拳打断+回能/散手6连击/抽枝回血5能）：应对触发加成+下次先手
- 配合后发制人等武系应对流：形成应对循环
- **核心价值**：持续应对对手技能→自己变强+先手+1→爆发
- **局限性**：需要对手主动攻击才触发；翼系3x弱点致命

## 五、叠加型技能发育分析

**识别**：能量刃（每应对成功1次，本技能威力永久+90）

- **能量刃**：初始80威力，每次应对+90。应对3次→350威力，5次→530威力
- **野性感官联动**：每次应对后先手+1，能量刃爆发时先手释放
- **队伍依赖度**：低。应对型自给自足。

## 六、技能池亮点

**应对爆发核心**：
- 能量刃 3能80 → 应对永久+90威力（叠加后500+威力）
- 无影脚 3能85 → 应对状态2倍威力=170
- 截拳 3能90 → 应对状态打断+回能（自回该技能能耗）
- 散手 3能35 → 应对状态6连击（210等效威力）
- 抽枝 4能90 → 应对状态回50%血+5能
- 偷袭 3能85 → 应对状态3倍=255威力

**高输出**：
- 顶端优势 3能100 → 稳定草系
- 影袭 3能100 → 稳定武系
- 藤绞 4能80 → 回5能（自给）
- 提气 4能 → 全技能威力+40（换人时+90）

**增益**：
- 力量增效 1能 → 物攻+100%
- 盛开 1能 → 威力+30（应对防御+60）
- 均势 3能 → 物防+140%

**功能**：
- 移花接木 2能 → 回血15%+脱离
- 晒太阳 1能 → 驱散敌方增益
- 先发制人 2能 → 先手+1

## 七、推荐配招方向

### 路线A：应对爆发流（收割+炮台）
- 能量刃 + 无影脚 + 力量增效 + 先发制人
- 能量刃应对叠90威力，野性感官先手+1，无影脚应对状态2倍威力

### 路线B：连击应对流（收割）
- 散手 + 截拳 + 抽枝 + 力量增效
- 散手应对6连击，截拳应对打断+回能，抽枝应对回血+5能

### 路线C：稳定输出流（炮台）
- 顶端优势 + 影袭 + 力量增效 + 提气
- 提气+40威力（换人+90），顶端优势100威力，影袭100威力

## 九、擅长应对的情况

- 依赖攻击的对手（野性感官+能量刃应对爆发）
- 水/地/电/恶系对手（草武双系克制）
- 消耗战节奏（应对积累能量刃威力）
- 低耐久高攻对手（应对反击后先手击杀）

## 十、棘手的对手特点

- 翼系克星（3x弱点必死）
