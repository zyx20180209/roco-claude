# 声波缇塔 PVP 对战分析

**主定位**：向心力传动位置型拦截炮台（向心力1/2号位技能+30威力+传动1+物攻124+物防122+10条半抗）
**战术角色**：拦截 + 炮台

## 二、种族值与速度

- HP92 / 攻124 / 特攻48 / 防122 / 特防92 / 速90（总568）
- **有效种族520**（去除特攻48）
- **极速220.4**（30%分位，中等）/ 满速192.0
- 启动速度：⭐⭐⭐⭐ 快速（向心力被动即时，1/2号位技能直接+30威力）

## 三、属性抗性（机械系）

- **2x弱点**：火、水、武（3条）
- **半抗**：普通、草、冰、龙、毒、虫、翼、萌、机械、幻（10条极多）
- **免疫**：无

## 四、特性「向心力」

> 1号和2号位技能获得传动1和威力+30。

1号和2号位技能自动获得传动1（每回合下移1位）和威力+30：

- **1号位技能**：+30威力+传动1（下回合移到2号位，仍享受+30威力）
- **2号位技能**：+30威力+传动1（下回合移到3号位，失去加成）
- **齿轮切开联动**（1/3号位-2能耗）：1号位齿轮切开+30威力=160威力，传动后到2号位仍+30=160威力
- **飞踢联动**（3能110威力）：1/2号位飞踢=140威力
- **齿轮扭矩联动**（每回合位置变化永久+20威力）：传动让技能持续移动=持续叠加
- **核心价值**：1/2号位技能自动+30威力，配合传动让技能循环享受加成
- **局限性**：速度90中等；火/水/武3条弱点；需要管理技能位置

## 五、传动位置系统说明

声波缇塔的向心力特性与传动机制深度联动：
- 1号位技能：+30威力+传动1（每回合下移1位）
- 2号位技能：+30威力+传动1
- 3/4号位技能：无加成
- 杠杆置换（0能）：交换两侧技能位置，可手动调整位置

## 六、技能池亮点

**向心力联动高威力**：
- 齿轮切开 5能130 → 1/2号位+30=160威力，1/3号位-2能耗=3能（双重加成）
- 飞踢 3能110 → 1/2号位+30=140威力
- 地刺 3能95 → 1/2号位+30=125威力+应对状态打断
- 齿轮扭矩 3能80 → 1/2号位+30=110威力+每回合位置变化永久+20
- 偷袭 3能85 → 1/2号位+30=115威力（应对状态3倍=345威力）

**传动位置联动**：
- 啮合传递 1能 → 速度+80，1/3号位物攻+60%，传动1
- 轴承支撑 3能 → 主动-1能耗，被动两侧-1能耗，传动1
- 传感器 1能20×2 → 1/3号位连击+1，传动1
- 杠杆置换 0能 → 回2能+交换两侧技能位置（手动调整）

**防御续航**：
- 石肤术 3能 → 物防+160%+魔防-60%
- 有效预防 1能 → 减伤50%+应对先手+1

**控场**：
- 腐化 1能 → 中毒每层敌方双攻-30%
- 连续毒针 2能15×2 → 每连击1层中毒（配合腐化）
- 毒囊 2能25 → 2层中毒（应对状态6层）
- 摇篮曲 5能 → 能耗+3+应对防御打断+眩晕
- 震击 3能80 → 敌方连击-3

**增益**：
- 力量增效 1能 → 物攻+100%

## 七、推荐配招方向

### 路线A：向心力爆发流（炮台核心）
- 齿轮切开 + 飞踢 + 力量增效 + 杠杆置换
- 向心力1/2号位+30威力，齿轮切开1号位160威力+3能，飞踢2号位140威力，杠杆置换调整位置

### 路线B：传动降费流（炮台）

- 传动系统让齿轮切开循环在1/3号位-2能耗，啮合传递物攻+60%，齿轮扭矩每次位置变化永久+20

### 路线C：中毒控场流（工兵）
- 腐化 + 连续毒针 + 毒囊 + 摇篮曲
- 连续毒针+毒囊叠中毒，腐化中毒每层敌方双攻-30%，摇篮曲能耗+3控场
