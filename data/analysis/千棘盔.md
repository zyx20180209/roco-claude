## 四、特性「溶解扩散」

> 每携带1个毒素技能进入战斗，水系技能使敌方获得1层中毒

携带的毒系技能越多，每次水系技能触发的中毒层数越多：

- 携带1个毒系技能 → 水系技能+1层中毒
- 携带2个毒系技能 → 水系技能+2层中毒
- 携带3个毒系技能 → 水系技能+3层中毒
- **最优构筑**：携带3个毒系技能→每次水系技能+3层中毒
- 配合甩水（0能水系）：0能即触发+3层中毒
- 配合毒液渗透（中毒层越多能耗越低）：快速叠满后0能120威力
- **局限性**：需要占用技能栏带毒系技能；地/电系2x弱点

## 五、技能池亮点

**水毒联动核心**：
- 甩水 0能30 → 水系触发溶解扩散+N层中毒，0能即可叠加
- 腐蚀酸液 2能35 → 毒系+2层中毒（同时触发溶解扩散）
- 毒液渗透 5能120 → 中毒层越多能耗越低（叠5层后0能）
- 剧毒 2能 → 3层中毒（应对防御8层）
- 盐水浴 2能 → 全技能能耗-2（应对防御-3）

**功能**：
- 腐化 1能 → 每层中毒双攻-30%（5层后-150%）
- 疫病吐息 3能 → 中毒印记（场地层面，换人保留）
- 毒雾 7能 → 敌方增益转化中毒
- 泡沫幻影 2能 → 减伤80%+脱离
- 润泽 7能 → 魔攻+190%（配合毒液渗透爆发）

## 六、推荐配招方向

### 路线A：溶解扩散破盾流（破盾）
- 甩水 + 腐蚀酸液 + 毒液渗透 + 腐化
- 携带3个毒系技能→甩水0能+3层中毒→腐化每层-30%双攻→毒液渗透0能终结

### 路线B：剧毒爆发流（破盾）
- 剧毒 + 甩水 + 毒液渗透 + 盐水浴
- 剧毒应对防御8层+甩水叠加，盐水浴能耗-2，毒液渗透低费终结

### 路线C：毒雾反增益流（工兵+破盾）
- 毒雾 + 甩水 + 毒液渗透 + 腐化
- 毒雾转化增益为中毒，甩水继续叠加，腐化压制双攻
