# 菊花梨 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 2维 | 特攻116（77%）+撒娇萌化威力永久+20+破罐破摔自身减益时+60威力 |
| 耐久 | 2维 | HP106+特防115（78%）；物防77偏脆 |
| 功能性 | 5维 | 无忧无虑弱化层数无限+示弱速度永久+150+击鼓传花增益传递+反弹萌化转敌方+三大清除齐全 |

**主定位**：速度永久传递型工兵（示弱+150速度永久→击鼓传花传给队友，不可替代的速度传递核心）
**战术角色**：工兵 + 中转

## 二、种族值与速度

- HP106 / 攻49 / 特攻116 / 防77 / 特防115 / 速60（总523）
- **有效种族474**（去除物攻49）
- **裸极速180.8**（6%分位，极慢）/ 满速159.0
- **示弱+150速后极速330.8**（顶级速度）
- 启动速度：⭐⭐⭐⭐ 快速（示弱1能触发）

## 三、属性抗性（萌系）

- **2x弱点**：毒、恶、机械（3条）
- **半抗**：虫、武（2条）
- **免疫**：无

## 四、特性「无忧无虑」

> 可获得的弱化层数不受限制

正常情况下弱化层数有上限（一般是-100%），无忧无虑突破上限：

- **示弱萌化速度+150**：可多次叠加，叠2次→+300速度永久
- 配合反弹：将萌化转移给敌方，清除自身退化形态风险
- **核心价值**：与示弱/击鼓传花形成"无限速度叠加+传递"体系
- **局限性**：只针对弱化层数上限，对增益上限无效

## 五、核心战术：示弱+击鼓传花传递链

**菊花梨最强战术**：示弱叠速度 → 击鼓传花脱离 → 下个入场的队友继承+150速度永久

**战术流程（以伊兰亚龙为例）**：
```
回合1：菊花梨上场，示弱（1能）→ 自己速度永久+150
回合2：菊花梨撒娇（3能）→ 威力永久+20（可选，看局势）
回合3：菊花梨击鼓传花（3能）→ 脱离，伊兰亚龙入场继承 +150速（+20威力）
回合4+：伊兰亚龙极速370，特攻190，可直接升龙咆哮爆发
```

**核心价值**：菊花梨1能代替伊兰亚龙的3能龙吟+1回合蓄力

## 六、技能池亮点

**传递核心**：
- 示弱 1能 → 萌化速度永久+150（无忧无虑可叠加）
- 击鼓传花 3能 → 脱离+下个入场精灵继承自己增益
- 生日蛋糕 4能 → 驱除减益+增益翻倍（传递前先翻倍）

**自身输出**：
- 破罐破摔 3能80 → 自己有减益时+60威力=140
- 撒娇 3能30×3 → 萌化后威力永久+20
- 月光合奏 3能30×N → 双方每层萌化连击数+1

**功能**：
- 反弹 4能 → 萌化转给敌方（清除退化风险+控场）
- 晒太阳 1能 → 驱散敌方增益
- 倾泻 3能70 → 未被应对则驱散双方印记
- 魔法增效 0能 → 魔攻+70%

## 七、推荐配招方向

### 路线A：示弱传递流（工兵，核心玩法）
- 示弱 + 击鼓传花 + 生日蛋糕 + 反弹
- 示弱叠+150速度（或2次+300）→ 生日蛋糕翻倍增益 → 击鼓传花传给队友

### 路线B：永久强化自身爆发流（炮台）
- 示弱 + 撒娇 + 月光合奏 + 生日蛋糕
- 示弱速度永久+150补速，撒娇威力永久+20/次滚雪球，生日蛋糕增益翻倍

### 路线C：反铺场工具流（清场+工兵）
- 晒太阳 + 倾泻 + 撒娇 + 月光合奏
- 双清除+自强化输出，反印记/强化队

### 路线D：萌化转嫁流（破盾）
- 撒娇 + 反弹 + 破罐破摔 + 生日蛋糕
- 反弹把萌化转给敌方退化；破罐破摔受益减益+60威力

## 十、擅长应对的情况

- 慢速高攻队伍（作为速度引擎传递+150速）
- 中转体系（击鼓传花完美契合）
- 依赖速度先手的对局（传递后队友极速暴涨）
- 恶系核心（萌2x克制）
- 印记/强化队（三大清除反制）

## 十一、棘手的对手特点

- 毒/恶/机械系先手压制（3条2x弱点）
- 驱散敌方buff的对手（晒太阳等清除传递增益）
- 石锁禁换（反制击鼓传花脱离）
- 物防77偏脆，高物攻威胁大
- 启动慢（铺强化需2-3回合）

## 十二、队伍需求

### 精灵缺的
- 毒/恶/机械系克星清理
- 物理面保护（物防77偏脆）
- **传递接收者**（慢速高攻，无自攻速双修）

### 精灵给的
- **顶级速度传递**（示弱+150甚至+300速度永久传给队友）
- 生日蛋糕翻倍增益
- 反弹萌化转敌方
- 三大清除齐全（晒太阳+倾泻+生日蛋糕）
- 萌系2x克恶（克龙息帕尔/恶魔狼）

### 体系归属
- 传递体系绝对核心（不可替代的工兵）
- 慢速高攻队友的"速度引擎"
- 反恶系针对队
- 永久强化主题队
