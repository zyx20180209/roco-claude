- **2x弱点**：冰、电（2条）

- **免疫**：无

## 四、特性「乘风连击」

> 使用翼系技能后，获得连击数+1。

每次使用翼系技能后连击数+1：

- **疾风刺联动**（先手3连击）：使用后+1=4连击，配合热身运动+3=7连击
- **乘胜追击联动**（永久+1连击）：每次使用后永久+1，乘风连击再+1=双重叠加
- **扇风联动**（先手+50%威力）：极速260几乎必先手，+50%威力后乘风连击+1连击
- **核心价值**：翼系技能使用后连击+1，配合连击叠加技形成高连击爆发
- **局限性**：特防55极低，被特攻先手必死；连击依赖翼系技能触发

## 五、技能池亮点

**乘风连击核心**：
- 疾风刺 2能25×3 → 先手3连击+乘风连击后4连击
- 乘胜追击 2能40×N → 每次使用永久+1连击+乘风连击再+1
- 扇风 3能75 → 先手+50%威力（极速260必先手）
- 热身运动 2能 → 连击+3（叠加乘风连击）

**迅捷输出**：
- 龙卷风 5能70 → 自带迅捷+应对状态1.5倍=105
- 飞羽 2能 → 迅捷+驱散敌方1种增益
- 羽化加速 2能 → 威力+20+迅捷入场

**应对控场**：
- 羽刃 2能75 → 应对状态强制换人
- 见招拆招 3能65 → 上回合状态技后威力+55=120

**破盾/增益**：
- 锐利眼神 2能 → 敌方双防-120%
- 三连破 2能 → 物攻+30%+3连击
- 化劲 2能 → 全技能威力+40
- 提气 4能 → 全技能威力+40（换人+50）

## 六、推荐配招方向

### 路线A：连击叠加流（炮台+收割）
- 疾风刺 + 乘胜追击 + 热身运动 + 扇风
- 热身运动+3连击，乘胜追击永久+1，乘风连击再+1，疾风刺先手多连击爆发

### 路线B：高速爆发流（收割）
- 扇风 + 龙卷风 + 飞羽 + 锐利眼神
- 极速260先手+扇风+50%威力，龙卷风迅捷1.5倍，飞羽驱散增益，锐利眼神破盾

### 路线C：应对控场流（拦截）
- 羽刃 + 见招拆招 + 飞羽 + 化劲
- 羽刃应对强制换人，见招拆招状态技后120威力，飞羽迅捷驱散
