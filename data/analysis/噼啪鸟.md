## 四、特性「连续负荷」

> 自己技能的迸发效果延长1回合。

所有迸发效果持续时间+1回合：

- **电弧联动**（迸发+40威力）：通常迸发1回合，连续负荷延长=2回合+40威力
- **天旋地转联动**（迸发+30威力+先手+1）：延长=2回合先手+1+威力+30
- **引雷联动**（迸发+20威力）：延长=2回合+20威力
- **增程电池联动**（蓄电印记迸发+10威力）：延长=2回合+10威力
- **雷暴联动**（迸发获得所有生效过的迸发效果）：连续负荷让每种迸发持续更久，雷暴积累更多种类
- **核心价值**：迸发效果持续2回合，每回合都能享受迸发加成，输出效率翻倍
- **局限性**：HP78+物防85偏低；地/冰2x弱点；迸发需要积累

## 五、技能池亮点

**连续负荷迸发核心**：
- 电弧 3能80 → 迸发+40威力（连续负荷=2回合+40威力）
- 天旋地转 3能60 → 先手+1+迸发+30威力（连续负荷=2回合先手+1+威力+30）
- 引雷 3能35×2 → 迸发+20威力（连续负荷=2回合+20威力）
- 增程电池 2能 → 蓄电印记迸发+10威力（连续负荷=2回合+10威力）

**高速输出**：
- 交叉闪电 3能100 → 稳定物攻（极速293先手）
- 回旋风暴 3能100 → 稳定魔攻
- 扇风 3能75 → 先手+50%威力（极速293必先手）
- 闪击 4能60 → 速度差越大威力越高（极速293差值极大）
- 疾风刺 2能25×3 → 先手3连击

**迅捷技**：
- 翼击 3能50 → 自带迅捷
- 龙卷风 5能70 → 自带迅捷+应对1.5倍

**增益**：
- 力量增效 1能 → 物攻+100%
- 魔法增效 0能 → 魔攻+70%
- 暴风眼 2能 → 连击+100%（翻倍）

**控场**：
- 麻痹 2能 → 敌方先手-1+应对防御双攻-70%
- 精神扰乱 0能 → 能耗+1（应对防御+3）
