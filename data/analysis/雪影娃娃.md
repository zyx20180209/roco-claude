**战术角色**：工兵 + 拦截

- HP130 / 攻103 / 特攻98 / 防66 / 特防130 / 速90（总617）

- **3x弱点**：机械（极危险）

- **半抗**：水、光、冰、虫
- **免疫**：无

## 四、特性「捉迷藏」

> 使敌方获得冻结时，也会使其获得全技能能耗+1

每次给对手施加冻结时自动触发能耗+1：

- 霜降（4层冻结）→ 触发捉迷藏+1能耗
- 冰点（5层冻结，应对防御+5层=10层）→ 触发捉迷藏+1能耗
- 冰墙（应对触发2层冻结）→ 触发捉迷藏+1能耗
- 暴风雪（1层冻结）→ 触发捉迷藏+1能耗
- 冬至（暴风雪天气每回合2层冻结）→ 每回合触发捉迷藏+1能耗
- 霜天（1层冻结+每层冻结+1能耗）→ 双重能耗压制
- **核心价值**：冻结不仅是DOT和斩杀判定，还附带能耗+1，克特攻主C+能耗压制
- **局限性**：对冰系精灵免疫冻结；物防66极脆；机械系3x弱点致命

## 五、技能池亮点

**捉迷藏联动冻结+能耗**：
- 霜降 1能 → 4层冻结+捉迷藏+1能耗（低费高效）
- 冰点 2能 → 5层冻结（应对防御10层）+捉迷藏
- 冰墙 2能 → 减伤80%+应对2层冻结+捉迷藏
- 冬至 7能 → 暴风雪天气每回合+2层冻结（连续触发捉迷藏）
- 霜天 4能 → 1层冻结+每层冻结+1能耗（叠加捉迷藏）

**冻结联动输出**：
- 碎冰冰 3能40 → 敌方每层冻结+20威力（5层冻结→140威力，10层→240威力）
- 暴风雪 3能85 → 1层冻结+捉迷藏
- 冰晶坠 4能90 → 敌方+1能耗

**萌化联动**：
- 撒娇 3能30×3 → 萌化威力永久+20
- 超级糖果 3能100 → 萌化+60威力
- 破罐破摔 3能80 → 自己有减益时+60威力
- 甜心续航 3能 → 萌化+附加回血40%+4能
- 赤子之心 2能 → 萌化+全技能能耗永久-3

**功能**：
- 雾气环绕 1能 → 回能=敌方技能能耗的一半
- 击鼓传花 3能 → 脱离+下个入场继承增益
- 瞬间零度 0能 → 本回合敌方技能能耗+3
