# 上岸蛙 PVP 对战分析

| 方向 | 支持维度 | 主要支持点 |

**主定位**：星陨叠层型物理炮台（超新星馈赠成长叠层+物攻106+极速260先手+张弛有度时间条件增益）
**战术角色**：炮台 + 拦截

- **3x弱点**：无

- **免疫**：无

## 四、特性「张弛有度」

> 周末时自己获得双攻+40%，其他时间获得双防+40%。

根据现实时间（周末/非周末）获得不同加成：

- **周末（周六/周日）**：双攻+40%，物攻106×1.4=148.4等效，进入顶级攻击层
- **非周末（周一至周五）**：双防+40%，物防101×1.4=141.4等效，进入顶级防御层
- **战术灵活性**：周末偏攻势，非周末偏拦截，同一精灵两种定位
- **与超新星馈赠联动**：周末双攻+40%+超新星馈赠叠星陨，爆发更强
- **与石肤术联动**：非周末双防+40%+石肤术物防+160%，物防极限叠加
- **核心价值**：时间条件增益，周末攻势/非周末防御，构筑灵活
- **局限性**：依赖现实时间；特防69极低；虫/幽2x弱点

## 五、核心战术：超新星馈赠成长+时间增益

**周末爆发流**：
1. 张弛有度双攻+40%（周末）
2. 超新星馈赠（2能）→ 叠星陨（每次+1层成长）
3. 双星（3能100）→ 物攻+40%爆发
4. 偷袭（3能85）→ 应对状态×3=255威力

**非周末拦截流**：
1. 张弛有度双防+40%（非周末）
2. 石肤术（3能）→ 物防+160%（+双防+40%=极限物防）
3. 鸣沙陷阱（4能60）→ 物防差威力极高

## 六、技能池亮点

**星陨叠层**：
- 超新星馈赠 2能 → 敌方+2层星陨（每次+1层成长）
- 空间压迫 3能70 → 物伤+1层星陨
- 二律背反 4能 → 敌方+3层星陨（应对防御翻倍）
- 心灵洞悉 7能 → 印记层数转星陨

**高威力输出**：
- 双星 3能100 → 稳定幻系物攻
- 偷袭 3能85 → 应对状态×3=255威力
- 鸣沙陷阱 4能60 → 物防差威力（非周末物防极高时威力暴增）
- 龙卷风 5能70 → 迅捷+应对1.5倍

**低费资源**：
- 偷师 0能30 → 0能物攻+回1能（资源循环）
- 乘胜追击 2能40 → 每次使用永久+1连击（成长）

**增益**：
- 力量增效 1能 → 物攻+100%
- 乘风 2能 → 速度+120
- 漫反射 1能 → 每系别技能+35

**防御**：
- 石肤术 3能 → 物防+160%（非周末+双防+40%=极限）

## 七、推荐配招方向

### 路线A：周末爆发流（炮台）

- 张弛有度周末双攻+40%，超新星馈赠叠星陨，力量增效+100%，偷袭应对255威力

### 路线B：非周末拦截流（拦截）

- 张弛有度非周末双防+40%+石肤术物防+160%，鸣沙陷阱物防差威力极高，偷师0能回能

- 超新星馈赠成长叠层，二律背反应对翻倍，双星100威力爆发，偷师0能回能
