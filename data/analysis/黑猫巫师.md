# 黑猫巫师 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 4维 | 特攻163（98%顶级）+预警被击杀前速度+50+彗星0能240威力+热身运动叠连击 |
| 耐久 | 2维 | HP152（97%顶级）+特防119（81%）；物防67极脆 |
| 功能性 | 2维 | 嘲弄魔攻+90%+聒噪/操控能耗压制+晒太阳驱散 |

**主定位**：顶级特攻主C（双流派：换命流/叠连击流）
**战术角色**：收割 + 炮台

## 二、种族值与速度

- HP152 / 攻91 / 特攻163 / 防67 / 特防119 / 速70（总662）
- **有效种族571**（去除物攻91）
- **极速194.0**（15%分位，极慢）/ 满速170.0
- 启动速度：⭐⭐⭐⭐ 快速（预警被动触发，魔法增效0能即可出手）

## 三、属性抗性（普通系）

- **2x弱点**：武（仅1条）
- **半抗**：幽（1条）
- **免疫**：无

## 四、特性「预警」

> 若敌方技能能够击败自己，回合开始时自己获得速度+50

被致命威胁时自动触发速度+50：

- 极速194+50=244，进入中高速段
- **换命流联动**：彗星0能240威力使用后消耗全部生命，预警在被击杀前触发速度+50
- **叠连击流联动**：热身运动叠连击后，预警触发速度+50保证先手
- **局限性**：需要被致命威胁才触发；物防67极脆，被物攻先手即危险

## 五、社区共识：双流派

**社区定位**：大核心（双流派可切换）

### 流派A：换命流（魔力换伤害）
- **配招**：魔能爆 + 彗星 + 嘲弄 + 热身运动
- **逻辑**：魔能爆囤能量→彗星0能240威力终结，自身力竭消耗1点魔力，可换掉对面1-2只
- **特点**：即时爆发，不需要发育，但消耗魔力

### 流派B：叠连击流（能量换伤害）
- **配招**：热身运动 + 乱打/午夜噪音 + 嘲弄 + 魔法增效
- **逻辑**：热身运动叠连击次数（+3/次），乱打5连击×N次=8-14连击高总伤，靠能量循环维持
- **特点**：不消耗魔力，天花板更高，但需要能量管理

## 六、技能池亮点

**换命流核心**：
- 彗星 0能240 → 消耗全部生命换爆发（满血时240威力）
- 魔能爆 0能 → 消耗所有能量，能量越多伤害越高

**叠连击流核心**：
- 热身运动 2能 → 连击+3（叠加后8-14连击）
- 乱打 4能25×5 → 5连击+热身叠加后高总伤
- 午夜噪音 4能20×5 → 幽系5连击（克普通/幽）

**增益**：
- 嘲弄 2能 → 魔攻+90%（敌方换人时额外速度+70）
- 魔法增效 0能 → 魔攻+70%
- 嗜痛 2能 → 减伤80%+每次受伤双攻+40%

**控场**：
- 聒噪 3能 → 敌方攻击技能耗+3持续3回合
- 操控 1能 → 敌方本回合技能能耗+7
- 晒太阳 1能 → 驱散敌方增益

## 七、推荐配招方向

### 路线A：换命流（社区主流）
- 魔能爆 + 彗星 + 嘲弄 + 热身运动
- 魔能爆囤能量→彗星0能240威力，嘲弄+90%魔攻加成

### 路线B：叠连击流（社区主流）
- 热身运动 + 乱打 + 嘲弄 + 魔法增效
- 热身运动叠连击，嘲弄+90%魔攻，乱打高总伤

### 路线C：能耗压制流（工兵）
- 聒噪 + 操控 + 许愿星 + 嘲弄
- 聒噪+操控彻底封锁能耗，嘲弄+90%魔攻后许愿星终结
