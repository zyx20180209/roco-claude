# 魔力猫 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 2维 | 双攻109均衡；叶绿光束120/仙人掌刺击150/丰饶+140%双攻 |
| 耐久 | 3维 | HP108+特防151（98%分位顶级）；氧循环草系回10%血；酶浓度调整90%减伤 |
| 功能性 | 3维 | 毒系技能池（孢子/剧毒/腐化）；光合印记；移花接木脱离；精神扰乱/摇篮曲 |

**主定位**：耐久型+功能型混合（特防顶级联防+毒系工兵）

**战术角色**：拦截 + 破盾 + 工兵 + 中转（移花接木）

## 二、种族值与速度

- HP 108 / 攻109 / 特攻109 / 防81 / 特防151 / 速55（总613）
- **有效种族504**（去除特攻109）
- **极速174.2**（4%分位，极慢）
- 特防151（98%分位）是绝对长板；物防81（28%分位）是明显短板

## 三、属性抗性

- **2x弱点**：火、冰、毒、虫、翼（5条偏多）
- **半抗**：水、光、地、电

## 四、特性「氧循环」

> 使用草系技能后，回复10%生命

每次使用草系技能自动回10%血，配合酶浓度调整（90%减伤+应对攻击回20%血）= 双重续航。

## 五、推荐配招

### 路线A：草系续航联防（拦截+炮台）
- 叶绿光束 + 酶浓度调整 + 丰饶 + 移花接木

### 路线B：毒系破盾（破盾+工兵）
- 毒孢子 + 腐化 + 叶绿光束 + 酶浓度调整

### 路线C：光合印记续航（工兵+拦截）
- 光合印记 + 叶绿光束 + 酶浓度调整 + 徒长

## 七、擅长应对的情况

- 高特攻魔攻核心（特防151能扛）
- 消耗战节奏（双重续航）
- 赖场型拦截（毒系破盾）

## 八、棘手的对手特点

- 高速物攻爆发（物防81极脆）
- 毒系精灵（草对毒无克制且自己怕毒）
- 速度55极慢被先手压制

- 草系打击面（克水/地）
- 毒系破盾（克权杖-V等拦截位）
