## 四、特性「侵蚀」

> 敌方每有1层中毒效果，自己获得连击数+1。

敌方中毒层数直接转化为连击数加成：

- **5层中毒联动**（毒孢子）：侵蚀+5连击，配合基础技能=多连击爆发
- **8层中毒联动**（剧毒应对防御）：侵蚀+8连击
- **腐化联动**（中毒每层双攻-30%）：8层中毒=腐化双攻-240%+侵蚀+8连击
- **毒雾联动**（增益转中毒）：对手增益越多，转化中毒越多，侵蚀连击越多
- **连续毒针联动**（每连击1层中毒）：连续毒针叠中毒，侵蚀再叠连击
- **核心价值**：中毒层数=连击数，配合腐化双攻-30%/层形成双重压制
- **局限性**：物攻80偏低；HP84+特防74偏低；光/地/幻3条弱点

## 五、技能池亮点

**侵蚀中毒叠连击核心**：
- 毒囊 2能25 → 2层中毒（应对状态6层）→ 侵蚀+2/+6连击
- 连续毒针 2能15×2 → 每连击1层中毒→ 侵蚀叠加
- 毒雾 7能 → 增益转中毒→ 侵蚀大量连击
- 感染病 4能85 → 击败后中毒转印记

**腐化联动（中毒双攻-30%/层）**：
- 腐化 1能 → 中毒每层敌方双攻-30%（8层=-240%）

**高威力输出（侵蚀连击后）**：
- 跌落 3能120 → 应对状态物攻+50%（否则-50%）
- 偷袭 3能85 → 应对状态3倍=255威力
- 撕咬 3能30×3 → 低血量+2连击（侵蚀叠加后多连击）
- 趁火打劫 3能35×2 → 击败后永久+2连击
- 见招拆招 2能65 → 上回合状态技后+55=120威力

**增益**：
- 力量增效 1能 → 物攻+100%
- 三连破 1能 → 物攻+30%+3连击
- 伺机而动 1能 → 下次攻击+70威力

**控场**：
- 毒雾 7能 → 增益转中毒（反强化神技）
- 欺诈契约 3能 → 交换增益和减益
- 羽刃 2能75 → 应对状态强制换人

## 六、推荐配招方向

### 路线A：中毒叠连击爆发流（收割核心）
- 毒囊 + 腐化 + 力量增效 + 跌落
- 毒囊6层中毒→侵蚀+6连击，腐化双攻-180%，力量增效+100%，跌落应对物攻+50%

### 路线B：毒雾反强化流（工兵）
- 毒雾 + 腐化 + 连续毒针 + 感染病
- 毒雾增益转大量中毒→侵蚀大量连击，腐化双攻-30%/层，感染病击败后转印记

### 路线C：趁火打劫叠加流（收割）
- 趁火打劫 + 毒囊 + 腐化 + 力量增效
- 毒囊叠中毒→侵蚀连击，趁火打劫击败后永久+2连击，腐化双攻-30%/层
