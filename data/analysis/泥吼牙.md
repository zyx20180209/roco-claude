## 四、特性「无差别过滤」

> 在场时，所有精灵连击数固定为2。

泥吼牙在场时，双方所有精灵的连击数强制固定为2：

- **克制高连击对手**：对手携带散手（6连击）/叠势（永久+2连击）/热身运动（+3连击）等高连击技，泥吼牙在场时全部变为2连击
- **克制连击叠加型精灵**：乘胜追击（永久+1连击）/能量刃（应对永久+90威力）等叠加型技能被强制限制
- **连续爪击联动**（应对状态连击翻倍）：固定2连击×翻倍=4连击
- **疾风刺联动**（先手3连击）：无差别过滤将3连击变为2连击（先手优势保留）
- **双方均受影响**：泥吼牙自己的连击技也被限制为2连击
- **核心价值**：针对连击叠加型精灵的克星，强制限制高连击爆发
- **局限性**：冰系3x弱点致命；双方均受影响；速度95中速

## 五、技能池亮点

**无差别过滤联动**：
- 连续爪击 2能30×2 → 应对状态连击翻倍=4连击（固定2×翻倍）
- 疾风刺 2能25×2 → 先手（无差别过滤后2连击）

**高威力输出**：
- 岩土暴击 8能140 → 被攻击永久-1能耗
- 地刺 3能95 → 应对状态打断
- 扇风 3能75 → 先手+50%威力（极速227先手）
- 鸣沙陷阱 4能60 → 物防越高威力越高（物防109+石肤术+160%）
- 地陷 5能80 → 应对状态翻倍+物防+70%

**增益**：
- 力量增效 1能 → 物攻+100%
- 泥浆铠甲 2能 → 物攻物防+60%+应对防御增益翻倍
- 石肤术 3能 → 物防+160%+魔防-60%
- 均势 3能 → 物防+140%+速度-30

**控场**：
- 羽刃 2能75 → 应对状态强制换人
- 摇篮曲 5能 → 能耗+3+应对防御打断+眩晕
- 晒太阳 1能 → 驱散敌方增益
- 欺诈契约 3能 → 交换增益和减益

**防御续航**：

- 暗突袭 4能70 → 吸血50%+应对状态翻倍

## 六、推荐配招方向

### 路线A：无差别过滤克连击流（拦截核心）
