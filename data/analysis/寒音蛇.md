## 四、特性「拨浪鼓」

> 己方精灵每使用1次状态技能，自己入场时毒系和萌系技能威力+10。

队友状态技使用次数转化为入场时毒萌技能威力加成：

- **队伍状态技密度**：队友使用力量增效/魔法增效/防御/棘刺等状态技，寒音蛇入场时毒萌技能+10/次
- **层数示例**：队友使用10次状态技=毒萌技能+100威力；20次=+200威力
- **与毒液渗透联动**：毒液渗透120威力+拨浪鼓+N威力+中毒层数降费=极限爆发
- **与以毒攻毒联动**：中毒层数×+30%魔攻+拨浪鼓+N威力
- **与生日蛋糕联动**：拨浪鼓增益翻倍
- **核心价值**：状态技密集队伍的中转收割位，入场即高威力爆发
- **局限性**：依赖队友状态技积累；恶系3x弱点致命；双攻79偏低

## 五、技能池亮点

**拨浪鼓联动高威力**：
- 毒液渗透 5能120 → 中毒每层-1能耗+拨浪鼓+N威力（双重叠加）
- 瘴气喷射 3能100 → 稳定毒系魔攻+拨浪鼓加成
- 鸩毒 3能75 → 中毒每层+10威力（应对状态+20）+拨浪鼓加成
- 超级糖果 3能100 → 萌化威力+60+拨浪鼓加成

**中毒叠层**：
- 毒孢子 3能 → 5层中毒
- 剧毒 2能 → 3层中毒（应对防御8层）
- 连续毒针 2能15×2 → 每连击1层中毒
- 毒囊 2能25 → 2层中毒（应对状态6层）
- 以毒攻毒 1能 → 中毒每层+30%魔攻

**萌化续航**：
- 赤子之心 2能 → 萌化全技能-3能耗
- 撒娇 3能30×3 → 萌化威力永久+20
- 月光合奏 3能30×N → 双方萌化层数+1连击
- 示弱 1能 → 萌化速度+150

**增益**：
- 生日蛋糕 4能 → 驱减益+增益翻倍
- 魔法增效 0能 → 魔攻+70%
- 力量增效 1能 → 物攻+100%

**控场**：
- 伪造账单 1能 → 敌方回血改为失去2倍+先手+1
- 冰晶坠 4能90 → 敌方全技能能耗+1
- 晒太阳 1能 → 驱散敌方增益

## 六、推荐配招方向

- 武/恶/龙系对手（萌系克制）
