# 翠顶夫人 PVP 对战分析

## 一、四维协同定位

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | 2维 | 双攻95/86面板中等；水炮/水刃永久降费滚雪球；水刃匹配圣羽翼王飓风迅捷 |
| 耐久 | 3维 | HP125+物防138（96%顶级）+等价交换减伤90%+应对50%吸血 |
| 功能性 | 4维 | 洁癖下场传递增益（不占技能栏）+飞羽0能迅捷驱散+风隐强制脱离+翼系迅捷工具池 |

**主定位**：联防+传递复合型（翼系迅捷体系关键搭档+物防顶级联防+洁癖不占栏位传递）
**战术角色**：拦截 + 工兵 + 中转

## 二、种族值与速度

- HP125 / 攻95 / 特攻86 / 防138 / 特防97 / 速115（总656）
- **有效种族570**（去除特攻86）
- **极速253.4**（79%分位，中高速）/ 满速219.5
- 启动速度：⭐⭐⭐⭐ 快速（洁癖被动，水系技能1能即可出手）

## 三、属性抗性（水/翼双属性）

- **3x弱点**：电（致命，社区主要克星点）⚠️
- **2x弱点**：冰
- **半抗**：火、虫、武、机械
- **免疫**：无

## 四、特性「洁癖」

> 离场后，自己的增益和减益会被更换入场的精灵继承

**关键性质**：
- 下场自动传递（**不占技能栏**，比菊花梨击鼓传花省栏位）
- **传递的增益随接收方下场消失**（不是永久强化）
- 用法是"翠顶铺强化 → 下场传给主C → 主C此局使用"，主C下场增益也没了
- **减益也传递**（需谨慎管理自身状态）

**与菊花梨的本质区别**：菊花梨是"永久强化跨精灵跨下场保留"，翠顶是"临时增益仅接收方本场有效"

## 五、圣羽翼王协同机制（社区共识核心）

**圣羽翼王飓风特性**：对本精灵的技能，若其他翼系精灵携带相同技能，则获得迅捷

**关键协同**：
- 翠顶夫人是**水+翼**双系，属于"其他翼系精灵"
- 翠顶和圣羽翼王**同时携带水刃** → 圣羽翼王的水刃获得迅捷（入场即攻击）
- 水刃（4能115威力，应对状态能耗永久-4）配合迅捷机制**极度强大**
- **翠顶夫人在翼系迅捷体系中是关键匹配技能提供者**

**可共享迅捷的候选技能**：水刃（水系翠顶自带）、力量增效（普通系，待验证）、风墙/翼击/龙卷风（翼系）

## 六、技能池亮点

**传递增益（洁癖载体）**：
- 力量增效 1能 → 物攻+100%（传给队友）
- 乘风 2能 → 速度+120（传给队友）
- 水刃永久-4能耗状态 → 可传给圣羽翼王

**翼系迅捷工具**：
- 飞羽 0能 → 迅捷+驱散敌方1种增益
- 风墙 2能 → 减伤50%+迅捷+应对攻击
- 龙卷风 5能70 → 迅捷+应对状态1.5倍
- 翼击 3能50 → 迅捷

**输出**：
- 水刃 4能115 → 应对状态能耗永久-4（叠加后0能115威力）
- 水炮 5能110 → 每次使用能耗永久-1（降费叠加）
- 气泡 3能100 → 稳定魔攻

**防御**：
- 等价交换 3能 → 减伤90%+应对攻击50%吸血
- 水泡盾 2能 → 减伤80%+应对魔攻+70%
- 泡沫幻影 2能 → 减伤80%+脱离

**控场**：
- 风隐 → 双方脱离+先手-1
- 聒噪 3能 → 敌方攻击技能耗+3持续3回合
- 飞羽 0能 → 驱散敌方1种增益

## 七、推荐配招方向

### 路线A：翼系迅捷体系伙伴（社区主流，搭配圣羽翼王）
- 水刃 + 等价交换 + 力量增效 + 风墙
- 水刃匹配圣羽翼王形成飓风迅捷链；等价交换应对攻击吸血续航；力量增效下场传递

### 路线B：纯联防传递流（拦截+工兵）
- 力量增效 + 乘风 + 等价交换 + 水炮
- 力量增效+乘风给队友临时buff，等价交换90%减伤吸血，水炮主力输出

### 路线C：迅捷反铺场流（清场+工兵）
- 飞羽 + 风墙 + 龙卷风 + 水炮
- 飞羽0能迅捷驱散，风墙迅捷减伤，龙卷风迅捷爆发

### 路线D：换人节奏压制流（中转+工兵）
- 风隐 + 聒噪 + 等价交换 + 气泡
- 风隐双方脱离，聒噪+3能耗封锁，气泡100威力终结

## 十二、与菊花梨的对比

| 对比 | 菊花梨 | 翠顶夫人 |
|------|--------|---------|
| **增益性质** | **永久**（跨精灵跨下场保留） | **临时**（接收方下场消失） |
| **传递方式** | 主动击鼓传花（占1栏） | 被动下场继承（不占栏） |
| **自身耐久** | HP106+特防115（脆） | HP125+物防138（顶级物盾） |
| **速度** | 60极慢（需示弱） | 115中上 |
| **输出面板** | 特攻116（77%） | 双攻95/86（平庸） |
| **独立作战能力** | 弱（纯辅助） | 强（可联防+输出） |
| **战略价值** | 永久速度+150给队友 | 临时buff+自己是物盾 |

**选择指南**：
- 队伍锁定高攻低速主C → **菊花梨**（永久强化价值更高）
- 队伍需要联防+辅助一体 → **翠顶夫人**（自带物盾+不占栏传递）
- 队伍是翼系体系 → **翠顶夫人**（飓风匹配水刃）
