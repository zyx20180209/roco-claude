# 武者鸡 PVP 对战分析

| 方向 | 支持维度 | 主要支持点 |
|------|---------|---------|

**战术角色**：拦截 + 炮台

- **免疫**：无

## 四、特性「斗技」

> 应对成功后，获得全技能威力永久+20。

每次应对成功后全技能威力永久+20：

- **层数示例**：应对5次=全技能+100威力；应对10次=全技能+200威力
- **与截拳联动**（应对状态打断+回能）：截拳应对触发斗技+20威力+回能继续出手
- **与散手联动**（应对状态6连击）：散手应对触发斗技+20威力+6连击
- **与爆冲联动**（应对状态5倍）：爆冲应对触发斗技+20威力+5倍=325+N威力
- **与怒火联动**（双攻+120%）：斗技叠加+怒火+120%=极限爆发
- **核心价值**：每次应对全技能都变强，消耗战中威力无限叠加
- **局限性**：需要被攻击才触发；水/翼/幻3条弱点；特防70偏低

## 五、斗技 vs 指挥家对比（武者鸡 vs 绅士鸡）

| 特性 | 叠加内容 | 叠加速度 | 适用场景 |
|------|---------|---------|---------|
| 斗技（武者鸡）| 全技能威力+20/次 | 每次应对+20 | 多技能均衡爆发 |
| 指挥家（绅士鸡）| 双攻+20%/次 | 每次应对+20% | 单技能高倍率爆发 |

斗技是绝对值叠加（+20威力），指挥家是百分比叠加（+20%双攻）。高威力技能下指挥家更强，低威力多连击技能下斗技更强。

## 六、技能池亮点

**斗技应对叠加核心**：
- 截拳 3能90 → 应对状态打断+回能（触发斗技+20威力）
- 散手 3能35×6 → 应对状态6连击（触发斗技+20威力）
- 爆冲 3能65 → 应对状态5倍=325+N威力（触发斗技）
- 闪燃 1能40 → 应对状态4倍=160+N威力（触发斗技）
- 防御 1能 → 减伤70%+应对攻击（触发斗技）

**高威力输出（斗技叠加后）**：
- 火焰切割 3能100 → 稳定物攻（斗技叠加后100+N威力）
- 回旋踢 3能80 → 换人时翻倍=160+N威力
- 截拳 3能90 → 应对打断+回能

**增益**：
- 怒火 1能 → 双攻+120%+双防-40%
- 力量增效 1能 → 物攻+100%
- 热身 1能 → 下次攻击翻倍（应对防御4倍）
- 暴风眼 2能 → 连击+100%（翻倍）
- 提气 4能 → 全技能威力+40（换人+50）

**脱离/控场**：
- 高温回火 2能40 → 魔伤+脱离
- 羽刃 2能75 → 应对状态强制换人
- 焚烧烙印 3能 → 驱散双方印记转灼烧
- 晒太阳 1能 → 驱散敌方增益

## 七、推荐配招方向

### 路线A：斗技叠加爆发流（拦截核心）
- 截拳 + 散手 + 怒火 + 力量增效
- 截拳应对打断回能触发斗技，散手应对6连击触发斗技，怒火+120%+力量增效+100%

### 路线B：应对翻倍流（炮台）
- 热身 + 火焰切割 + 截拳 + 怒火
- 热身翻倍，截拳应对触发斗技+回能，怒火+120%，火焰切割100+N威力

### 路线C：换人惩罚流（拦截）
- 回旋踢 + 羽刃 + 截拳 + 力量增效
- 回旋踢换人时160+N威力，羽刃应对强制换人，截拳应对打断回能
