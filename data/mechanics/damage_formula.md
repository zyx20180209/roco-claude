# 伤害与能力值计算公式

来源：《洛克计算器 2.0》(bilibili 爱心尾巴)，本文档为公式与机制的完整整理。

---

## 一、能力值计算

### 1. 生命值公式

```
生命值 = ROUND(1.7 × (种族值 + 3 × 个体值) + 70) × 性格修正 + 100
```

### 2. 其他五项公式（物攻 / 魔攻 / 物防 / 魔防 / 速度）

```
能力值 = ROUND(1.1 × (种族值 + 3 × 个体值) + 10) × 性格修正 + 50
```

### 3. 性格修正

| 性格对该项 | 修正系数 |
|-----------|---------|
| 加 | 1.2 |
| 无 | 1.0 |
| 减 | 0.9 |

**示例**（种族 100 / 个体 10 / 加速性格 1.2）：
- 速度 = ROUND(1.1 × (100 + 30) + 10) × 1.2 + 50 = 153 × 1.2 + 50 = 234

## 二、伤害计算

### 1. 实际威力

```
实际威力 = 原始威力 × 本系加成 × 属性克制 × 增减益 × 天气影响 × 减伤系数 × 其他加成
```

| 项 | 取值 |
|---|------|
| 本系加成 | 使用与自身属性相同的技能 = 1.25；否则 = 1 |
| 属性克制 | 见 `data/raw/type_chart.json`：2x / 1x / 0.5x（双属性叠乘） |
| 增减益 | 场上攻击 buff / debuff 带来的乘数（能力等级） |
| 天气影响 | 目前仅"雨天"对水系技能有增强 |
| 减伤系数 | 受击方防御技能、特性（如秩序鱿墨减伤）等 |
| 其他加成 | 特性、道具、印记等其他来源 |

### 2. 最终伤害

```
最终伤害 = INT(ROUND(攻击 × 连击数 × 实际威力 × 减伤 × 37/41) / 防御)
```

- **攻击**：技能为物理时取物攻，魔法时取魔攻
- **防御**：对应物防 / 魔防
- **37/41 ≈ 0.9024**：系统系数（来自洛克王国世界版）
- **连击数**：单次技能为 1；多段技能乘对应段数
- **取整**：内层 ROUND，整除时 INT 截断

> 旧公式 `CEIL(0.9 × 连击 × 威力 × 攻击 / 防御)` 与新公式相比，伤害结果差异在 ±2 点之内（约 0.27%）。

### 3. 属性差阶梯威力（特殊技能）

部分技能威力取决于**攻击方与防御方某项能力的最终差值**（含种族/个体/性格/buff 等所有修正后的战斗中数值）。

**典型技能：** 闪击（速度差）、鸣沙陷阱（防御差）

**阶梯表：**

| 差值区间 | 威力 |
|---------|------|
| < 0 | 60 |
| 0 ~ 14 | 100 |
| 15 ~ 29 | 130 |
| 30 ~ 44 | 140 |
| 45 ~ 59 | 150 |
| 60 ~ 74 | 160 |
| 75 ~ 89 | 170 |
| 90 ~ 104 | 180 |
| 105 ~ 119 | 190 |
| 120 ~ 134 | 200 |
| ≥ 135 | 200 |

规则：差值 < 0 时固定 60；0 ~ 14 为 100；之后每 15 点差值递增 10 威力，上限 200。

### 3. 百分比伤害（击杀判定）

```
百分比 = 100 × 最终伤害 / 对方生命值
击杀次数 = ROUNDUP(对方生命值 / 单次伤害)
```
