# 决策收益框架

本文档描述如何量化每回合决策的收益，用于辅助攻击/防御/状态/换人的选择。

---

## 一、核心原则

每个决策的净收益 = **造成的资源优势** - **承受的资源损失**

资源包括：血量、能量、场上状态（buff/debuff/印记）、精灵数量。

---

## 二、速度线是决策的前提

在计算任何决策收益前，必须先判断本回合出手顺序：

1. **应对关系优先**：部分技能之间存在应对关系，应对成功方先手并获得额外收益
2. **无应对时比速度**：速度高者先手

**应对关系规则：**

- **所有防御技能**可以应对攻击技能（应对成功减伤，部分有额外效果）
- **部分攻击技能**可以应对状态技能（打断或伤害增加，具体看技能描述）
- **部分状态技能**可以应对防御技能（状态效果提升，具体看技能描述）
- 能否应对**以技能具体描述为准**，不是所有攻击都能应对状态

出手顺序决定了"我能否在承伤前完成斩杀"，这是攻击决策的核心变量。

---

## 三、攻击决策

### 收益计算

```
攻击收益 = 造成伤害% × 斩杀概率权重
```

| 情形 | 收益评级 |
|------|---------|
| 本回合斩杀对方 | 最高（+1精灵优势） |
| 造成 >50% 伤害且对方无法反手斩杀 | 高 |
| 造成伤害但对方可反手斩杀 | 需权衡（见下） |
| 造成伤害但对方本回合必死于DoT | 等同斩杀 |

### 攻击 vs 被反手斩杀的权衡

若我先手且能斩杀对方 → 直接攻击，收益最大化。

若我后手且对方能先手斩杀我：
- 考虑换人（规避斩杀）
- 考虑防御应对（应对成功则我先手）
- 若场下无更好换人选择，接受交换（用我方精灵换对方精灵）

### 技能选择：威力 vs 能耗 vs 回合数

同一只精灵的攻击技能往往有"高威力高能耗"和"低威力低能耗"两种选项，需要根据局势选择。

**示例：**
- **隼鳞**：5能 / 140威力 → 一回合打满
- **冰爪**：2能 / 80威力 → 单回合伤害低，但费用比更优

**权衡维度：**

| 维度 | 高能耗高威力 | 低能耗低威力 |
|------|------------|------------|
| 单回合输出 | 高 | 低 |
| 能耗效率（伤害/能量） | 较低 | 较高 |
| 完成同等伤害所需回合 | 1 | 2~3 |
| 适用局势 | 一回合斩杀线 / 必杀机会 | 长线消耗 / 能量受限 |
| 风险 | 能量耗尽后无法续航 | 多回合内对方有反制时间 |

**决策规则：**

1. **能在本回合斩杀** → 选高威力技能，确保斩杀
2. **能量充足且对方下回合可能换人** → 选高威力，避免换人后伤害浪费
3. **长线对位且能量紧张** → 选低能耗，最大化能耗效率
4. **对方血线接近斩杀** → 算清楚低能耗技能能否一次斩杀（80威力够 vs 不够）
5. **预期对方会用防御** → 选低能耗（即使被防住，损失小）

**关键判断：剩余血量 / 单次伤害**

```
完成斩杀所需回合数 = ⌈对方剩余血量 / 单次伤害⌉
```

- 若高威力技能能 1 回合斩杀，低威力技能能 2 回合斩杀，对方下回合可能换人或反制 → 优先高威力
- 若双方都需要 2+ 回合，能耗效率更重要 → 选低能耗

## 四、状态技能决策

### 状态技能的类型

| 类型 | 示例 | 收益形式 |
|------|------|---------|
| 自身强化 | 力量增效、加速 | buff乘区，多回合输出提升 |
| 布设印记 | 风起印记、光合印记 | 场地长期收益 |
| 清除对方印记/强化 | 漫反射、清雾 | 解除对方的优势 |
| 开天气 | 放晴、降雨 | 全场属性增伤/减伤 |
| 给对方加 debuff | 麻痹、催眠 | 限制对方行动或降速 |
| 给对方挂 DoT | 毒孢子、引燃、寒风吹 | 持续消耗对方血量，逼迫换人 |

### DoT 类状态技能

DoT（Damage over Time）通过持续掉血实现间接输出，典型代表：毒孢子（中毒）、引燃（灼烧）、寒风吹（冰冻）等。

**核心价值：**

- **不依赖伤害公式**：DoT 按目标最大生命值百分比扣血（毒3%/层、灼烧2%/层），绕过物防/魔防计算
- **消耗对手回合**：对方需要使用清除技能（如净化、特定特性）解除，否则持续承受
- **逼迫换人**：毒/灼烧随精灵下场消失，但**冻结会保留**（换人无法清除冻结）

**适合使用 DoT 的情形：**

| 情形 | 说明 |
|------|------|
| 对方为高血量肉盾，难以直接斩杀 | DoT 按%扣血，不受防御影响 |
| 对方场下精灵都能克制我方 | 逼迫对方在劣势中选择（清除/换人/硬抗） |
| 多回合僵持，正面输出效率低 | DoT 累计消耗 |
| 对方刚使用过防御技能（冷却中） | 防御无法消除 DoT |

**不适合使用 DoT 的情形：**

- 对方本回合就能斩杀我方主力 → 优先防御/换人
- 对方携带快速清除手段（净化/解除特性）→ DoT 收益低
- 对方剩余精灵少，DoT 兑现周期超过对局长度

**DoT 配合换人逼迫：**

DoT 持续掉血让对方面临选择困境：
- 留场 → 持续掉血直到斩杀线
- 换人 → 我方建立优势对位，可继续输出/铺场地

### 印记类状态技能

印记是**场地层面的持续效果**，每方场地最多 1 个正面 + 1 个负面印记。印记的关键特性：**留在场地，不随精灵换人离场**——这是它与 debuff 的本质区别。

**常见印记效果：**

| 印记 | 效果 |
|------|------|
| 光合印记 | 我方每回合末获得1能量 |
| 风起印记 | 我方威力+20% |
| 棘刺印记 | 对方换人入场扣血 |
| 降灵印记 | 对方换人入场失能 |
| 湿润印记 | 我方所有技能能耗-1 |

**印记的战略价值：**

1. **长期收益累积**：6回合战斗中，光合印记 = 6 能量 / 风起印记 = 6 回合 × 20% 增伤
2. **跨精灵收益**：印记不随精灵换人消失，全队都能享受
3. **施压换人**：棘刺/降灵印记让对方换人变得昂贵，限制其切换属性博弈

**印记使用时机：**

| 情形 | 建议 |
|------|------|
| 优势对位下，对方无法立刻反制 | 优先铺设印记（长线收益） |
| 我方有清除手段保护印记 | 大胆铺设（应对漫反射等） |
| 对方场上有清除技能携带者 | 谨慎，可能被立即清除 |
| 对方频繁换人 | 棘刺/降灵印记针对性极强 |

**印记 vs 自身强化的选择：**

- 印记：跨精灵长期收益，但单回合不直接增加战力
- 自身强化：单精灵当前回合大幅增伤，但下场即失效

铺印记的精灵通常是"工具人"定位，铺完后可换出去让主力输出享受收益。

### 收益计算

```
状态收益 = buff乘区 × 后续回合预期输出提升 - 本回合放弃的输出机会
```

| 情形 | 收益评级 |
|------|---------|
| 物攻+100%（力量增效）后下回合必杀 | 高（1回合兑现） |
| 速度buff使我从后手变先手 | 高（改变出手顺序） |
| 清除对方关键印记/强化 | 高（消除对方资源优势） |
| 开有利天气配合下回合输出 | 中高 |
| buff需要3+回合才能兑现 | 低（对方可能换人或斩杀） |

### 应对风险：被对方攻击应对成功

**状态技能会被对方攻击应对**，应对成功后对方获得额外强化或伤害加成。

```
我方出状态 → 对方攻击应对成功 → 对方先手 + 伤害提升 + 我方buff被取消（或部分技能特殊效果）
```

| 情形 | 风险 |
|------|------|
| 对方有能量且攻击意图明显 | 高（被应对概率大） |
| 对方刚使用过攻击技能（能量消耗） | 低（更可能用状态/防御） |
| 对方处于防御冷却期 | 中（更倾向攻击） |
| 我方状态技能耗时长且兑现慢 | 高（即使没被应对，机会成本也高） |

### 状态技能使用的判断流程

1. 对方本回合是否能斩杀我？→ 能则不出状态
2. 对方本回合大概率出什么？→ 攻击则避免出状态
3. 收益能否在 1~2 回合内兑现？→ 否则风险大于收益
4. 是否有更好的清除/限制目标？→ 优先清除对方关键资源

### 应对机制的影响

- 我方出防御 → 对方攻击被应对 → 我先手且对方伤害减半
- 我方出状态 → 对方防御被应对 → 我先手
- 我方出状态 → 对方攻击被应对 → 对方先手且伤害加成（负收益）

## 五、能量机制

### 能量基本规则

- 每只精灵有 0~10 点能量，**没有每回合自动回复**
- 恢复能量的唯一通用方式是使用**聚能**（状态技能，回复5能量）
- 部分特性/印记/技能效果可以额外回能（如光合印记每回合末+1能）
- 能量不足时无法使用对应技能

### 能量差的战略价值

- 对方能量耗尽时，只能使用低威力技能或聚能，我方可以安全使用状态/buff技能
- 预判对方能量不足 → 可以放心使用需要多回合兑现的buff
- 我方能量充裕 → 可以在关键回合连续使用高能耗技能完成斩杀

### 建立能量差的途径

| 途径 | 说明 |
|------|------|
| 防御应对攻击 | 对方消耗高能耗技能却未造成伤害 |
| 报复类技能 | 直接消耗对手能量 |
| 强制对方换人（吓退/遁地） | 对方换人无法回能 |
| 低能耗技能 + 高输出 | 自身能量储备充足 |

## 六、防御技能决策

### 防御技能的特殊规则

- **冷却限制**：防御技能使用后，下一回合不能再次使用防御技能（必须间隔至少一回合）
- **能耗优势**：防御技能能耗普遍低于大威力攻击技能
- **应对成功的额外收益**：每个防御技能有自己独特的应对成功效果

### 常见防御技能及其应对成功效果

| 技能 | 应对成功效果 |
|------|------------|
| 风墙 | 迅捷防御，刚换上场的精灵可立刻释放 |
| 火焰护盾 | 给攻击方赋予灼烧 |
| 冰墙 | 给攻击方赋予冰冻 |
| 遁地 | 减伤并换人（主动换人不受换人惩罚） |
| 吓退 | 迫使进攻方换人 |
| 有效预防 | 下回合获得先手 |
| 嗜痛 | 自身获得攻击提升 |
| 报复 | 消耗对手能量 |
| 无畏之心 | 自身回血 |

不同防御技能的应对成功收益差别很大，选择时需匹配具体战术目标（控场/换人压力/能量压制/续航）。

### 防御建立的能量优势

```
我方出防御（低能耗）→ 应对对方攻击（高能耗）→ 对方消耗大量能量却未造成有效伤害
→ 我方能量相对充裕，对方核心技能能量被提前消耗
```

| 情形 | 能量收益 |
|------|---------|
| 防御应对成功，对方消耗5能攻击 | 对方损失5能，我方仅损失防御能耗（通常1~3能） |
| 使用「报复」应对成功 | 直接消耗对手能量，能量差进一步扩大 |
| 防御冷却期被对方攻击 | 无法防御，需改用其他指令 |

### 防御技能的使用时机

**适合使用防御的条件（同时满足）：**
1. 对方本回合大概率使用攻击技能（有能量且有攻击意图）
2. 我方防御技能不在冷却中
3. 对方攻击无法通过状态应对绕过防御

**不适合使用防御的条件：**
- 对方能量不足，大概率使用状态技能（防御被应对，反而吃亏）
- 防御技能处于冷却期
- 对方使用的是无法被防御应对的技能类型

## 七、换人决策

### 换人成本

主动换人回合对方可以**免费攻击一次**（换人惩罚）。

换人收益必须超过这个惩罚才值得。

### 流转机制（技能/特性触发的免费换人）

部分技能或特性带有"流转"效果——使用后获得一次**免费换人**（不承受换人惩罚）。

**流转的关键规则：出手顺序决定承伤者**

| 出手顺序 | 结果 |
|---------|------|
| 流转技能先出手 | 原精灵触发效果后离场，**换上场的精灵承受对手行动** |
| 流转技能后出手 | 原精灵承受对手行动后再触发流转，**场上的精灵承担伤害后离场** |

实战意义：
- 速度高的流转更安全（先出手，换上的精灵接受对手指令）
- 速度低的流转有损耗（场上精灵先承伤再换人，换人本身的资源依然消耗）

### 无伤换人：预判对手指令

主动换人可以与对手指令对应，达成"对方行动无效化、我方免费换人"的局面：

| 对手指令 | 我方主动换人结果 |
|---------|----------------|
| 聚能 | 对方仅获得能量，无攻击行动 → 无伤换人 |
| 防御技能 | 防御未触发（无攻击应对），相当于浪费一回合 → 无伤换人 |
| 部分状态技能（强化自身） | 对方获得 buff 但未造成伤害 → 几乎无伤（仅承担对方buff） |

**预判时机：**
- 对方刚使用过高能耗技能 → 大概率聚能
- 对方处于劣势但能量充足 → 可能用防御等待时机
- 对方有强力 buff 体系（力量增效等） → 可能选择强化

**预判失败的代价：**
- 对方实际选择攻击 → 换人惩罚 + 换上的精灵承受攻击
- 对方使用 debuff 类状态技能 → 换上的精灵承受 debuff（如中毒/灼烧）

### 优势对位：通过换人切换属性博弈

换人不只是"规避当前威胁"，更核心的作用是**建立优势对位**——换上一只克制对方或抵抗对方技能的精灵，使博弈天平倒向我方。

**优势对位形成的机制：**

```
对方主力打输出 → 我方换上抵抗其技能的精灵
→ 对方消耗能量却造成极少伤害 → 我方建立血量+能量双重优势
→ 对方留在场上的精灵无法威胁我方新上场精灵
```

**优势对位下的主动权：**

| 选择 | 收益 |
|------|------|
| 直接打出大威力攻击 | 对方无法反手斩杀，安全输出 |
| 强化自身（力量增效/速度增效等） | 免费 buff，为后续斩杀蓄力 |
| 铺设场地（印记/天气） | 场上精灵的优势延续到全队 |
| 预判对方换人，针对换上的精灵打 | 对方换人的免费回合用来打输出 |
| 预判对方继续硬刚，继续输出 | 对方损血累积，下回合面临斩杀线 |

**关键判断：对方是否会换人**

当我方建立优势对位后，对方通常会选择换人脱离不利对位。此时我方应预判：

- 对方场下哪只精灵能克制我方当前精灵？→ 提前准备针对它的技能
- 对方场下是否有迅捷换人/流转？→ 预判其换人时机
- 对方是否会硬刚（场下无更好选择）？→ 继续输出施压

**反过来：避免被对方建立优势对位**

- 不要把容易被克制的精灵留在场上承担长线对位
- 主力精灵被克制时优先换人，而不是硬刚
- 关注对方场下的"针对位"，提前做属性切换准备

---

### 适合换人的情形

| 情形 | 说明 |
|------|------|
| 当前精灵必死（对方下回合必杀） | 换人规避，保留精灵 |
| 换入精灵对对方有属性克制 | 3x伤害 vs 换人惩罚 |
| 换入精灵能抵抗对方技能 | 建立优势对位（对方输出无效化） |
| 换入精灵携带迅捷技能 | 换入即出手，抵消换人惩罚 |
| 当前精灵能量不足，换入精灵满能量 | 能量优势 |
| 预判对方使用聚能/防御/状态 | 无伤换人，免费调整阵容 |
| 携带流转技能且速度占优 | 用流转替代主动换人，规避惩罚 |

### 不适合换人的情形

- 场下精灵对对方无克制优势
- 换入精灵会被对方属性克制
- 当前精灵本回合能斩杀对方
- 预判对方攻击但场下无能承担伤害的精灵

## 八、多回合视角：资源差

单回合决策应服务于**多回合资源差**的积累：

```
资源差 = Σ(我方精灵存活数 × 血量%) - Σ(对方精灵存活数 × 血量%)
```

**优先级排序（从高到低）：**

1. 斩杀对方精灵（+1精灵优势，最高收益）
2. 规避我方精灵被斩杀（保留精灵）
3. 建立血量差（为后续斩杀创造条件）
4. 建立能量差（控制技能使用节奏）
5. 建立buff差（乘区优势）

## 九、特殊技能的博弈影响

部分技能改变了普通的应对/出手/能量逻辑，单独列出以便决策时检索。

### 地刺（攻击技能，打断任何状态技能/聚能）

- **效果**：地刺是攻击技能，可以应对并打断对方本回合使用的**任何状态技能**（包括聚能）。对方原本应获得的能量/效果直接消失，并承受地刺伤害
- **博弈意义**：
  - 对方频繁聚能/铺场地的精灵会被地刺压制
  - 我方使用状态技能前要预判对方是否带地刺
  - 地刺让"对方聚能=安全回合"的预判失效，且对方损失能量

**应对地刺的方式：**
- 对方有地刺携带者在场时，避免单纯聚能或慢节奏铺场
- 改为攻击/防御绕过地刺的应对窗口
- 用换人逼对方释放地刺后再切回主力

### 水刃（应对状态成功后永久免费）

- **效果**：水刃应对对方状态技能成功后，**该精灵直到战斗结束都可以无能耗释放水刃**
- **博弈意义**：
  - 一次应对成功 = 该精灵后续每回合都有"白嫖"输出选项
  - 对方面对水刃携带者时，使用状态技能的代价是"送一只精灵的免费输出权"
  - 我方使用水刃的精灵在场时，对方铺场地/强化决策极度谨慎

**水刃应对成功后的战略意义：**
- 该精灵几乎不会再有能量瓶颈，可以把能量全留给其他高能耗技能
- 多回合战斗中，免费水刃的累计伤害极高
- 即便后续换人下场再上场，免费效果**仍然保留**

### 通用规律

带有"打断"或"应对增益"特殊机制的技能会**改变状态/聚能决策的预期收益**。决策前必须确认：

1. 对方场上精灵是否携带这类技能？
2. 我方场上精灵是否携带这类技能（用作主动威慑）？
3. 对方是否可能换人切到这类精灵来针对我方状态决策？

## 十、特殊情形

### DoT 斩杀判定

若对方回合末 DoT 伤害足以击杀：
- 本回合无需攻击，可以使用状态/防御技能
- 收益 = 免费获得一回合buff + 对方必死

### 冻结即时斩杀

冻结在伤害结算后立即判定斩杀线（不等回合末）：
- 若伤害+冻结能即时斩杀，优先于回合末DoT

### 能量管理

- 高能耗技能（5能+）需要提前规划能量积累
- 对方能量不足时，可以预判其无法使用核心技能
