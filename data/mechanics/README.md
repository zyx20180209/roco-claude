# Mechanics

存放底层对战机制文档。

## 文件

- **`battle_terms.md`** — 对战词条库，按类别组织所有机制词条（基础机制 / 资源 / 天气 / debuff / 印记 / 玩家技能 / 血脉）
- **`turn_flow.md`** — 单回合结算的瀑布算法，从指令输入到回合末结算的完整流程
- **`damage_formula.md`** — 能力值与伤害计算公式（人类阅读版）
- **`damage_formula.json`** — 伤害公式的结构化定义（代码可读版）
- **`kill_threshold.md`** — 秒杀阈值分析：由伤害公式反推"要秒杀某精灵需要什么攻击+威力组合"

## 维护约定

- 机制按"词条"维护，不按系别文件拆分
- 系别只作为提问和索引线索，不作为存储主键
- 词条描述应尽量写清触发条件、结算方式、离场是否保留、是否免疫
