# 虫群伤害计算器

虫系奉献体系的专用伤害计算器，支持三种攻击者预设、奉献层数配置、目标库管理和队伍批量计算。

**在线使用：** https://zyx20180209.github.io/roco-claude/apps/swarm_calculator/

---

## 虫群机制说明

虫群（虫系物攻，基础威力 20，能耗 7，1 连击）可以通过奉献叠加获得强化：

| 奉献类型 | 效果 |
|---------|------|
| 威力奉献 | 每层 +20 基础威力 |
| 连击奉献 | 每层 +1 连击数 |
| 中毒奉献 | 每次虫群施加 2z 层中毒（每层每回合扣 3% 最大 HP） |
| 能耗奉献 | 每层 -2 能耗（3 层后虫群免费） |

奉献层数无上限，不会被清除。威力层数 x 和连击层数 y 效果对称（x/y 与 y/x 伤害相同）。

|--------|--------|------|

## 使用方法

### 1. 选择攻击者

点击顶部三个按钮切换攻击者，下方显示对应攻击力。

### 2. 输入奉献层数

- **威力层 (x)**：威力奉献层数
- **连击层 (y)**：连击奉献层数
- **中毒层 (z)**：中毒奉献层数（每次虫群施加 2z 层中毒）

### 3. 选择目标

三种模式：

**预定义**：从保存的目标库中选择，支持新建/编辑/删除。每个目标可配置：
- HP 个体值（默认 10）
- 物防个体值（默认 10）
- 性格：无 / 加HP / 加防御
- 物防 Buff 层数（每层 +10%）

**手动**：直接输入精灵名（自动补全），配置个体/性格/Buff。

**队伍**：选择预先保存的 6 只精灵队伍，批量计算对所有精灵的伤害。

### 4. 设置减伤

在目标区顶部输入敌方防御技能的减伤百分比（如 50 表示减伤 50%）。

### 5. 查看结果

**单目标结果：**
- 直接伤害数值和百分比
- 中毒每回合伤害和百分比
- 斩杀判定：
  - ✅ 直伤斩杀
  - ✅ 直伤+回合末中毒斩杀（同一回合内）
  - 🟡 第 N 回合末中毒斩杀
  - ❌ 无法斩杀

**队伍模式结果：**
6 行表格，显示每只精灵的伤害、%血量和斩杀判定。

## 属性克制规则

- 虫系技能对目标的克制倍率影响直接伤害
- 中毒（毒系）对目标的克制倍率影响 DoT 伤害
- **毒系精灵免疫中毒**（poisonPerTurn = 0）
- PVP 双属性规则：双弱 = 3x，双抗 = 1/3x

---

## 数据持久化

目标库和队伍配置存储在浏览器 `localStorage`，刷新页面后保留，清除浏览器数据后丢失。不同设备/浏览器之间不同步。
