# 精灵分析框架（analyze-pokemon）

> 本文档是 AI 分析精灵的完整框架，公开供社区参考、讨论和改进。
> 有问题或建议？欢迎提 [Issue](https://github.com/zyx20180209/roco-claude/issues)。

本框架用于生成一只精灵的完整分析报告，涵盖四维协同定位、战术角色、技能池、PVP 潜力、擅长应对的情况、棘手对手特点和自身弱点。

---

## 分析框架

### 1. 数据准备

使用 `queries.py` 工具库加载数据：

```python
from queries import (
    get_repo, max_speed, all_pokemon_skills, effective_total,
    find_by_skill, find_by_attribute, find_by_stats,
    find_by_ability, find_skills_by_keyword, summary,
    MULTI_TO_ONE_BOSS
)

repo = get_repo()
p = repo.find('精灵名')
defending_types = p['attributes_list']
defense_profile = repo.compute_defense_profile(defending_types)
stat_comp = repo.build_stat_comparison(p['stats'])

# 技能池（3类来源）
all_names = []
skill_source = {}
for k, label in [('moves', '自身学会'), ('jinengshi', '技能石'), ('xuemai', '血脉技能')]:
    for name in p.get('skills', {}).get(k, []):
        if name not in skill_source:
            skill_source[name] = label
            all_names.append(name)

# 速度公式
spd = int(p['stats']['speed'])
max_spd = (1.1 * (spd + 30) + 10) * 1.2 + 50  # 极速
full_spd = (1.1 * (spd + 30) + 10) + 50        # 满速

# 有效种族值
eff, removed = effective_total(p['stats'])
```

**首领化检测**：分析前检查精灵是否有首领化形态，若有则对比基础形态 vs 首领化形态的种族值差异、特性升级、战术价值变化。

**多形态检测**：检查同一 `base_name` 下的多个形态，对每个形态分别执行完整分析，最后输出"形态对比总结"章节。

**状态机制参考**：涉及萌化、冻结、中毒、灼烧等状态时，以 `data/mechanics/battle_terms.md` 为权威来源。

### 2. 四维协同定位

评估精灵在【攻击力度 / 耐久 / 功能性】三个方向上的维度支持数：

- **攻击力度信号**：攻/特攻百分位高、速度快、特性加攻、技能池有低费高效输出技、有攻速双修技
- **耐久信号**：HP+防+特防总和高、特性减伤/复活/续航、技能池防御技多、属性抗性广
- **功能性信号**：特性工具型（偷印记/换人/传递）、技能池有印记/状态/控场/净化/换人技、属性有针对性

| 方向 | 支持维度数 | 主要支持点 |
|------|---------|---------|
| 攻击力度 | X维 | ... |
| 耐久 | X维 | ... |
| 功能性 | X维 | ... |

**主定位**：支持维度最多的方向（4维=绝对主定位，3维=主定位，2维=明显潜力）

### 3. 战术角色（7种）

基于四维定位，确定精灵在队伍中的战术角色（可多个）：

| 角色 | 判断标准 |
|------|---------|
| **收割** | 极速≥240+攻击≥120+先手技；或强化后满血秒杀路线 |
| **炮台** | 有效种族≥550+攻击≥110+bulk≥270，主动建立输出优势 |
| **拦截** | 能在场上存活并发挥作用（多种实现：高bulk/特性复活/技能减伤/入场压制/广抗性）|
| **破盾** | 有百分比DOT特性/技能（毒/冰/灼烧）；或超高输出逼迫拦截位下场 |
| **工兵** | 有印记铺设/强化传递/天气开启/奉献产出技能 |
| **清场** | 有驱散敌方buff/印记的技能（晒太阳/倾泻/食腐/翅刃等）|
| **中转** | 有主动换人技（吓退/掩护脱离）或特性触发换人（哨兵/洁癖）|

**拦截的多种实现**（不只靠种族值）：
- 种族值型：bulk≥330
- 特性复活型：不朽/化茧/3条命
- 特性减伤型：绝对秩序-50%/偏振-40%
- 技能减伤型：有2+个高质量减伤技（无畏之心/风墙/嗜痛等）
- 入场压制型：入场即削弱对手攻击力
- 属性抗性型：7+条半抗

### 4. 种族值与速度

- 总种族 / 有效种族（去除弱势攻击）
- 极速 / 满速
- 各项百分位
- 启动速度评估：即时/快速/中速/慢速/条件启动

### 5. 属性抗性

- 3x/2x弱点、半抗/1/3抗、免疫
- 属性打击面覆盖

### 6. 特性分析

- 触发条件和发挥场景
- 对攻击/防御/资源的实际影响
- 局限性

### 7. 技能池亮点

按功能分组（输出/增益/防御/资源/控场/DOT），标注技能来源（自身/技能石/血脉）：
- 重点标注：低费高效技、核心联动技、应对触发技、永久强化技
- 血脉技能中的高价值变体（如风墙、升龙咆哮等）

### 8. 叠加型技能发育分析

**识别条件**：技能描述含"永久+X"或"每次使用后威力/攻击永久增长"（如吹火、山火）。

若精灵有此类技能且以其为核心输出路线，必须分析：

- **发育曲线**：叠加N次后威力/攻击达到什么水平？几回合能完成基础发育？
- **发育阶段风险**：发育期间精灵是否能自保？遇到克星被迫下场会损失多少叠加进度？（永久buff下场不消失，但出手机会被浪费）
- **队伍依赖度**：是否需要中转精灵为其创造安全出手窗口？
- **发育路线 vs 即时路线对比**

### 9. 攻速双修评估

- 是否有攻速双修技（啮合传递/龙吟等）
- 强化后极速是否能进入先手击杀循环

### 10. 推荐配招方向

结合战术角色给出2-3套配招，每套必含输出技（含DOT）。

### 11. 擅长应对的情况

**不涉及具体精灵名，描述对手的特征**：

- 擅长应对**什么类型的对手**（如"高能耗技能池的炮台"、"依赖强化的收割位"、"赖场型拦截"）
- 擅长在**什么战局节奏**下发挥（如"消耗战后期"、"对手铺设完成前"、"己方有联防保护时"）
- 擅长应对**什么体系**（如"印记体系"、"强化体系"、"高速爆发队"）

### 12. 棘手的对手特点与自身弱点

**不涉及具体精灵名，描述对手的特征**：

- 什么样的对手让它**难以发挥**（如"极速超过它的先手压制"、"带自清debuff技能的对手"）
- 自身的**结构性弱点**（如"启动慢被打断"、"物防极脆被物攻爆发秒杀"、"特性依赖条件触发"）
- 什么样的**队伍配置**会让它陷入困境

### 13. PVP潜力评价（8维）

| 维度 | 评分 | 理由 |
|------|------|------|
| 独特性 | ⭐⭐⭐⭐ | 机制是否不可替代 |
| 环境适应度 | ⭐⭐⭐ | 对局覆盖率 |
| **启动速度** | ⭐⭐⭐⭐ | 即时/快速/中速/慢速/条件 |
| 构筑门槛（容错率）| ⭐⭐⭐⭐ | 面板能否自保，独立作战能力 |
| 弱点严重度 | ⭐⭐⭐ | 弱点属性热门程度 |
| 环境克星密度 | ⭐⭐⭐⭐ | 几只热门能克它 |
| 克制热门能力 | ⭐⭐⭐ | 能反制几只热门核心 |
| **博弈威慑力** | ⭐⭐⭐⭐ | 对手需要多少额外准备 |

**综合评级**：S/A/B/C/D 级

**实战推荐**：是否建议带入PVP，适合什么队伍，替代方案

### 14. 队伍需求

**精灵缺的（反向需求）**：
- 属性补位（2x/3x弱点需要谁接招）
- 克星清理
- 机制补位（速度劣势/无续航/无解场等）
- 发育保护（若有叠加型核心技能）

**精灵给的（正向贡献）**：
- 对队友的支援
- 针对性克制
- 与通用外部解法（愿力冲击+血脉）对比的独特价值

**体系归属**：最适配的体系名+在体系中的位置

## 核心机制约束（分析时必须考虑）

- 每只精灵初始10点能量，聚能回复5点（不占技能栏）
- 全队共享4点魔力，精灵力竭消耗1点，耗尽判负
- 技能栏4个，构筑取舍是核心难点
- 速度公式：`[1.1×(种族值+30)+10]×性格系数+50`，道具/buff加算
- 属性倍率：双克3x，双抗1/3，一克一抗中和，免疫优先
- 印记在场地层面，每方1正1负；debuff随精灵下场消失（永久buff除外）
- 愿力冲击：玩家技能，2能80威力，属性跟血脉走，与首领化二选一，可应对状态类技能，造成1.5倍伤害

## 赖场型精灵特殊考量

- 是否怕百分比叠加状态（中毒/冰冻/灼烧）— 毒/冰/火系可以免疫相应状态
- 是否有主动脱离手段
- 输出下限（赖场成功后能否打出足够伤害）

## 先手吞招与攻速双修

- 先手击杀后对方本回合动作被吞掉
- 攻速双修技（如啮合传递1能+80速+60%物攻）是战术拐点
- 中速精灵一次攻速双修后可跃升高速段

## 印记与Debuff系统

- Debuff随精灵走，永久buff（技能描述含"永久"）跨场次保留
- 印记在场地层面，不随换人消失，每方1正1负
- 清除系统三类不通用：驱散敌方buff / 驱散印记 / 清除自身debuff

## 血脉属性与技能来源

- 3类技能来源：moves（自身）/ jinengshi（技能石）/ xuemai（血脉）
- 血脉技能随血脉属性变化，可定制抗性和技能变体
- 应对XX类技能有双触发条件，战术价值常被低估

## 首领化形态识别

- `form_name == '首领形态'` 的精灵不能直接入队，需通过首领化技能从基础形态转换
- 多对一首领化：霜翼领主（4只岚鸟）、棋契陛下（8只棋子）、钻石蜗（6只晶石蜗）

## PVP评分校准原则

- 种族值不决定一切：有效种族值（去除弱势攻击）更准确
- 启动速度是决定使用率的关键维度
- 博弈威慑力：多套玩法/一击秒杀威胁/博弈失败代价
- 环境克星密度：克星是否是热门精灵
- 精灵有多套玩法时，启动速度按最快玩法评分

## 强化传递精灵的占位价值判断

- 接收精灵原速度越低+攻击越高+无自攻速双修 → 传递价值越大
- 菊花梨+伊兰亚龙（永久+150速从220→370）是典型案例
- 翠顶夫人洁癖传递是临时buff（接收方下场消失），与菊花梨永久传递不同
